📋 听点总表（全部 18 锚 · ⭐ = 核心听点 · 6 项板块边界守门）
类别听点学生说到位时，话里应包含什么
两种陈述⭐⭐ 克劳修斯陈述不可能把热量从低温物体传到高温物体而不引起其他变化；「自发 / 不引起其他变化」是关键限定词。
⭐⭐ 开尔文陈述不可能从单一热源吸取热量全部用来做功而不引起其他变化；等价于「没有第二类永动机」。
两陈述等价逻辑等价（任一可推出另一）；严格证明留给进阶卡诺循环板块。
⭐ 「自发」的定义「自发」= 不引起其他变化 / 不借助外力；空调冰箱消耗电能 → 不算自发，因此不挑战陈述。
自发方向 + 不可逆⭐ 自发方向自然过程都有单向性（热高→低 / 气体扩散 / 油墨入水 / 摩擦消耗机械能 / 鸡蛋摔碎）。
⭐ 不可逆性反向不自发（即使第一定律允许）；走反向需付出代价（消耗外功 / 引起其他变化）。
自发判据（定性）自发过程 = 系统（或孤立系统）无序度增加 = 熵增。
热机 + 永动机⭐⭐ 第二类永动机不可能单一热源吸热全变功；满足第一定律但违反开尔文陈述。
⭐ 第一类 vs 第二类对比第一类违反能量守恒 / 第二类违反方向；两类同时不可能但理由不同 → 第一二定律独立作用。
⭐ 热机能量流向Qh = W + Qc：吸热部分变功、部分变废热。
⭐⭐ 热机效率 η < 1η = W/Qh = 1 − Qc/Qh；Qc > 0 是开尔文陈述的物理要求，不是工程局限。
⭐ 需要温差热机必须有 Th vs Tc，不能单靠一个恒温热源（否则违反开尔文陈述）。
熵 + 独立性⭐ 熵的定性熵 S = 系统无序度的度量；本板块不引 dS = dQ/T 与玻尔兹曼公式。
⭐⭐ 熵增原理（定性）孤立系统 ΔS ≥ 0：可逆 = 0 / 自发 > 0 / 不可能 < 0。
⭐ 局部熵减不违反第二定律局部熵可减少（如水结冰、生命系统），但「局部 + 环境」孤立系统总熵增。
⭐ U 与 S 独立U 由第一定律管（数量）、S 由第二定律管（方向）；理想气体绝热自由膨胀 = U 不变但 S 增 的典型例。
两定律分工⭐⭐ 第一定律管「数量」、第二定律管「方向」独立而互补：能量守恒不能告诉我们方向；第二定律是新的独立约束。

📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写两陈述」或「画 Qh = W + Qc 热机能量流向图」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
