STEP 0 · 13 题位 · 板块碰撞骨架介绍

在做任何一道题之前，先用 5 分钟把这一题位的"骨架长什么样"刻进脑子。下面 4 段是骨架本身，不是技巧——技巧在 3 道例题里。

骨架 1 · 板块碰撞 = 多体相互作用 + 摩擦 + 共速
题面里出现 2 个或更多物体（小球 / 物块 / 滑板 / 子弹），并且物体之间通过 摩擦或瞬时碰撞 发生力的相互作用，整套系统最终趋向 "全部同速"或"分组同速"——这就是 13 题位的标准骨架。
"光滑水平面 + 滑板 + 物块"的组合是该骨架最常用的载体：光滑地面保证 系统外力沿水平方向为零，因而水平方向动量守恒；板块之间的摩擦是 系统内力，不破坏动量守恒但消耗机械能。
骨架 2 · 高考 13 题位高频骨架（动量守恒类）
据近 5 年全国卷与新高考统计：13 题位的 10-13 分压轴题中，"动量 + 能量"双观点解答出现率高居榜首（参见 2022 河北 13、2020 山东 17 等真题骨架）。出题方式以 "2 小问" 为主：第 (1) 问考动量守恒（碰撞 / 嵌入 / 黏合），第 (2) 问考能量守恒 + 几何约束（相对位移 / 板长 / μ 范围）。
本任务卡选取的 3 道例题分别对应该骨架的三类典型变体：两板对向碰撞（例题 1）/ 子弹嵌入板上物块再共速（例题 2）/ 单物块滑上多板组（例题 3）。三例覆盖率约占 13 题位历年真题骨架的 70% 以上。
骨架 3 · 识别关键字（题面里只要见到就立即对号入座）
「碰撞」：两物体在极短时间内发生力的相互作用 → 优先考虑"动量守恒"，再判断是否完全弹性 / 非弹性。
「黏合」/「嵌入」：碰后两物速度相同 → 完全非弹性碰撞 → \(v = (m_1 v_1 + m_2 v_2)/(m_1 + m_2)\)，必有动能损失 = 系统内能增量。
「共速」/「不滑离」：物体间最终达到同一速度 → 摩擦力即将消失的临界时刻 → 用动量守恒求共速、用能量守恒求摩擦生热 \(Q = f\Delta s_{\text{相对}}\)。
「相对滑动」/「不脱离板面」：典型几何约束 → 相对位移 \(\Delta s_{\text{相对}} \le L\)（板长），由此反推 μ 范围 / 初速 / 板长。
骨架 4 · 核心判据 3 条（板块碰撞的"通关密钥"）
判据 A · 共速时刻：相对速度 = 0 的瞬间 = 摩擦力即将消失的瞬间 = 系统内能停止增加的瞬间。三个等价描述任写其一即可，但必须在卷面上明确指出。
判据 B · 相对位移：摩擦生热 \(Q = f \cdot \Delta s_{\text{相对}}\) 中的 \(\Delta s_{\text{相对}}\) 是两个相互摩擦面之间的相对滑动距离，不是任何单物体的绝对位移。这是 13 题位最高频扣分点（约占该题位扣分的 30%）。
判据 C · 系统选取：动量守恒要先选系统。规则：① 瞬间事件（碰撞 / 嵌入 / 黏合）→ 选直接作用的两物为系统，外力冲量趋于零，动量守恒；② 持续过程（摩擦滑动）→ 选所有发生相对滑动的物体为整体，水平方向外力为零（光滑地面）则水平方向动量守恒。系统不同 → 方程不同 → 答案错。

读完以上 4 段，先合上页面，在草稿上把"识别关键字 → 选系统 → 守恒律 → 共速 → 相对位移"这条链路默写一遍。能默出来，再进入例题 1；默不出来，再回上面读一次。

STEP 1核心知识速复习
5-8 分钟

下面 4 条公式 + 2 条判据是例题 1 用到的全部工具。先合上页面回忆，再对照。

动量 + 能量速查清单
动量守恒方程：\(m_1 v_1 + m_2 v_2 = m_1 v_1' + m_2 v_2'\)（向量量，必须先约定正方向）
完全非弹性碰撞：碰后共速 \(v_{\text{共}} = (m_1 v_1 + m_2 v_2)/(m_1 + m_2)\)
滑动摩擦力：\(f = \mu N\)（方向沿接触面，与相对运动方向相反）
摩擦生热：\(Q = f \cdot \Delta s_{\text{相对}}\)（必须用相对位移）
判据 · 共速时刻=相对速度为零 = 摩擦力即将消失
判据 · 系统选取=瞬间事件选直接相碰两物 / 持续过程选发生相对滑动的所有物体
STEP 2题面
双板对向 + 物块黏合
2 小问
光滑水平面上有两块等长的滑板 A 与 B，质量分别为 \(m_A = 1\) kg、\(m_B = 2\) kg，板长均为 \(L = 0.5\) m。A 板右端、B 板左端分别静止放置物块 C、D，质量均为 \(m = 1\) kg。某时刻起，A 与 C 整体以 \(v_0 = 2\) m/s 向右运动，同时 B 与 D 整体以 \(v_0 = 2\) m/s 向左运动。两组对向运动后，C 与 D 在两板之间的中点相遇并立即黏合成新物块 P。物块（含 P）与板之间的动摩擦因数均为 \(\mu\)，板与水平面之间光滑。取 \(g = 10\) m/s²。

(1) 求 C、D 黏合后新物块 P 的速度（大小和方向）。[约 4 分]
(2) 若要使 P 与某一板最终达到共速且不滑离该板的板面（板长 \(L = 0.5\) m），求动摩擦因数 \(\mu\) 应满足的取值范围。[约 7 分]

STEP 4方法路径
5 步串起整题
路径 1 · 选系统（第 (1) 问）
C 与 D 在中点黏合 = 一维完全非弹性碰撞。黏合瞬间时间极短 → 摩擦力的冲量 = \(f \cdot \Delta t \to 0\) → 可忽略 → 选 C+D 为系统，动量守恒。
"我把黏合视作瞬间事件，外力冲量忽略。请反问我：为什么'外力存在'与'外力冲量为零'不矛盾，不要直接告诉我结论。"
系统：C + D（瞬间事件，外力冲量 ≈ 0，动量守恒）
取向右为正：\(m v_0 + m(-v_0) = 2m \cdot v_P\)
解得 \(v_P = 0\)（P 黏合后静止）。
路径 2 · 写动量守恒（第 (2) 问开局）
第 (1) 问得 \(v_P = 0\)。此时 A 板仍以 \(v_0 = 2\) m/s 向右、B 板仍以 \(v_0 = 2\) m/s 向左。P 落在两板正中。问自己：P 此刻同时压在 A、B 之上，摩擦力如何作用？
"请帮我列出此刻 P 所受的全部水平力（如果 P 同时压在 A、B 两板之上）。不要替我决定 P 最终落到哪板。"
由题意"P 与某一板共速且不滑离"——意味着题目暗示 P 最终被某一板带走（设为 B 板向左，由于 \(m_B > m_A\)，左行的总动量更大）。
选 P + B 板为系统，水平方向地面光滑 → 动量守恒：\(2m \cdot 0 + m_B \cdot (-v_0) = (2m + m_B) \cdot v_{\text{共}}\)
代入 \(m = 1, m_B = 2, v_0 = 2\)：\(v_{\text{共}} = -2 \times 2 / 4 = -1\) m/s（即向左 1 m/s）。
路径 3 · 分情况讨论（关键步）
注意：上面"P 跟 B 共速"是一种结果，题目要求"不滑离板面"——这意味着 P 在 B 板上滑动的相对位移不能超过板长 \(L\)。这是 μ 的下限来源（μ 过小 → 摩擦小 → 相对滑动距离过长 → 滑出板面）。
"我要建立'相对位移 ≤ L'的不等式。请反问我：相对位移应该用 P 相对 B 板的滑动距离，而不是 P 或 B 各自的绝对位移，原因是什么？"
分两种情况：
情况 i：P 被 B 板带走（左行共速，本题主线）。
情况 ii：P 留在 A 板上（右行共速）——分析：碰后 A 向右 \(v_0\)、B 向左 \(v_0\)、P 静止。P 同时受 A 向右摩擦 \(\mu mg\) 与 B 向左摩擦 \(\mu mg\)，二者大小相等方向相反 → P 加速度起步为 0；但 B 减速比 A 慢（\(m_B > m_A\) → B 受 P 摩擦减速度小）→ B 先与 P 共速，A 仍向右 → 之后 A 单独继续向右 → P 留在 B 板上向左共行。故主线即情况 i。
路径 4 · 共速条件（能量守恒求 μ 下限）
共速时刻：P 与 B 板速度都等于 \(v_{\text{共}} = -5\) m/s。摩擦生热 \(Q = \mu \cdot (2m) g \cdot \Delta s_{\text{相对}}\)（注意 P 对 B 板的正压力 = P 的重力 = \(2mg\)）。能量守恒：损失的动能 = Q。
"请帮我写'初末动能差 = 摩擦生热'，并把相对位移用 \(\mu, m, m_B, v_0, g\) 表达。不要直接代数。"
初动能：\(E_{k0} = \tfrac12 (2m) \cdot 0^2 + \tfrac12 m_B v_0^2 = \tfrac12 \cdot 2 \cdot 4 = 4\) J
末动能：\(E_k = \tfrac12 (2m + m_B) v_{\text{共}}^2 = \tfrac12 \cdot 4 \cdot 1 = 2\) J
ΔE = 2 J = \(\mu \cdot 2mg \cdot \Delta s_{\text{相对}}\) → \(\Delta s_{\text{相对}} = 2 / (\mu \cdot 2 \cdot 1 \cdot 10) = 0.1/\mu\) m
路径 5 · μ 下限 + 上限
几何约束：\(\Delta s_{\text{相对}} \le L = 0.5\) m → \(0.1/\mu \le 0.5\) → \(\mu \ge 0.2\)。这一下限对应"摩擦正好把 P 与 B 共速所需相对滑动 0.5 m 全用完"。μ 越大相对位移越短，越保险。
"请帮我检查上述推导：① 系统选取 ② v_共 的符号 ③ ΔE 的代数。不要替我重新设计题面。"
最终答案：\(\boxed{0.2 \le \mu \le 1}\)。
下限来自"相对位移 ≤ 板长"几何约束：\(\Delta s_{\text{相对}} = 0.1/\mu \le L = 0.5\) → \(\mu \ge 0.2\)；上限来自动摩擦因数的物理意义（一般 \(\mu \le 1\)）。
物理意义：μ 越大摩擦越强，P 与 B 共速所需相对滑动越短，越容易满足"不滑离 B 板"的几何条件。
提示：写卷面时只要诚实写出 \(0.1/\mu \le 0.5\) 这一步的来历，老师就给完整的几何约束分。μ 上限不写也不扣分（题面没要求物理上限），但写出"μ ≤ 1（动摩擦因数物理意义上限）"是加分项。

例题 2 · 子弹嵌入 + 板上系统共速（基于 53 模 山东青岛期末 改编）

▼
学习目标：以"子弹嵌入物块"为骨架，掌握"瞬间动量守恒 → 后续摩擦能量守恒"两阶段分段；理解为什么子弹嵌入这一段不能用能量守恒倒推
建议时长：70 分钟（速复习 8 + 题面 5 + 自己做 20 + 对照思考 22 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

本题与例题 1 的区别：子弹嵌入这一段必须单独作为"瞬间动量守恒"处理，不能把它与后续板上滑动合并成一个能量方程。理由是：子弹的初动能极大（速度 300 m/s），嵌入瞬间产生的内能远远大于后续滑动摩擦生热，混在一起列方程会丢掉 4-6 分。

两阶段分段速查
阶段 I · 子弹嵌入：时间极短，子弹 + 物块系统动量守恒；与板 / 地面间摩擦力的冲量 ≈ 0；这一段动能损失 = 阶段 I 内能。
阶段 II · 板上滑动：嵌入完成后，把（子弹+物块）视作整体（记为 M），与板之间 \(\mu_1\) 摩擦，板与地面之间 \(\mu_2\) 摩擦；持续过程，水平方向系统动量守恒（M + 板的水平外力 = ? 注意要判断板是否相对地面滑动）。
关键判断：板与地面之间是否相对滑动？取决于 \(\mu_1 M g\) 与 \(\mu_2 (M + m_{\text{板}}) g\) 的大小关系。若前者 ≤ 后者最大静摩擦，则板不动；若前者 > 后者，则板被拖动。
能量守恒（阶段 II）：\(\tfrac12 M v_M^2 - \tfrac12 (M+m_{\text{板}}) v_{\text{共}}^2 = Q_1 + Q_2\)，其中 \(Q_1\) 是 M 与板之间的摩擦生热，\(Q_2\) 是板与地面之间的摩擦生热（若板滑动）。
STEP 2题面
子弹嵌入 + 板上共速
水平地面上静止放置一长木板，板的质量 \(m_{\text{板}} = 10\) kg；板上静止放置一物块，物块质量 \(m_{\text{物}} = 0.99\) kg。一颗质量 \(m_{\text{弹}} = 0.01\) kg 的子弹以水平速度 \(v_0 = 300\) m/s 射入物块并迅速嵌入物块内部（嵌入时间极短，可视为瞬间事件）。嵌入完成后，（子弹+物块）视作一个整体 M，M 与板之间的动摩擦因数 \(\mu_1 = 0.2\)，板与地面之间的动摩擦因数 \(\mu_2 = 0.1\)。取 \(g = 10\) m/s²。

(1) 求子弹嵌入物块瞬间，（子弹+物块）整体 M 的共同速度 \(v_M\)。[约 4 分]
(2) 求 M 与板系统最终达到共同速度时，全过程（含阶段 II）损失的总机械能 = 系统内能增量。[约 8 分]
SVG 2 · 子弹以 300 m/s 水平射向静止物块；物块与长板间 μ₁ = 0.2，板与地面间 μ₂ = 0.1。

STEP 4方法路径
阶段 I → 阶段 II → 总内能
路径 1 · 阶段 I 子弹嵌入（瞬间动量守恒）
嵌入时间极短 → 摩擦力（物块–板间 \(\mu_1\)、板–地间 \(\mu_2\)）的冲量都趋于 0 → 选子弹 + 物块为系统，动量守恒。这一段板和地面都不参与。
"请反问我：为什么这里选'子弹+物块'而不选'子弹+物块+板'？不要直接说答案。"
系统：子弹 + 物块（瞬间事件，外部冲量 ≈ 0）
取向右为正：\(m_{\text{弹}} v_0 + m_{\text{物}} \cdot 0 = (m_{\text{弹}} + m_{\text{物}}) \cdot v_M\)
代入 \(0.01 \times 300 = 1 \times v_M\) → \(v_M = 3\) m/s（向右）。
整体 M 的质量 \(M = m_{\text{弹}} + m_{\text{物}} = 1\) kg。
路径 2 · 阶段 I 内能（顺手算一下，用得上）
阶段 I 损失的动能 = 子弹的初动能 - M 的末动能。这一段内能不是题目问的（题目问的是阶段 II），但你可能在 STEP 5 评分锚点里需要它来检验"总内能"的合理性。
\(Q_{\text{I}} = \tfrac12 m_{\text{弹}} v_0^2 - \tfrac12 M v_M^2 = \tfrac12 \cdot 0.01 \cdot 90000 - \tfrac12 \cdot 1 \cdot 9 = 450 - 4.5 = 445.5\) J（这就是阶段 I 损失的动能，绝大部分都化为子弹嵌入物块时的内能）。
路径 3 · 阶段 II 判断板是否被拖动
M 以 \(v_M = 3\) m/s 在板上滑动。M 给板的摩擦力 \(f_1 = \mu_1 M g = 0.2 \times 1 \times 10 = 2\) N（向右）。板若要动，必须克服地面给板的最大静摩擦 \(f_2^{\max} = \mu_2 (M + m_{\text{板}}) g = 0.1 \times 11 \times 10 = 11\) N。\(f_1 = 2 < f_2^{\max} = 11\) → 板不动。
"请反问我：如果 f₁ < f₂_max，板是不是'完全不动'？地面摩擦力等于多少？不要直接说。"
由于 \(f_1 = 2\) N < \(f_2^{\max} = 11\) N，板始终静止不动。此时地面给板的实际摩擦力 = M 给板的摩擦力 = 2 N（向右，静摩擦，平衡板上的水平受力）。
板既然不动 → "M 与板系统达到共速" = M 减速到 0（共速速度 \(v_{\text{共}} = 0\)）。
路径 4 · 阶段 II 内能（M 在板上滑行 → 停止）
由于板不动，M 在板上滑行的相对位移 = M 的绝对位移。M 的减速度 \(a = \mu_1 g = 2\) m/s²，由 \(v_{\text{共}}^2 - v_M^2 = -2a \cdot s_M\) 得 \(s_M = v_M^2/(2a) = 9/4 = 2.25\) m。
阶段 II 摩擦生热 \(Q_{\text{II}} = f_1 \cdot \Delta s_{\text{相对}} = 2 \times 2.25 = 4.5\) J。
校验：阶段 II 损失的动能 = \(\tfrac12 M v_M^2 - 0 = 4.5\) J ✓ 与 \(Q_{\text{II}}\) 一致。
（地面摩擦力 2 N 是静摩擦，板没动 → 这段地面摩擦不做功、不产生内能。）
路径 5 · 总损失机械能（题目要问的）
题目第 (2) 问"全过程损失的总机械能"包含阶段 I + 阶段 II。注意题面说"M 与板系统达到共速时"——这暗指从子弹射入到 M 停止这一全过程。
\(\Delta E_{\text{总}} = Q_{\text{I}} + Q_{\text{II}} = 445.5 + 4.5 = \boxed{450 \text{ J}}\)。
等价于：\(\tfrac12 m_{\text{弹}} v_0^2 - \tfrac12 (M + m_{\text{板}}) \cdot 0^2 = 450\) J（初动能全部转化为内能）。
物理意义：450 J 全部转化为系统内能（绝大部分来自子弹嵌入，少部分来自 M 在板上的滑行）。
提示：若题目只问"阶段 II 损失"，则答案为 4.5 J；若问"全过程损失"，则为 450 J。卷面上务必看清问的是哪一段。

例题 3 · 单物块 P 滑上 3 块紧贴板（自创变式 · 多体多过程升级）

▼
学习目标：以"P 滑上多板紧贴序列"为骨架，掌握"多体共速判定 + 相对位移分段累加"的高阶处理；理解为什么紧贴板可视为整体而 P 越过哪板需单独追踪
建议时长：80 分钟（速复习 8 + 题面 5 + 自己做 25 + 对照思考 27 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

本题与例题 1、2 的区别：3 块紧贴板共同受 P 的摩擦推动。只要紧贴，三板可视为整体（共同加速度）；但 P 的相对位移需要追踪它滑过了哪些板，这是几何判定，不能用整体方法替代。

紧贴板速查清单
紧贴判据：三板紧贴 = 板间接触力 ≥ 0（不能为负，否则分离）。本题设三板始终紧贴 → 三板共加速度。
P 与三板系统的动量守恒：地面光滑 → 水平动量守恒：\(m_P v_0 = (m_P + 3m_{\text{板}}) v_{\text{共}}\)。
共速判定：P 减速 + 板加速 → 共速时刻 = 相对速度为零。
能否共速的反向判据：若 P 减到与板等速前，相对位移已超过"三板总长"3L，则 P 必从 C 板右端滑出，不可能共速。
相对位移分段：\(\Delta s_{\text{相对}} \in [0, L]\) → P 仍在 A 上；\([L, 2L]\) → P 在 B 上；\([2L, 3L]\) → P 在 C 上；\(> 3L\) → P 已滑出。
STEP 2题面
3 板紧贴 + P 滑入
光滑水平面上有 3 块完全相同的小滑板 A、B、C，每块质量 \(m_{\text{板}} = 1\) kg，板长 \(L = 1.5\) m，三板紧贴排成一行（A 在最左，C 在最右）。最左 A 板上、靠左端，静止放置一物块 P，质量 \(m_P = 2\) kg。P 与所有板（A、B、C）之间的动摩擦因数均为 \(\mu = 0.3\)，地面与板之间光滑。某时刻 P 受一瞬时冲量后获得水平向右的初速 \(v_0 = 6\) m/s（冲量作用极短，不改变 P 的位置）。取 \(g = 10\) m/s²。

(1) 判断 P 与三板系统能否最终达到共同速度（即 P 是否会从 C 板右端滑出）。[约 5 分]
(2) 若可共速，求共同速度的大小；并指出 P 最终停在哪一段板上（A / B / C），且距离该板左端多远。[约 7 分]
SVG 3 · 三块完全相同板紧贴排列；P 静置 A 板左端，瞬间获得 v₀ = 6 m/s 向右。

STEP 4方法路径
假设共速 → 验算 → 定位
路径 1 · 假设共速 → 动量守恒求 v_共
地面光滑 → P + 三板系统水平动量守恒。三板紧贴 → 共加速度 → 共末速度。
系统：P + (A+B+C)（地面光滑，水平动量守恒）
\(m_P v_0 = (m_P + 3 m_{\text{板}}) v_{\text{共}}\)
\(2 \times 6 = (2 + 3) \cdot v_{\text{共}}\) → \(v_{\text{共}} = 2.4\) m/s。
路径 2 · 能量守恒求"假设共速"所需相对位移
摩擦生热 = 初末动能差。\(f = \mu m_P g\)（P 对板的摩擦力，板对 P 的反作用力大小相同方向相反）。
\(\Delta E_k = \tfrac12 m_P v_0^2 - \tfrac12 (m_P + 3m_{\text{板}}) v_{\text{共}}^2 = \tfrac12 \cdot 2 \cdot 36 - \tfrac12 \cdot 5 \cdot 5.76 = 36 - 14.4 = 21.6\) J
\(f = \mu m_P g = 0.3 \cdot 2 \cdot 10 = 6\) N
\(\Delta s_{\text{相对}} = \Delta E_k / f = 21.6 / 6 = 3.6\) m
路径 3 · 与"三板总长"3L 比较 → 第 (1) 问结论
三板紧贴总长 \(3L = 4.5\) m。P 最初在 A 板左端，向右滑动；它在板上"能走"的相对位移最多就是 \(3L\)（从 A 板左端走到 C 板右端）。
\(\Delta s_{\text{相对}} = 3.6\) m < \(3L = 4.5\) m → P 在滑出 C 板前就与三板共速。
第 (1) 问答案：✅ 能共速，不会滑出。
路径 4 · 定位 P 最终停在哪板
P 起点在 A 板左端，相对位移 3.6 m。把 3L = 4.5 m 切成 3 段：[0, 1.5]→A 上；[1.5, 3.0]→B 上；[3.0, 4.5]→C 上。3.6 m 落在第 3 段 [3.0, 4.5] → P 最终在 C 板上。距离 C 板左端：\(3.6 - 3.0 = 0.6\) m。
P 最终位置：停在 C 板上，距离 C 板左端 0.6 m（共速速度 2.4 m/s 向右，与三板同速一起向右匀速滑行——因地面光滑，无外力，整体匀速）。
路径 5 · 物理图像复盘
回顾整个过程：P 减速 a_P = f/m_P = 3 m/s²；三板加速 a_板 = f/(3m_板) = 2 m/s²。共速时间 \(t = (v_0 - v_共)/a_P = (6-2.4)/3 = 1.2\) s。校验：P 位移 \(s_P = v_0 t - \tfrac12 a_P t^2 = 7.2 - 2.16 = 5.04\) m；板位移 \(s_板 = \tfrac12 a_板 t^2 = 1.44\) m；相对位移 \(s_P - s_板 = 3.6\) m ✓。
校验通过。最终答案稳定：v_共 = 2.4 m/s，P 停在 C 板上距左端 0.6 m。
注意"三板紧贴"是题设——在实际物理中，紧贴需要满足"板间接触力 ≥ 0"。本题 P 给 A 推力 6N，三板共加速度 2 m/s² → A 推 B 的力 = (1+1)·2 = 4 N > 0 ✓，B 推 C 的力 = 1·2 = 2 N > 0 ✓，紧贴假设成立。
提示：若 P 越过 A 后，A 板没有外力作用（P 已离开 A），那么 A 还会与 BC 紧贴吗？——本题假设三板"始终紧贴"，但严格上 P 离开 A 后 A 不再被推，A 会以已获速度继续匀速 → 与 BC 共速 → 仍紧贴。结论一致。

综合反思 · 三例打通这条骨架

3 道例题难度递增，但都在重复同一条主链路：识别情境 → 选系统 → 选守恒律 → 判定共速 → 算相对位移 → 守恒方程联立。下面把"共性"和"易扣分"两件事钉死。

一、3 例题骨架共性

  • 多体系统：2 体（例 1 板 + 物，或例 2 子弹 + 物块）/ 3 体（例 1 加 P 黏合，或例 3 P + 3 板）—— 永远先问"我选哪些对象作为系统"。
  • 动量守恒优先于能量守恒：摩擦、碰撞、嵌入都让能量"漏"，但只要选对系统、找对方向，动量在那一段是守恒的。能量方程留作"算 ΔU"的工具，不要拿来当主线。
  • 共速 = 摩擦结束：所有相对滑动的判定都以"共速时刻"为分段标志点。共速之前算动量+能量，共速之后看是否分离。
  • 相对位移决定内能：ΔU = f · Δs相对。3 道例题里 Δs相对 的计算方法都不同（例 1 用 L、例 2 用动量-能量联立、例 3 用几何分段），但公式形式永远不变。
  • 方向 + 正负：板块碰撞 90% 的错都出在方向上。每写一个动量方程都要在草稿上画箭头，并写"取右为正"或"取碰前 A 方向为正"。

二、5 类学生常见卡点 + 怎么破

  1. 系统选错（例 1 把"A+B"作整体而漏掉 C+D）→ 破：先标"哪些有相对运动 / 哪些受外力"，再选系统
  2. 共速时刻识别错（例 1 误以为 v_共 = v₀ 或 0）→ 破：每段开头先写"始末态"，把共速作为终态求解
  3. 相对位移 vs 绝对位移混淆（最高频扣分点）→ 破：ΔU 里只填 Δs相对，写在公式旁明确标注
  4. 方向正负（例 1 B+D 是负速度，例 2 子弹是正速度但板初速为 0）→ 破：草稿先画"取正方向"箭头
  5. 多过程未分段（例 2 把"子弹嵌入"和"板上滑动"合成一个能量方程）→ 破：每出现一个相对速度跳变（碰撞、嵌入、分离），就必须分段
