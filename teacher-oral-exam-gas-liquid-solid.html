📋 听点总表（全部 23 锚 · ⭐ = 核心听点 · 5 项板块边界守门）
类别听点学生说到位时，话里应包含什么
气体三定律 + 温标⭐ 玻意耳定律 PV = 常数等温、定量；P 与 V 反比；P-V 图是双曲线。
⭐ 查理定律 P/T = 常数等容、定量；T 用开尔文 ⇒ P-T 图过原点的直线。
⭐ 盖-吕萨克定律 V/T = 常数等压、定量；T 用开尔文 ⇒ V-T 图过原点的直线。
开尔文与摄氏度关系TK = t°C + 273.15；过原点特性只在开尔文下成立。
绝对零度外推由 P ∝ T 外推到 P = 0 定义 T = 0 K，对应分子平均动能 = 0。
理想气体方程⭐⭐ 理想气体方程 PV = nRT能在纸上写出，并逐项指认 P / V / n / R / T 含义与单位；本板块核心成果。
普适气体常数 R = 8.314 J/(mol·K)「普适」= 对所有理想气体相同；不随气体种类变。
⭐ 结合方程 P₁V₁/T₁ = P₂V₂/T₂定量气体三量同时变化时的方便形式；T 用开尔文。
气体微观机制⭐ 玻意耳微观（数密度 + 频率）V↓ → 数密度↑ → 撞壁频率↑ → P↑；T 不变 → 单次撞壁力不变。
⭐ 查理微观（动能 + 力 + 频率）T↑ → 平均动能↑ → 撞壁力 + 频率都↑ → P↑；V 不变 → 数密度不变。
理想气体 2 假设分子点状（大小可忽略）+ 分子间无相互作用（碰撞瞬间除外）；高压低温偏离。
液体性质⭐ 表面张力本质表面分子少了上方邻居的引力束缚 → 能量更高 → 系统倾向最小化表面积。
表面收缩 · 液滴呈球形同体积下表面积最小 = 球形；自由液滴的形状由此决定。
⭐ 浸润判据（液-固 vs 液-液引力）液-固引力 > 液-液 → 浸润；反之不浸润；浸润是液 + 固组合属性，不是单独液体的属性。
毛细现象浸润液体在细管中上升（凹面）/ 不浸润液体下降（凸面）；管越细效应越显著。
固体性质⭐ 晶体四特征规则点阵 + 固定熔点 + 解理性 + 单晶各向异性。
⭐ 非晶体四特征无规则排列 + 渐变软化（无固定熔点）+ 各向同性 + 亚稳态。
⭐ 晶体固定熔点的微观机制规则点阵 → 所有分子束缚均一 → 某温度同时冲破约束 → 瞬间熔化；非晶体不均一 → 渐变软化。
⭐ 单晶 vs 多晶单晶整体规则 → 各向异性；多晶 = 大量小晶粒随机取向 → 宏观各向同性。
各向异性 vs 各向同性单晶各向异性 / 多晶 + 非晶各向同性；微观规则方向性决定。
思维方式⭐ 一套分子动理论解释三态气体 PV = nRT / 液体表面张力 / 固体晶体 — 全建立在 F-r 曲线 + 分子数密度 + 温度 ∝ 平均动能 同一套工具上。
从实验综合到方程PV = nRT 不是新原理，是三定律 + 阿伏伽德罗定律的实验综合；物理方程常源于实验。
⭐ 变质量等效定质量法气体定律要求「一定量」→ 重选研究对象，把容器内气 + 进出气打包成定质量整体；4 场景：充气 / 放气 / 抽气 / 灌装。

📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 PV = nRT 并逐项指认 P / V / n / R / T 含义」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
