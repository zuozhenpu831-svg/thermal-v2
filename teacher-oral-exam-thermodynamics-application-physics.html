📋 听点总表（capstone 跨板块 · ⭐ = 核心听点 · 含三板块连接性听点）

本融合板块是 capstone 整合级，听点 = 三个基础板块（分子动理论 / 气液固 / 第一定律）各自核心锚的跨板块组合。重点不在新增结论，而在「连接处」—— 听学生能否把不同板块的结论接到同一条综合题的解答步骤里。

来源板块听点学生说到位时，话里应包含什么
分子动理论⭐⭐ 温度 ∝ 分子平均动能T 是分子平均平动动能的标志；同 T 不同气体平均动能相同；T 升 → 平均动能升 → 撞壁更快 + 每次冲力更大。
⭐ 分子数密度 n数 = N/V质量一定时由体积单独决定；V↓ → n数↑ → 撞壁频率↑（玻意耳定律微观）。
⭐⭐ 内能微观本质U = 分子总动能 + 总势能；理想气体只算动能 ⇒ U 只由 T 与分子数决定；这是「理想气体等温 ΔU = 0」的根据。
宏观-微观桥接p 由两个微观因素共同决定（撞壁频率 × 每次冲力）；判 p / 平均动能 / n数 各自变化时先翻译到对应微观量。
气液固⭐⭐ 理想气体方程 PV = nRT能在综合题里写出并联立 p1V1/T1 = p2V2/T2；本板块综合题状态定位的第一工具。
⭐ 受力平衡定压强对活塞列平衡方程把「看不见的气体压强」用「看得见的重力 + 大气压 + 卡销力」表达出来；三步链第一步。
⭐ 三气体定律图像判别p-T 过原点直线 = 等容；V-T 过原点直线 = 等压；p-1/V 过原点直线 = 等温（玻意耳）；先看坐标轴再下判断。
温度统一用 K气体定律一律用热力学温度 K = t + 273.15；ΔT 在 K 与 ℃ 下数值相同。
变质量 → 等效定质量法充气：初态打包瓶内 + 待充共总体积；放气：末态合并；变质量化为定质量后再套定律。
第一定律⭐⭐ 第一定律方程能在纸上写出 ΔU = Q − W 并指着每项说含义与正方向（约定 W = 气体对外做功为正）；综合题三步链第三步。
⭐ 4 过程变形等容 W=0、ΔU=Q；等压 W=pΔV、Q=ΔU+W；等温 ΔU=0、Q=W；绝热 Q=0、ΔU=−W。
⭐ 状态量 vs 过程量 + 循环 ΔU=0U 状态量 / Q、W 过程量；同始末态 ΔU 必同；闭合循环回到初态 ⇒ ΔU(一周) = 0。
ΔU = CΔT 桥综合大题命题人递的内能算尺；C 为已知常数；用它把「温度变化」直接换成「内能变化」。
气体做功 W = pΔV（等压）由 F = pA、W = FΔx 推；几何上 = P-V 图过程线下方面积；做功计算必须用国际单位制 Pa 配 m3。
等温 ΔU = 0 的根据「理想气体 U 只依赖 T」—— 必须主动接回分子动理论板块。
跨板块整合
方法论		⭐⭐ 综合题三步链① 受力平衡定压强 → ② 气体方程定状态 → ③ ΔU = Q − W 算热量；物理热学综合大题的标准流水线。
⭐ 循环 / 复合两轮扫描第一轮：用 pV/T 把每个状态点的 (p, V, T) 三个参量填满；第二轮：逐段套第一定律算能量；不凭感觉判 ΔU 方向。
分段处理（突变点）过程性质突变时（卡销脱落 / 摩擦换向 / 充放气结束等）必须以临界条件为分界分段，各段各选各的定律。
边界守门
不展开		不引焓 H跨到化学侧才用，留给融合 · 化学板块 / 盖斯定律。
不引熵 S / 方向判据留给第二定律基础板块；学生主动提 → 口头肯定不展开。
不推卡诺效率 η = 1 − T2/T1循环热机题里若学生写出 → 留给进阶卡诺循环板块。
不推 PV = NkT 严格压强公式定性桥接到位即可，留给进阶微观推导。
不引等温做功对数式 W = nRT ln(V2/V1)留给进阶。
不引绝热方程 PVγ = const定性「绝热更陡」即可，留给进阶。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上完整列出综合题解答步骤（受力平衡 → 气体方程 → 第一定律）」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
