吉布斯自由能 · 基础

把 ΔH 和 ΔS 合到一起 · 用一个统一判据预测任何反应的自发性
模块名称吉布斯自由能 · 基础（基础模块）

学习目标

  1. 能陈述吉布斯自由能的定义 ΔG = ΔH − TΔS（等温等压条件下），并说出 ΔG 是状态量（由 ΔH、T、ΔS 都是状态量推出）
  2. 能用 ΔG 自发判据: ΔG < 0 自发 / ΔG = 0 平衡 / ΔG > 0 不自发（等温等压下）
  3. 能分析4 种 ΔH/ΔS 组合下的反应自发性：(a) ΔH<0 ΔS>0 任何温度自发 (b) ΔH<0 ΔS<0 低温自发 (c) ΔH>0 ΔS>0 高温自发 (d) ΔH>0 ΔS<0 任何温度不自发
  4. 能用转变温度 T转 = ΔH/ΔS 找"自发与不自发的分界点"（ΔG = 0 处）
  5. 能在化学反应案例中应用（如 CaCO₃ 分解 / NH₄NO₃ 溶解 / 燃烧反应）分析自发性
  6. 能在物质相变案例中应用（如冰融化 / 水蒸发 / 水凝固）分析自发性，并解释为什么"0°C 是冰水转变临界点"
  7. 知道本板块到 ΔG = ΔH − TΔS 公式应用为止：严格推导 + ΔG 与化学平衡常数 K 的关系（ΔG = −RT ln K）+ ΔG 与电池电动势 E 的关系（ΔG = −nFE）全部留给进阶 吉布斯自由能 板块
驱动问题

三个让"光看 ΔH"或"光看 ΔS"都答不上来的反常现象 ——

  • 同样吸热, 一个自发一个不自发: NH₄NO₃ 溶于水是吸热反应（ΔH > 0, 约 +25.7 kJ/mol），室温下却自发；CaCO₃ 分解也是吸热反应（ΔH > 0, +178.5 kJ/mol），但室温下完全不自发，必须烧到约 839°C 才发生。同是吸热, 为什么自发性完全相反？
  • 吸热却自发: 冰袋里的 NH₄NO₃ 溶于水时吸热（ΔH > 0），按"放热才自发"的旧思维它不该自发溶解 —— 但实际上室温下一倒水就溶。为什么？
  • 临界点: 冰在 0°C 以下自发结成冰，0°C 以上自发融化成水。这个 0°C 是怎么从热力学算出来的？

这三个问题的共同点 —— 单独看 ΔH 或单独看 ΔS 都不够。需要一个综合两者的判据。这就是吉布斯自由能 ΔG。

（提示: 系统倾向 ΔH < 0（能量降低）+ 系统倾向 ΔS > 0（熵增 / 无序度增大）。这两个倾向有时一致、有时冲突。冲突时谁说了算？温度 T 是调节二者权重的关键。）

本板块的"深化路径"推荐 — 学有余力的学生

本板块直接给出 ΔG = ΔH − TΔS 的公式应用，不做严格推导。如果你已掌握以下 3 个进阶板块（它们之间存在前置链 ——卡诺循环 → 状态函数 → 熵的物理意义），建议按顺序学完再回来读本板块，然后接着学进阶 吉布斯，你会从"公式应用"层升到"原理推导"层：

  1. 卡诺循环（进阶） —— 进阶链起点。严格推导卡诺效率 η = 1 − T₂/T₁
  2. 状态函数与非状态函数（进阶） —— 前置：卡诺循环。严格引入熵 S 的定义 dS = dQ可逆/T
  3. 熵的物理意义（进阶） —— 前置：状态函数。玻尔兹曼公式 S = kB ln Ω + 熵增定律严格论证

在学完这 3 个进阶板块的基础上，吉布斯自由能 · 进阶（待建）会严格推 ΔG 的来源 + ΔG 与化学平衡常数 K（ΔG = −RT ln K）+ ΔG 与电池电动势 E（ΔG = −nFE）。

没学进阶也完全可以学懂本板块（本板块自成体系，只要前置板块 第二定律基础 + 盖斯定律 学完即可）。深化是"扩展理解"，不是"必读前置"。

1

吉布斯自由能的定义 + ΔG 自发判据

▼
学习目标: 引入 ΔG = ΔH − TΔS 的定义 → 理解 ΔG 综合 ΔH（能量）和 ΔS（熵）两个倾向的物理意义 → 掌握 ΔG 自发判据（<0 自发 / =0 平衡 / >0 不自发）
建议时长: 50-60 分钟
STEP 1先自己想
5-8 分钟

不查笔记，先答 ——

  • "放热反应一定自发" 这句话对吗？给出一个反例。
  • "熵增反应一定自发" 这句话对吗？给出一个反例。
  • 既然 ΔH 和 ΔS 单独都不能判断自发性，那应该用什么综合？凭直觉猜一个形式。
先回顾（核心理论根基 ⭐）: 前置 第二定律 · 基础 给出了"自发方向 = 熵增方向"的定性判据 ——但这个判据要求对孤立系统（不与外界交换能量）。
实际化学反应通常发生在恒温恒压条件下（如大气压下、室温），系统与环境会换热、做体积功。这时系统自身的熵增不一定等于总熵增。本板块就是要给这个更现实场景下的统一判据。
STEP 2吉布斯自由能 ΔG = ΔH − TΔS

对一个等温等压反应，定义吉布斯自由能变化:

吉布斯自由能定义 ΔG = ΔH − TΔS ⭐⭐
形式: 把反应焓变 ΔH（系统能量变化，盖斯定律给出算法）和系统熵变 ΔS（第二定律 · 基础的定性概念）用温度 T 调节后组合: ΔG = ΔH − TΔS。
为什么这样组合（物理图像）: 系统有两个"倾向" —— ① 能量降低（ΔH < 0 偏好放热）+ ② 熵增加（ΔS > 0 偏好无序度升）。温度 T 决定后者的"权重"。低温下能量项 ΔH 主导；高温下熵项 TΔS 主导。
ΔG 也是状态量: 由 H、S 都是状态量 + T 是状态参数 → ΔG = ΔH − TΔS 只依赖始末态。
本板块直接给定义: 严格推导（从第二定律 + 等温等压条件推出 ΔG ≤ 0 ⇔ 自发）留给 进阶 吉布斯自由能 板块。本板块用 ΔG 公式做反应自发性判断即可。

物理意义: ΔG 是"系统在等温等压下能对外做的最大有用功"的变化量（对外做功为负 = 系统失去自由能 = 自发）。"自由"指的是"可以用来做有用功"的那部分能量。

图 1 · ΔG = ΔH − TΔS 的结构图。左侧红色方框代表能量项 ΔH（系统偏好 ΔH < 0，即放热）；中间蓝色方框代表熵项 TΔS（系统偏好 ΔS > 0，即无序度增大），温度 T 在这里作为权重；右侧紫色方框代表综合判据 ΔG（ΔG < 0 即自发）。三者由公式 ΔG = ΔH − TΔS 连接。
思考点 1 · 为什么 ΔG 比单看 ΔH 或单看 ΔS 都好？
驱动问题里的"同样吸热, 一个自发一个不自发"（NH₄NO₃ 溶解自发 vs CaCO₃ 分解不自发）—— 用 ΔG = ΔH − TΔS 重新分析它们，看 ΔG 怎么"修正"了单看 ΔH 的错误判断。
两个反应都是 ΔH > 0（吸热）+ ΔS > 0（熵增）, 都属于组合 (c) 高温自发。区别在 T转 = ΔH / ΔS:
① NH₄NO₃ 溶于水: ΔH = +25.7 kJ/mol（吸热）+ ΔS = +108 J/(mol·K)（固态规则点阵 → 离子自由扩散, 熵增很大）。T转 = +25.7×10³ / +108 ≈ 238 K = −35°C。室温 298 K > T转 → TΔS ≈ +32.2 kJ/mol > ΔH = +25.7 kJ/mol → ΔG < 0 → 自发 ✓。
② CaCO₃ → CaO + CO₂: ΔH = +178.5 kJ/mol（强吸热）+ ΔS = +160.5 J/(mol·K)（1 mol 固体 → 1 mol 固体 + 1 mol 气体, 气体让熵大幅增）。T转 = +178.5×10³ / +160.5 ≈ 1112 K = 839°C。室温 298 K < T转 → TΔS ≈ +47.8 kJ/mol < ΔH = +178.5 kJ/mol → ΔG > 0 → 不自发 ✓; 必须烧到 839°C 以上才自发。
核心收获: 同样吸热(组合 c) 的两个反应, 自发性的关键在 T 和 T转 = ΔH/ΔS 的对比。ΔG 通过把 ΔH、ΔS 和 T 统一在一个公式里, 一举解决"光看 ΔH"无法判断的所有反常案例。
在纸上: ① 算两个反应各自的 T转；② 用 T > T转 还是 T < T转 判断室温自发性；③ 一句话总结"ΔG 比 ΔH 单独看的优势"。
STEP 3ΔG 自发判据
ΔG 自发判据（等温等压下）⭐⭐

· ΔG < 0 ⇔ 反应自发（正反应方向能发生）
· ΔG = 0 ⇔ 反应处于平衡（正逆反应速率相等 / 相变临界点）
· ΔG > 0 ⇔ 反应不自发（正反应不能自发发生，但逆反应自发）

（这里的"自发"和 第二定律 · 基础 一致：不需要外力 / 不引起其他变化）
思考点 2 · ΔG = 0 的物理意义
"ΔG = 0 意味着反应处于平衡" —— 这是什么意思？反应是"停止了"还是"还在进行"？
"平衡"不是反应停止，而是正逆反应速率相等，宏观上看不到净变化。
· 相变的平衡: 0°C 时水和冰共存（在隔绝外界下，固液比例不变，但实际上分子还在两相之间穿梭，只是数量相等）。
· 化学反应的平衡: 反应进行到一定程度后，[反应物] 和 [产物] 比例稳定（正反应消耗反应物 = 逆反应产生反应物，净变化为 0）。化学平衡的定量描述（平衡常数 K）留给后续板块。
· "平衡"对应 ΔG = 0 是热力学最深的结论之一: 系统达到"最大化稳定性"的状态。
在纸上: ① 用"正逆速率相等"解释平衡 ≠ 停止；② 一句话总结"为什么 ΔG = 0 对应平衡"。
本板块对"平衡"只到定性 ΔG = 0 这一层。严格的 ΔG° = −RT ln K（标准吉布斯自由能与平衡常数 K 的关系）+ 反应商 Q 与 ΔG 的关系留给进阶。
STEP 4反思
完整的热力学判据体系

到本板块为止，热力学有了完整的判据体系：

定律判据管什么
第一定律ΔU = Q − W（能量守恒）能量"数量"是否守恒（能不能）
第二定律孤立系统 ΔS ≥ 0（熵不减）过程"方向"（会不会）
本板块（吉布斯自由能）等温等压下 ΔG < 0 ⇔ 自发现实条件下（等温等压）的统一判据

这就是为什么吉布斯自由能被称为化学反应自发性的"最终判据" —— 它对应于现实中最常见的反应条件（等温等压），并把第一定律的能量观、第二定律的熵增观统一到一个公式里。

4 种 ΔH/ΔS 组合 + 转变温度

▼
学习目标: 按 ΔH 和 ΔS 的正负号分 4 种组合，分析各组合下的自发性 → 引入转变温度概念（ΔG = 0 时分界）→ 学会从 ΔH、ΔS 数据直接判断"任何温度自发 / 低温自发 / 高温自发 / 任何温度不自发"
建议时长: 35-45 分钟
STEP 14 种组合分析

用 ΔG = ΔH − TΔS，按 ΔH（+/−）和 ΔS（+/−）的4 种组合分析自发性：

图 2 · 按 ΔH（横轴）和 ΔS（纵轴）的正负号分 4 个象限，每个象限标注 (a)(b)(c)(d) 4 种组合。绿色 (a) 左上："放热 + 熵增"双重有利，任何温度自发。橙色 (c) 右上："吸热 + 熵增"，高温自发。蓝色 (b) 左下："放热 + 熵减"，低温自发。红色 (d) 右下："吸热 + 熵减"双重不利，任何温度不自发（但逆反应任何温度自发）。(b) 和 (c) 这两组合存在"转变温度" T转 = ΔH/ΔS，是 ΔG = 0 自发与不自发的分界点。
组合ΔHΔS自发性例子
(a)< 0> 0✓ 任何温度自发2H₂ + O₂ → 2H₂O(g)（高温下还成立, 但 ΔS 会随温度变化, 简化分析）
(b)< 0< 0低温自发（T < T转）水 → 冰（−ΔH 凝固放热, −ΔS 液态→固态熵减）
(c)> 0> 0高温自发（T > T转）CaCO₃ → CaO + CO₂（+ΔH 分解吸热, +ΔS 1 mol 固体 → 1 mol 固体 + 1 mol 气体, 熵增）
(d)> 0< 0✗ 任何温度不自发（但逆反应自发）2NO → N₂ + O₂ 部分情况（吸热 + 熵减），逆反应 N₂ + O₂ → 2NO 才自发
STEP 2转变温度 T转 = ΔH/ΔS

对组合 (b) 和 (c) —— ΔH 和 ΔS 同号的情况，存在一个临界温度，让 ΔG 在这个温度恰好等于 0：

转变温度推导
设 ΔG = 0: ΔH − TΔS = 0
解 T: T转 = ΔH / ΔS ⭐
物理意义: 在 T转 时反应处于平衡。
· 对组合 (b)（ΔH < 0, ΔS < 0）: T < T转 时 ΔG < 0（自发）, T > T转 时 ΔG > 0（不自发）
· 对组合 (c)（ΔH > 0, ΔS > 0）: T < T转 时 ΔG > 0（不自发）, T > T转 时 ΔG < 0（自发）
思考点 3 · 冰水转变 — 经典的 0°C
水→冰的相变: ΔH = −6.01 kJ/mol（凝固放热）, ΔS = −22.0 J/(mol·K)（液→固熵减）。用 T转 公式算转变温度 T转。
T转 = ΔH / ΔS = −6.01 × 10³ J/mol / (−22.0 J/(mol·K)) = 273.2 K = 0.05°C ≈ 0°C。
物理意义:
· T < 0°C: 水→冰 ΔG < 0, 自发结冰 ✓
· T = 0°C: ΔG = 0, 冰水共存（"冻雨"的平衡态; 实验室中 0°C 的水和冰可以稳定共存而不变化）
· T > 0°C: 水→冰 ΔG > 0, 不自发结冰; 反过来冰→水 ΔG < 0, 冰自发融化 ✓
0°C 不是一个"任意约定" —— 它是 ΔH 和 ΔS 的热力学算出来的转变温度 ⭐⭐。
在纸上: ① 算 T转 = ΔH / ΔS（注意单位换算）；② 验证算出来 ≈ 273 K = 0°C；③ 一句话总结"为什么 0°C 是冰水转变临界点"。
单位换算: 1 kJ = 1000 J。所以 ΔH = −6.01 kJ/mol = −6010 J/mol。然后 T转 = −6010 / −22.0 = 273.2 K ≈ 273 K = 0°C ✓。
思考点 4 · CaCO₃ 分解 — 工业上为何要高温？
水泥工业的关键反应: CaCO₃(s) → CaO(s) + CO₂(g)。给定 ΔH = +178.5 kJ/mol（吸热分解）, ΔS = +160.5 J/(mol·K)（1 mol 固体 → 1 mol 固体 + 1 mol 气体，气体让熵大幅增加）。算 T转，并解释工业上为什么必须高温烧 CaCO₃。
T转 = ΔH / ΔS = +178.5 × 10³ / +160.5 = 1112 K ≈ 839°C。
物理意义:
· T < 839°C: ΔG > 0, 分解不自发 → CaCO₃ 稳定
· T = 839°C: ΔG = 0, 反应在平衡点
· T > 839°C: ΔG < 0, 分解自发 ✓
所以水泥窑必须烧到 1000°C 以上（实际工业操作 1400-1500°C，部分是为了让反应充分快），让 CaCO₃ 自发分解 → 得到 CaO 作为水泥主成分。
这就是 ΔG 在工业中的直接应用: 算 T转 可以给"反应需要多高温度才能进行"提供准确预测。
在纸上: ① 算 T转，验证 ≈ 1100 K；② 解释为什么工业生产温度高于 T转；③ 一句话总结"T转 在化工设计中的工程价值"。
✓ Card 2 检查点

读表小结: 物质相变的 T转 都是日常熟悉的"沸点 / 熔点 / 升华温度"。这些不是任意约定 —— 都是热力学按 T转 = ΔH/ΔS 算出来的。这就是为什么沸点 / 熔点是物质的固有性质。

思考点 5 · 综合应用 — N₂ + 3H₂ → 2NH₃ 工业合成氨
工业合成氨反应: N₂(g) + 3H₂(g) → 2NH₃(g), ΔH = −92.4 kJ/mol（放热）, ΔS = −198 J/(mol·K)（4 mol 气体 → 2 mol 气体, 熵减）。
(a) 这是哪种组合？
(b) 算 T转。
(c) 按 T转 结果，理论上工业上应该用什么温度？
(d) 但实际合成氨工业用约 450°C（723 K）的"中等温度"——为什么不用更低温（让 ΔG 更负）？
(a) ΔH < 0, ΔS < 0 → 组合 (b)，低温自发。
(b) T转 = ΔH / ΔS = −92400 / −198 = 467 K = 194°C。
(c) 理论上 T < 194°C 时 ΔG < 0，反应自发。低温越好。
(d) 这就是热力学 + 动力学 + 压力三重约束的拉锯战:
· 热力学（ΔG < 0）说: 低温好。但低温下反应速率极慢（动力学屏障），等不及反应完成。
· 动力学说: 升温能让反应快（活化能容易越过）, 但 ΔG 会变正。
· 工业实际: 450°C = 723 K > T转 = 467 K → 按 ΔG = ΔH − TΔS 严格算 ΔG ≈ +51 kJ/mol > 0, 表面上不自发。
· 但工业还用高压(150-300 atm)! 反应方程 N₂ + 3H₂ → 2NH₃ 是 4 mol 气体 → 2 mol 气体（ΔS < 0 = 气体减少）, 高压让气体减少方向有利（吕沙特列原理）, 把"严格 ΔG > 0"重新拉回 ΔG < 0。
· 加上催化剂（铁）大幅降低活化能让反应快达平衡。
三重组合: 中温(让动力学可行)+ 高压(让 ΔG 重新 < 0)+ 催化剂(让反应快)。这是合成氨工业的精髓 —— 单看 ΔG 公式不够, 需要"压力对 ΔG 的修正"（本板块只到等温等压下的 ΔG 公式; 高压下 ΔG = ΔG° + RT ln Q 修正留给进阶 吉布斯）。
(关于催化剂): 催化剂不改变 ΔH、ΔS、ΔG 任何一个 —— 只降低活化能让反应快。这是反应动力学的内容（留给后续板块）。
在纸上: ① 算 T转 = 467 K；② 解释为什么工业不用更低温度；③ 一句话总结"ΔG 给上限, 动力学给速度, 工业找妥协"。
