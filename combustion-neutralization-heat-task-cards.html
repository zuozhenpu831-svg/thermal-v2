学习目标

  1. 能陈述燃烧热的严格定义（25°C、1 atm 下，1 mol 可燃物完全燃烧生成稳定氧化物时放出的热量，\(\Delta H_燃 < 0\)）
  2. 能陈述中和热的严格定义（25°C、1 atm 下，强酸强碱稀溶液中和生成 1 mol 液态水时放出的热量，\(\Delta H_中 \approx -57.3\) kJ/mol）
  3. 能解释"完全燃烧"、"稳定氧化物"、"生成 1 mol H₂O" 等限定词的物理意义（为什么必须这么规约才不歧义）
  4. 能用 Q = mcΔT 算实验测得的反应热（简易量热计的核心算式）
  5. 能比较强酸强碱 vs 弱酸弱碱中和热的差异（数值 + 物理原因：弱电解质先吸热电离再中和）
  6. 能把燃烧热 / 中和热看作盖斯定律算 ΔH 的特殊形式 / 高频输入数据，二者本质同根（都建立在 H 是状态量上）
  7. 知道本板块不深入：弹式量热计的等容 \(Q_V\) vs 等压 \(Q_p\) 的修正 / 量热计工程设计 / 弱电解质电离热定量分析 —— 这些留给进阶板块或大学物理化学
驱动问题

三个化学反应能量的核心问题 ——

  • 燃料热值比较：厨房煤气（甲烷 CH₄）vs 木炭（碳 C）vs 汽油（约 C₈H₁₈），哪个"热值"更大？用什么物理量比较？单位是什么？
  • 同种反应不同热量：同样是中和反应，HCl + NaOH 和 CH₃COOH（乙酸）+ NaOH 放出的热量一样吗？为什么？
  • 实验直接测 ΔH：盖斯定律给了我们"用已知反应算未知反应 ΔH"的方法 —— 但已知反应的 ΔH 又是怎么测来的？实验室里如何直接测一个反应的 ΔH？

本板块的任务，是把两类最常用的"实验直接测 ΔH" 的形式 —— 燃烧热和中和热 —— 讲清。它们是化学手册里 ΔH 数据的主要来源，也是高频考点。

（提示：盖斯定律给"计算方法"，本板块给"实验测量方法"。两者互补：算 ΔH 时需要查数据，而数据本身是由燃烧热 / 中和热等实验测出来的。）

燃烧热 ΔH燃 — 定义 + 实验测量

▼
学习目标: 严格陈述燃烧热定义（含 4 个关键限定词）→ 理解弹式量热计的实验原理 → 学会查表算具体反应的燃烧热
建议时长: 40-50 分钟
STEP 1先自己想
5-8 分钟

不查笔记，先答 ——

  • 燃烧反应都放热（ΔH < 0）。但"放多少热"和什么变量有关？反应物的量？反应物的种类？产物的状态（CO 还是 CO₂、H₂O 液态还是气态）？
  • "煤气热值高"是什么意思？这里"热值"的单位是什么？
  • 如果你要在实验室测一个燃料的"热值"，怎么测？需要什么设备？
先回顾（核心理论根基）：
前置 热力学第一定律 板块讲过 —— 在恒压条件下（实验室空气压力下反应），系统吸收的热量恰好等于焓变：Qp = ΔH。
所以化学反应在标准实验条件（25°C、1 atm）下测得的"放热量"就是 −ΔH（注意符号：放热时系统失去热量，Qp < 0，对应 ΔH < 0）。
前置 盖斯定律 板块讲过 —— 反应焓变 ΔH 是状态量，只依赖始末态，与路径无关。所以"测某反应的 ΔH" = 找一个能控制的实验路径让始末态明确。
本板块的燃烧热和中和热，就是两类最容易控制始末态的实验测量方法。
STEP 2燃烧热的严格定义
燃烧热定义 ⭐⭐

在标准条件（25°C、1 atm）下，1 mol 可燃物完全燃烧生成稳定氧化物时放出的热量，称为该可燃物的燃烧热，记作 \(\Delta H_燃\)（单位：kJ/mol），数值始终为负（因为燃烧总是放热）。

定义里的 4 个关键限定词:

限定词含义不这么规约会怎样？
标准条件 25°C/1 atm规定测量温度和压力的"基准点"温度 / 压强不同 ΔH 会有微小差异，没有统一标准就没法对比
1 mol 可燃物归一化到"每摩尔可燃物"不规定 mol 数, 数值随反应量变化, 没法比较
完全燃烧所有可氧化的元素都被氧化到最稳定的形式(碳 → CO₂ 不是 CO; 氢 → H₂O 不是 H₂O₂; 硫 → SO₂ 不是 SO₃)"不完全燃烧"放热少, 没法区分是燃料种类不同还是燃烧程度不同
稳定氧化物产物用"标准条件下最稳定的形式": CO₂(g) / H₂O(液态) / SO₂(g)H₂O 液态 vs 气态相差约 +44 kJ/mol(蒸发吸热), 不约定就有 ±44 kJ/mol 误差
特别注意 H₂O 是液态: 燃烧热约定生成 H₂O(l) 液态，不是 H₂O(g) 气态。
· 高位热值（HHV，按 H₂O 液态算）= 化学手册中的"燃烧热"
· 低位热值（LHV，按 H₂O 气态算）= 实际工程中的"可用热值"
两者差 = 燃料中氢生成的水的蒸发热。本板块用高位热值（即标准燃烧热）。

典型燃烧热数据（标准燃烧热 ΔH燃, kJ/mol, 25°C/1 atm）:

可燃物反应方程式ΔH燃 (kJ/mol)说明
碳（石墨）C(s) + O₂(g) → CO₂(g)−393.5木炭的核心反应
氢气H₂(g) + ½O₂(g) → H₂O(l)−285.8注意 H₂O 是液态
甲烷CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)−890.3厨房煤气 / 天然气主成分
乙醇（液态）C₂H₅OH(l) + 3O₂(g) → 2CO₂(g) + 3H₂O(l)−1366.8生物燃料
乙炔C₂H₂(g) + 5/2 O₂(g) → 2CO₂(g) + H₂O(l)−1299.6焊接用气
辛烷C₈H₁₈(l) + 25/2 O₂(g) → 8CO₂(g) + 9H₂O(l)−5470汽油主成分（约值）
STEP 3弹式量热计 — 怎么实验测燃烧热

要直接测一个燃料的燃烧热，标准实验装置是弹式量热计(bomb calorimeter):

图 1 · 弹式量热计核心结构。中心由内到外三层：深灰色矩形为钢弹（厚壁耐高压钢制容器，标注在右侧），内放可燃物 + 过量 O₂；浅蓝色矩形为水浴（标注在左侧, 已知质量 m 和比热 c）；最外层灰色边框为保温外壳（标注在左上, 减少热损失）。温度计（左侧细白色矩形 + 底部红色水银柱, 标注在左侧）插入水浴测温升 ΔT；搅拌器（右侧灰色杆 + 螺旋叶片, 标注在右侧）保持水温均匀；电极（绿色 2 根, 从顶部进入钢弹）通过电火花引燃可燃物。底部蓝色公式 Q燃 = m × c × ΔT + C弹 × ΔT 表示水吸热与钢弹吸热之和。整个反应是等容过程（钢弹体积不变），测得的是 QV；严格说 QV 与 ΔH(Qp) 有微小修正(留给进阶板块)，但对大多数燃料修正很小，可近似 Q燃 ≈ ΔH燃。

测量原理:

  1. 把已知质量 \(m_可燃\) 的可燃物 + 过量 O₂ 装进钢弹
  2. 钢弹放入水浴，记录初始水温 \(T_1\)
  3. 电火花引燃可燃物，可燃物完全燃烧释放热量给水浴和钢弹
  4. 搅拌均匀后记录最高水温 \(T_2\)（ΔT = T2 - T1）
  5. 算: 热量 = 水吸热 + 钢弹吸热 = \(m \cdot c \cdot \Delta T + C_弹 \cdot \Delta T\)
  6. 除以可燃物摩尔数得每摩尔放热 = 燃烧热
思考点 1 · 为什么燃烧热是"1 mol 可燃物"而不是"1 mol O₂"？
燃烧反应是可燃物 + O₂。为什么定义燃烧热按"1 mol 可燃物"归一化, 而不是"1 mol O₂"？
关键: O₂ 是所有燃烧反应共有的反应物, 但可燃物种类不同。我们关心的是"不同燃料的能量密度", 所以应该按可燃物归一化才便于对比。
另一个原因: 不同可燃物的化学计量数中 O₂ 系数不同(CH₄ 需 2 O₂, C 需 1 O₂, 乙醇需 3 O₂...), 按"1 mol O₂"算反而是变量, 不像"1 mol 可燃物"那么干净。
这就是为什么燃烧热的方程式总是写成"1 mol 可燃物 + 配平所需 O₂ → 产物"的形式, 而 O₂ 的系数可以是 1, 2, 3, 甚至分数(如 ½, 5/2)。
在纸上: ① 写"为什么按可燃物归一化"的两个理由; ② 用 CH₄、C、C₂H₅OH 三个燃烧方程的 O₂ 系数说明"O₂ 系数不固定"。
思考点 2 · 算 CH₄ 燃烧热（用 ΔHf° 数据验证）
用盖斯定律 + 标准生成焓 ΔHf° 数据算 CH₄ 的燃烧热, 看与表格 -890.3 kJ/mol 是否一致。给定数据：ΔHf°(CH₄) = -74.8, ΔHf°(O₂) = 0（稳定单质）, ΔHf°(CO₂) = -393.5, ΔHf°(H₂O,l) = -285.8 kJ/mol。
反应: CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)。
用盖斯定律的"反应焓变 = Σν·ΔHf°(产物) − Σν·ΔHf°(反应物)" 公式:
ΔH燃 = [1×(-393.5) + 2×(-285.8)] − [1×(-74.8) + 2×0]
= [(-393.5) + (-571.6)] − [(-74.8) + 0]
= -965.1 - (-74.8)
= -890.3 kJ/mol ✓
与表格完全一致!
核心收获: 燃烧热不是新原理, 它是盖斯定律算 ΔH 的特殊情形(始态 = 可燃物 + O₂, 末态 = 完全燃烧产物)。手册上的燃烧热数据可以由 ΔHf° 数据算出, 也可以由实验测得 —— 两种来源互相验证。
在纸上: ① 写出 CH₄ 燃烧的方程式; ② 用 ΔHf° 公式算; ③ 验证与表格 -890.3 一致; ④ 一句话总结"燃烧热与盖斯定律的关系"。

中和热 ΔH中 — 定义 + 实验测量

▼
学习目标: 严格陈述中和热定义（含"强酸强碱"、"生成 1 mol 液态水"两个关键约定）→ 理解强酸强碱中和热 ≈ -57.3 kJ/mol 的本质（H⁺ + OH⁻ → H₂O）→ 解释弱电解质对中和热的影响 → 学会简易量热计 Q = mcΔT
建议时长: 40-50 分钟
STEP 1先自己想
5 分钟

不查笔记，先答 ——

  • 酸碱中和反应"放热"的根本原因是什么？反应中哪一步真正在放热？
  • HCl + NaOH → NaCl + H₂O 和 H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O 这两个反应的"中和热"应该一样吗？
  • 如果换成弱酸(如乙酸 CH₃COOH)和强碱 NaOH 中和，放出的热量会和强酸强碱一样吗？

给一个核心猜测：中和反应的本质是 H⁺ + OH⁻ → H₂O，所以中和热应该按"生成 1 mol 水"归一化。强酸强碱的中和热基本相同；弱酸弱碱因为先要"电离"才能参与反应，所以净放热会少一些。Card 2 把这点讲清。

STEP 2中和热的严格定义
中和热定义 ⭐⭐

在标准条件（25°C、1 atm）下，强酸强碱稀溶液发生中和反应生成 1 mol 液态水时放出的热量，称为中和热，记作 \(\Delta H_中\)。

关键数值: 对所有强酸强碱稀溶液，\(\Delta H_中 \approx -57.3\) kJ/mol。

定义里的 3 个关键约定:

约定含义为什么这么规定？
强酸强碱稀溶液强酸 = HCl / H₂SO₄ / HNO₃ 等完全电离; 强碱 = NaOH / KOH / Ba(OH)₂ 等完全电离; "稀"= 浓度 ≤ 1 mol/L强电解质 100% 电离, 反应本质是 H⁺ + OH⁻ → H₂O, 不需要先电离吸热
生成 1 mol 液态水归一化到"每摩尔生成水"所有酸碱中和的化学计量不同(1 H₂SO₄ + 2 NaOH 生成 2 mol H₂O), 按"水的 mol"归一化才统一
液态水产物 H₂O 是液态(实验在水溶液中)实验温度 25°C 远低于沸点 100°C, 水必然是液态; 与燃烧热"H₂O 液态"约定一致

本质 — 离子方程式 ⭐:

H⁺(aq) + OH⁻(aq) → H₂O(l)　　\(\Delta H = -57.3\) kJ/mol

这是所有强酸强碱中和反应的共同本质。所以无论反应是 HCl + NaOH 还是 H₂SO₄ + 2NaOH 还是 HNO₃ + KOH，归一化到"每生成 1 mol 水"放出的热量都是 -57.3 kJ/mol(在常温稀溶液下)。

典型例 — H₂SO₄ + 2NaOH: 反应方程式 H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O, ΔH = 2 × (-57.3) = -114.6 kJ/mol H₂SO₄(因为生成 2 mol 水)。中和热 ΔH中 仍是 -57.3 kJ/mol (按 mol H₂O 算)。

STEP 3弱酸弱碱的中和热 — 电离吸热的修正

如果换成弱酸(如乙酸 CH₃COOH, 简写 HAc)或弱碱(如氨水 NH₃·H₂O), 中和热会和强酸强碱一样吗？

弱酸弱碱中和反应的"两步走"分析
强酸强碱(参考): HCl + NaOH → NaCl + H₂O。HCl 和 NaOH 在水中已经 100% 电离 → 反应本质就是 H⁺ + OH⁻ → H₂O 放热 57.3 kJ/mol。
弱酸 + 强碱 (HAc + NaOH): 反应可看作两步 ——
· 步骤 ①: 弱酸先电离 HAc → H⁺ + Ac⁻ (微吸热, ΔH₁ ≈ +0.4 ~ +2 kJ/mol)
· 步骤 ②: 然后 H⁺ + OH⁻ → H₂O (放热 57.3 kJ/mol)
· 总反应 ΔH = ΔH₁ + (-57.3) ≈ -55 ~ -57 kJ/mol(略小于 57.3 的绝对值)
结论 ⭐: 弱酸弱碱中和热的绝对值 < 57.3 kJ/mol。差额 = 弱电解质电离吸热 ΔH电离。
这就是为什么实验测中和热必须用强酸强碱 —— 排除弱电解质电离的"干扰", 让数值准确反映 H⁺ + OH⁻ → H₂O 的本质放热。
思考点 3 · 为什么中和热按"生成 1 mol H₂O" 而不是"1 mol 酸"？
用 H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O 这个例子。如果按"1 mol 酸"算 ΔH = -114.6 kJ/mol, 按"1 mol 水"算 ΔH = -57.3 kJ/mol。为什么标准约定是"1 mol 水"？
关键: 中和反应的本质是 H⁺ + OH⁻ → H₂O, 每生成 1 mol 水放热 57.3 kJ。这是普遍规律(对所有强酸强碱都成立)。
如果按"1 mol 酸"算, H₂SO₄ 是 -114.6, HCl 是 -57.3, HCl + H₂SO₄ 混合时数值各异 —— 不便于比较和综合。
按"1 mol H₂O"算, 所有强酸强碱中和热都 ≈ -57.3 kJ/mol —— 统一一个数, 反映 H⁺ + OH⁻ → H₂O 的本质。
这就是为什么化学手册中"中和热"按"每摩尔生成水"列, 而不是"每摩尔酸"。
在纸上: ① 写"为什么按生成水归一化"; ② 用 H₂SO₄ + 2NaOH 计算 "按 mol 酸" vs "按 mol 水" 的数值, 看哪个更合理。
思考点 4 · HCl + NaOH vs HAc + NaOH 哪个放热多？
取等量的 NaOH(过量), 分别用 1 mol 的 HCl 和 1 mol 的 HAc(乙酸)中和。两种情况放出的热量一样吗？哪个多？为什么？
不一样, HCl 中和放热更多。
· HCl + NaOH → NaCl + H₂O, ΔH ≈ -57.3 kJ/mol (强酸 + 强碱, 直接 H⁺ + OH⁻ → H₂O 反应)。
· HAc + NaOH → NaAc + H₂O, ΔH ≈ -55.6 kJ/mol(数值近似估算; 弱酸先电离吸热, 净放热略少)
差额约 +1.7 kJ/mol = HAc 电离的吸热消耗了一部分能量。
实验验证: 实测确实是 HCl + NaOH 放热略多于 HAc + NaOH —— 这是判断"是否强酸"的一个间接方法(虽然不如电导率法准确)。
在纸上: ① 用"两步走"分析 HAc + NaOH 的总 ΔH; ② 解释为什么 HCl 中和放热更多; ③ 一句话总结"中和热数值差异 = 弱电解质电离吸热"。
弱电解质电离吸热的定量分析(电离平衡常数 Ka, 电离度 α 等)留给"水溶液化学"板块(待建)。本板块只用定性"|ΔH弱| < |ΔH强|"。
STEP 4简易量热计 — Q = mcΔT 直接测中和热

中和反应在水溶液中进行, 比燃烧热实验简单得多 —— 用一个保温杯就能测:

图 2 · 简易量热计核心结构。中间灰色矩形为保温杯（玻璃 / 塑料 / 不锈钢真空保温），用引线标注"减少热损失"。保温杯内浅蓝色为酸碱混合反应溶液（标注质量 m / 比热 c ≈ 4.18 J/(g·K) / 温升 ΔT = T₂ − T₁）。垂直白色矩形为温度计（左侧引线标注），底部红色为温度计水银柱。深灰色直线为搅拌器（右侧引线标注）。右侧白色公式盒标注核心算式 Q = m·c·ΔT 和"中和热 = Q / nH₂O"。底部小字注明 nH₂O = 生成液态水的摩尔数。

测量原理（核心算式 Q = mcΔT）⭐:

  1. 取已知体积 + 已知浓度的强酸(如 50 mL 1 mol/L HCl)和强碱(如 50 mL 1 mol/L NaOH), 温度都是 T₁
  2. 快速混合, 测最高温度 T₂, 算 ΔT = T₂ − T₁
  3. 溶液总质量 \(m \approx 100\) g(稀溶液近似为水, 密度 1 g/mL)
  4. 溶液比热 \(c \approx 4.18\) J/(g·K)(近似水的比热)
  5. 反应总放热 \(Q = m \times c \times \Delta T\)
  6. 生成的水的摩尔数 \(n_{H_2O} = 0.05\) mol(因为 50 mL × 1 mol/L = 0.05 mol H⁺ 完全中和 0.05 mol OH⁻ 生成 0.05 mol H₂O)
  7. 中和热 = \(-Q / n_{H_2O}\)(负号: 系统放热 → ΔH < 0)
实验典型结果: 用 50 mL 1 mol/L HCl + 50 mL 1 mol/L NaOH, ΔT 约 6.85 K (从 25°C 升到 31.85°C)。
算: \(Q = 100 \times 4.18 \times 6.85 = 2863\) J = 2.863 kJ;
中和热 = \(-2.863 / 0.05 = -57.3\) kJ/mol ✓ 与理论值吻合。

综合应用 — 燃烧热 / 中和热与盖斯定律的连接

▼
学习目标: 把燃烧热 / 中和热放到盖斯定律的"大图"里 — 它们是 ΔH 的特殊形式 + 化学手册数据的主要来源 — 学会用燃烧热数据反算其他反应 ΔH
建议时长: 25-35 分钟
STEP 1燃烧热 / 中和热与盖斯定律的关系
图 3 · 燃烧热 / 中和热与盖斯定律的关系。左侧红色方框为"实验直接测量"（本板块提供 ΔH 数据的实验来源：弹式量热计测燃烧热, 简易量热计测中和热）；中间橙色方框为"盖斯定律"（已学板块提供算法：用已知 ΔH 算未知 ΔH）；右侧绿色方框为"任意反应的 ΔH"（应用：包括无法直接测的反应）。三者构成完整的反应焓变 ΔH 体系: 输入 → 算法 → 输出。

关键认识:

  • 燃烧热 / 中和热都是 ΔH 的特殊形式(始末态明确, 易于实验测量)
  • 它们是化学手册中 ΔH 数据的主要来源
  • 盖斯定律的"算 ΔH"工作 必须有数据输入 — 这些数据要么是 ΔHf°(进阶板块讲), 要么是燃烧热, 要么是中和热
  • 所以本板块和盖斯定律是互补的: 一个给"数据", 一个给"算法"
STEP 2综合应用 — 用燃烧热反算其他反应 ΔH
思考点 5 · 用燃烧热数据算 CH₄ → CO + 2H₂ 的 ΔH
考虑工业上的"甲烷蒸汽重整反应"的简化版: CH₄(g) + ½O₂(g) → CO(g) + 2H₂(g) (合成气制取的核心反应之一)。
给定燃烧热数据(都是 kJ/mol):
· ΔH燃(CH₄) = −890.3 (生成 CO₂ + 2H₂O(l))
· ΔH燃(CO) = −283.0 (生成 CO₂)
· ΔH燃(H₂) = −285.8 (生成 H₂O(l))
用盖斯定律算 CH₄ + ½O₂ → CO + 2H₂ 这个反应的 ΔH。
把目标反应"凑"出来的代数:
① CH₄ 燃烧: CH₄ + 2O₂ → CO₂ + 2H₂O(l), ΔH₁ = -890.3
② CO 燃烧(反向): CO₂ → CO + ½O₂, ΔH₂ = +283.0
③ 2H₂ 燃烧(反向): 2H₂O(l) → 2H₂ + O₂, ΔH₃ = 2 × (+285.8) = +571.6

加 ①+②+③:
· 反应物侧 CH₄ + 2O₂ + CO₂ + 2H₂O(l)
· 产物侧 CO₂ + 2H₂O(l) + CO + ½O₂ + 2H₂ + O₂
· 消去两侧相同的 CO₂、2H₂O(l) → 反: CH₄ + 2O₂; 产: CO + ½O₂ + 2H₂ + O₂ = CO + 3/2 O₂ + 2H₂
· 再消去两侧 O₂: CH₄ + (2 - 3/2) O₂ → CO + 2H₂ → CH₄ + ½O₂ → CO + 2H₂ ✓ (得到目标反应)

ΔH目标 = ΔH₁ + ΔH₂ + ΔH₃ = -890.3 + 283.0 + 571.6 = -35.7 kJ/mol(微放热)

核心收获: 用"3 个燃烧反应的代数组合"算出目标反应 — 这是盖斯定律的标准应用。这种"用燃烧热反算其他反应 ΔH"的技巧, 是高中化学反应能量计算的核心方法之一。
在纸上: ① 写 3 个燃烧反应; ② 调整方向 + 系数让它们相加得目标反应; ③ 加上 3 个 ΔH 得 -35.7 kJ/mol; ④ 一句话总结"燃烧热数据 + 盖斯定律 = 算任意 ΔH"。
这个反应叫"甲烷部分氧化"(partial oxidation), 是工业生产合成气(syngas = CO + H₂)的方法之一, 比"完全燃烧到 CO₂"对化工更有用(合成气进一步合成甲醇 / 氨等)。
STEP 3实验设计的关键问题 — 误差来源

无论是弹式量热计还是简易量热计, 实验测量都有误差。常见误差来源:

  • 热损失: 反应放出的热量没全部加热水, 有部分散到杯外 / 容器壁 / 空气中 → 测得的 ΔT 偏小 → 测得的 ΔH 绝对值偏小
  • 仪器吸热: 容器壁 / 温度计 / 搅拌器本身也会吸热 → 测得的 ΔT 偏小
  • 反应不完全 / 副反应: 如燃料没完全燃烧, 产生 CO 而不是 CO₂; 中和反应有副离子反应等
  • 温度读数延迟: 温度计需要一段时间达到溶液温度
  • 溶液比热不严格 = 水的比热: 稀溶液近似为水, 浓溶液偏差较大

为了减小误差: 用保温好的容器(真空保温瓶 / 双层保温杯) + 快速混合 + 多次实验取平均。

误差的定量分析(校准系数 / 水当量 / 卡列度量等)留给大学物理化学实验。本板块只需定性理解误差来源。
