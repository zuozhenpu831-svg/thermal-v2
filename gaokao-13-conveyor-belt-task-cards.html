STEP 0 · 13 题位 · 传送带骨架介绍

在做任何一道题之前，先用 5 分钟把这一题位的"骨架长什么样"刻进脑子。下面 4 段是骨架本身，不是技巧——技巧在 3 道例题里。

骨架 1 · 传送带 = 摩擦驱动 + 多过程分段
题面里出现 "传送带 / 履带 / 行李 / 物块滑上" 等字样，并且物体与带之间存在 相对滑动或趋于共速，整套问题最终在 "加速段 / 减速段 / 共速段"或"共速后分离段" 之间切换——这就是 13 题位的标准骨架。
"水平带 / 倾斜带 + 物块（行李）"的组合是该骨架最常用的载体：摩擦力是 带对物的驱动力或阻力（方向取决于 v_物 与 v_带 的比较），重力沿斜面分量可能与摩擦同向也可能反向——这正是产生 4-6 种亚类的根源。
骨架 2 · 高考 13 题位高频骨架（传送带 + 动能定理类）
据近 5 年全国卷与新高考统计：13 题位的 10-13 分压轴题中，"传送带 + 动能定理"骨架约占动能定理类题目的 30%，2021 全国乙卷 13 题（机场行李传送带，11 分）是经典母题。出题方式以 "2 小问" 为主：第 (1) 问考"加速段位移 / 加速度"，第 (2) 问考"几何约束（带长够不够） / 共速后是否分离 / μ 取值范围"。
本任务卡选取的 3 道例题分别对应该骨架的三类典型变体：水平带 + 物块同向滑上慢于带（例题 1）/ 倾斜带 + μ vs tanθ 临界（例题 2，基于 2021 全国乙 13 改编）/ 水平带 + 反向运动 + 反复（例题 3，自创变式）。三例覆盖率约占 13 题位历年真题骨架的 65% 以上。
骨架 3 · 识别关键字（题面里只要见到就立即对号入座）
「传送带 / 履带 / 行李」：标志性载体 → 立即写"取带运动方向为正"+ 标 v_带 在草稿上。
「轻放上 / 静止放上」：物块初速 v_物 = 0 → 比 v_带 慢（无论带运动方向）→ 摩擦力沿带运动方向作用在物块上。
「以 v₀ 同向 / 反向滑上」：先比 v_物 与 v_带 大小 + 方向 → 物块相对带的运动方向 → 摩擦力方向（与相对运动方向相反）。
「共速 / 同速 / 不滑离」：物体与带达到同一速度 → 滑动摩擦力即将消失 → 转为静摩擦或脱离 → 必须独立分析"共速后是否继续被带"。
骨架 4 · 核心判据 3 条（传送带题的"通关密钥"）
判据 A · v_物 vs v_带 比较定方向：摩擦力的方向永远沿带表面、与物相对带的运动方向相反。具体：v_物 < v_带（同向）→ 摩擦力沿带运动方向推物；v_物 > v_带 → 摩擦力沿带反方向阻物；v_物 与 v_带 反向 → 摩擦力沿带方向（推 / 阻取决于哪个为正方向约定）。每一段都要重新比较一次，方向变了加速度就变。
判据 B · 共速时刻：v_物 = v_带 的瞬间 = 滑动摩擦力即将消失的瞬间 = 一段过程的终点。共速 ≠ 终态——共速后必须独立判断"静摩擦是否足以维持共速"，若不能则进入新一段（例题 2 的核心陷阱）。
判据 C · 倾斜带 μ 与 tanθ 比较：倾斜带共速后物块是否被带继续带走，取决于 μ 与 tanθ 的比较——① μ ≥ tanθ：静摩擦足以平衡重力分量 → 共速后随带匀速；② μ < tanθ：静摩擦不足以平衡重力分量 → 共速后物块下滑（即使带继续向上）。这是 2021 全国乙 13 的核心判据，也是 13 题位最容易遗漏的一步（约占该题位扣分的 25%）。

例题 1 · 水平传送带 + 物块同向慢于带

▼
学习目标：以"水平带 + 物块同向慢于带"为骨架，掌握"v_物 与 v_带 比较 → 摩擦方向 → 动能定理求位移"的标准三步法
建议时长：70 分钟（速复习 8 + 题面 5 + 自己做 20 + 对照思考 22 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

下面 4 条公式 + 2 条判据是例题 1 用到的全部工具。先合上页面回忆，再对照。

摩擦 + 动能定理速查清单
滑动摩擦力：\(f = \mu N\)（方向沿接触面，与"物相对带"的运动方向相反）
水平带 N：\(N = mg\)（无其他竖直力）
动能定理：\(W_{\text{合}} = \Delta E_k = \tfrac12 m v^2 - \tfrac12 m v_0^2\)
匀变速运动学：\(v^2 - v_0^2 = 2as\)（同方向上）
判据 · 共速时刻=v_物 = v_带 = 滑动摩擦力即将消失
判据 · 共速后=水平带共速后无外力扰动 → 物块随带匀速（静摩擦为零，因为不需要力维持匀速）
STEP 2题面
水平带 + 物块同向慢于带
2 小问
水平传送带以恒定速度 \(v_{\text{带}} = 4\) m/s 向右匀速运动。某时刻，将一质量 \(m = 1\) kg 的物块以初速度 \(v_0 = 2\) m/s 向右（与带同向） 轻放在传送带左端。物块与传送带之间的动摩擦因数 \(\mu = 0.3\)，取 \(g = 10\) m/s²。

(1) 求物块从放上瞬间到与传送带共速时，物块相对地面的位移。[约 5 分]
(2) 若传送带的有效长度 \(L = 5\) m，物块从最左端放上后，能否在带上加速到与传送带共速？若能，加速段结束时距左端多远？[约 6 分]
SVG 1 · 物块以 v₀ = 2 m/s 同向轻放在 v_带 = 4 m/s 的传送带上。比较 v_物 与 v_带 → 摩擦力沿带方向（向右）推物加速。

STEP 4方法路径
4 步串起整题
路径 1 · 比较 v_物 与 v_带（定摩擦方向）
v_物 = 2 m/s 向右，v_带 = 4 m/s 向右。物块比带慢 → 物块相对带向左滑动 → 滑动摩擦力作用在物块上、方向沿带表面向右（与相对运动方向反，即沿带运动方向推物）。
"我判断摩擦力沿带向右推物加速。请反问我：如果我把'物块速度方向'和'物块相对带的速度方向'搞混了，会导致摩擦力方向写反吗？不要直接给我结论。"
取向右为正。
v_物 = 2 m/s < v_带 = 4 m/s（同向）→ 物块相对带向左 → 摩擦力 f 沿带向右（推物加速）。
f = μmg = 0.3 × 1 × 10 = 3 N
路径 2 · 求加速度 a（牛顿第二定律）
水平方向只有一个力——摩擦力 f = 3 N。由 \(F_{\text{合}} = ma\) 直接得 a。注意：加速度方向 = 摩擦力方向 = 沿带向右。
"请帮我把摩擦力代入牛顿第二定律。不要替我算数值，只验证方程的方向约定是否一致。"
水平方向：\(f = ma\) → \(a = f/m = 3/1 = 3\) m/s²（向右）
路径 3 · 求共速位移（运动学 / 动能定理）
物块从 v₀ = 2 m/s 加速到 v_带 = 4 m/s，加速度 a = 3 m/s²。两种等价方法：① 运动学公式 \(v^2 - v_0^2 = 2as\)；② 动能定理 \(fs = \tfrac12 m v_{\text{带}}^2 - \tfrac12 m v_0^2\)。两种各算一遍互相验证。
"请帮我用动能定理列方程，并与运动学公式得到的位移做对比。不要直接告诉我答案。"
方法 i（运动学）：\(v_{\text{带}}^2 - v_0^2 = 2as_1\) → \(16 - 4 = 2 \times 3 \times s_1\) → \(s_1 = 2\) m。
方法 ii（动能定理）：\(f \cdot s_1 = \tfrac12 m v_{\text{带}}^2 - \tfrac12 m v_0^2\) → \(3 s_1 = \tfrac12 \times 1 \times 16 - \tfrac12 \times 1 \times 4 = 6\) → \(s_1 = 2\) m。
两法一致，第 (1) 问答案：\(\boxed{s_1 = 2 \text{ m}}\)（方向向右）。
路径 4 · 几何验证（第 (2) 问）
第 (1) 问算出加速段位移 \(s_1 = 2\) m。带长 \(L = 5\) m。比较：\(s_1 = 2 < L = 5\) → 物块在带上加速到共速的位移没有超出带长 → 能在带上加速到共速。共速时物块距左端正好是 \(s_1 = 2\) m。
共速后物块速度等于带速 → 滑动摩擦力消失 → 物块以 4 m/s 匀速向右继续走，直到走完剩余 \(L - s_1 = 3\) m 滑离右端。
"请反问我：如果带长 \(L\) 改为 1.5 m（即比 s₁ 还短），物块到达右端时的速度该怎么算？不要替我做这个变式。"
由 \(s_1 = 2\) m < \(L = 5\) m → 能在带上加速到共速，加速段结束时距左端 \(s_1 = 2\) m。
第 (2) 问答案：\(\boxed{\text{能；距左端 2 m}}\)。
（共速后物块匀速 4 m/s 走完剩余 3 m，但题目只问到加速段结束位置，不需要计算后续。）
提示：本题"算出位移 < 带长"是常见情形。若题面将 L 设为 1 m 这类"小于 s₁"的值，则物块未达共速就滑离右端——此时要用 \(v^2 = v_0^2 + 2aL\) 求出右端速度，不再有"共速"这一段。这是 13 题位最高频的变式陷阱之一。

例题 2 · 倾斜传送带 + μ vs tanθ 临界（基于 2021 全国乙 13 改编）

▼
学习目标：以"倾斜带 + 行李轻放"为骨架，掌握"共速后判 μ vs tanθ → 决定是继续随带还是下滑"这一 13 题位最易遗漏的步骤
建议时长：80 分钟（速复习 10 + 题面 5 + 自己做 25 + 对照思考 25 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

本题与例题 1 的区别：倾斜带引入重力沿斜面分量 \(mg\sinθ\)，与摩擦力同沿斜面方向。两者大小关系决定加速度方向 + 共速后是否继续被带。

倾斜带速查清单
正压力：\(N = mg\cosθ\)（垂直斜面方向）
最大滑动摩擦：\(f = \mu N = \mu m g \cosθ\)（沿斜面，方向由 v_物 vs v_带 决定）
重力沿斜面分量：\(G_\parallel = mg\sinθ\)（永远指向斜面下方）
沿斜面动力学方程：\(f \pm mg\sinθ = ma\)（符号取决于两者方向）
判据 · 共速后随带条件：\(\mu \ge \tanθ\)（静摩擦足以平衡重力分量，物随带匀速向上）
判据 · 共速后下滑条件：\(\mu < \tanθ\)（静摩擦不足以平衡重力分量，物相对带向下滑 → 摩擦方向反转）
STEP 2题面
倾斜带向上 + 行李轻放
如图所示，机场的倾斜传送带与水平方向夹角 \(θ = 37°\)（取 \(\sinθ = 0.6, \cosθ = 0.8\)），传送带以恒定速率 \(v_{\text{带}} = 4\) m/s 沿斜面向上匀速运动。在传送带的底端将一件质量 \(m = 2\) kg 的行李 轻轻放上（初速度为零）。行李与传送带之间的动摩擦因数 \(\mu = 0.8\)，传送带有效长度 \(L = 10\) m，取 \(g = 10\) m/s²。

(1) 判断行李在到达带速之前的加速过程中所受摩擦力的方向，并求该阶段行李的加速度 \(a_1\)。[约 4 分]
(2) 求行李从底端到顶端所需的总时间，并分析行李在到达带速后是否继续被带向上加速 / 匀速。[约 7 分]
SVG 2 · 倾斜带 θ = 37° 沿斜面向上以 4 m/s 匀速。行李轻放（v = 0）→ 物相对带向下 → 摩擦力沿带向上推物。重力沿斜面分量 mg sinθ 与摩擦力反向。

综合反思 · 三例打通传送带骨架

3 道例题难度递增，但都在重复同一条主链路：取正方向 → 比较 v_物 与 v_带 → 定摩擦方向 → 列动能定理或牛顿第二定律 → 共速判定 → 几何/μ 验证。下面把"共性"和"易扣分"两件事钉死。

一、3 例题骨架共性

  • v_物 vs v_带 是一切起点：每段开头先比较两者的大小 + 方向，由此决定摩擦力方向。摩擦力方向不是"凭感觉"或"看物块速度方向"，而是看相对速度（v_物 - v_带）的符号。
  • 共速 = 一段过程的终点，但不一定是终态：水平带共速后无外力 → 物随带匀速（例题 1、3）；倾斜带共速后必须比 μ 与 tanθ（例题 2）。共速后必须独立分析下一段。
  • "v_物 = 0" ≠ "物块停下不动"：当带仍在运动时，v_物 = 0 只是瞬时状态，摩擦仍存在并可能继续推物（例题 3）。判断停下要看"v_物 = v_带 = 0"或"物块离开带"。
  • 动能定理与运动学是双工具：水平带 + 单段过程用运动学最快（例题 1）；倾斜带 + 多力作用用动能定理或沿斜面牛二式（例题 2）；多段过程必须分段列式（例题 3），不能一气合并。
  • 几何约束（带长 L）的双重作用：① 检验"加速段能否完成"（例题 1、2 的 s₁ vs L）② 决定"共速后能走多远"（例题 3 的剩余 1.5 m）。所有传送带题都要至少做一次"位移与 L 的比较"。

二、5 类学生常见卡点 + 怎么破

  1. 摩擦方向判错（凭物速方向而非相对速度）→ 破：每段开头都先在草稿上写"v_物 - v_带 = ?"，由符号定方向
  2. 共速误判终态（以为共速后题就结束）→ 破：每段结束后强制问自己"共速后还有什么力？能否维持共速？"
  3. v_物 = 0 误以为摩擦消失（例题 3 的核心陷阱）→ 破：写"摩擦消失的判据是 v_物 = v_带，不是 v_物 = 0"
  4. 倾斜带遗漏 μ vs tanθ（例题 2 的核心评分点）→ 破：每道倾斜带题最后强制写一行"μ ___ tanθ → 共速后 ___ "
  5. 多段过程不分段（例题 3 把三段合一）→ 破：每出现一次"速度变零 / 共速 / 离开带"，就强制断段重写

三、13 题位 10-12 分钟应试建议

📌 13 题位定位：前两道实验题约用 20 min；选择 + 多选约 25 min；13 题位是"必须稳拿"的中档大题，给 10-12 min 是合理的。

📌 3 分钟读题：识别关键字（水平 / 倾斜 / 同向 / 反向 / 轻放）→ 决定走哪类骨架（例 1 / 例 2 / 例 3 还是其他变体）。读完题不动笔最高效。

📌 2 分钟搭草稿：示意图 + 取正方向箭头 + 标 v_物、v_带 数值与方向 + 标 L。这 5 件事不画，后面必错。

📌 5-7 分钟列方程 + 求解：每段开头先写"v_物 - v_带 = ?"定摩擦方向；用动能定理或牛二式求加速度 + 位移；倾斜带必写 μ vs tanθ 一行；最终结果给单位 + 方向。
