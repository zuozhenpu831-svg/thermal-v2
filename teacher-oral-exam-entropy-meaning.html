📋 听点总表（全部 17 锚 · ⭐ = 核心听点 · 3 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义⭐ 操作定义 vs 本质定义ΔS = Qrev/T 是操作定义（怎么算），S = k ln Ω 是本质定义（是什么 = 微观状态数的量度）；两层互补、缺一不可。
微观状态数 ΩΩ = 系统所有可能的微观排布数；玩具模型 Ω = XN（乘法原理：独立选 + 分步）。
玻尔兹曼常数 kk = R/NA ≈ 1.38 × 10⁻²³ J/K；把宏观（mol、R）与微观（粒子、k）连起来。
关键结论⭐⭐ 玻尔兹曼公式 S = k ln Ω能在纸上写出，逐项指认（S 熵 / k 玻尔兹曼常数 / Ω 微观状态数）；本板块本质定义之冠。
⭐ 自由膨胀 ΔS = Nk ln 2微观（状态数比 2N）与宏观（绕道可逆等温 Qrev/T = nR ln 2 = Nk ln 2）两条路径殊途同归。
⭐⭐ 熵增定律 ΔSuniverse ≥ 0把"宇宙 = 系统 + 外界"作为整体；下一板块（吉布斯自由能）的理论根基。
⭐ 两条路径殊途同归微观数状态与宏观 Qrev/T 独立走到同一答案；玻尔兹曼公式的"出生证明"。
熵是广延量 → 必须取 lnΩAB = ΩA·ΩB（乘法）+ SAB = SA + SB（广延）⇒ f(xy) = f(x) + f(y)，连续解只有对数。
热流方向由 ΔS = Q/T 推出ΔS = Q(1/Tc − 1/Th) > 0；反向被熵增禁止。
第二定律⭐ 第二定律三种等价表述克劳修斯 / 开尔文-普朗克 / 熵表述 三者等价，可两两互推（反证法接反向卡诺机）。
摩擦创造熵含摩擦循环 ΔSuniverse = Qf/Tc > 0；耗散必产熵。
思维方式时间之矢"过去 / 未来"方向由熵增定义；力学方程不分正反，热力学才有箭头。
第二定律是概率性的不是绝对禁止，反向概率小到天文级别（玻尔兹曼洞见）。
信息熵 H 与焓 H 同字母H = −Σ pi ln pi（信息熵）与焓 H 同字母、不同含义；语境分辨。
符号约定Q 取正值的符号约定方向通过 ±/流入流出体现（与卡诺、状态函数板块一致）。

📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 S = k ln Ω 并逐项指认每项含义」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
