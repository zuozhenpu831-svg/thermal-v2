1
状态图像互判 + 微观解释 · 刚性储气瓶昼夜温差
P5
基础
题目
高原科考站的一只刚性金属储气瓶内封有一定质量的理想气体. 从凌晨到正午, 瓶内气体由状态 A 变为状态 B. 在 p-T 图像上, A、B 两点位于同一条延长线过原点的直线上, 且 TB > TA. 下列说法正确的是 ( )

A. 该过程中气体体积增大
B. 该过程中气体分子数密度减小
C. 该过程中气体分子的平均动能增大
D. 单位时间内撞击单位面积瓶壁的分子数增多, 但每次碰撞的平均冲力不变

📊 状态图像三对照 · 题中"p-T 图过原点直线"= (a) 等容线 ⇒ V 不变. (b)(c) 仅作对照

📚 ① 知识点梳理
  • 图像判别铁律 (锚: 气体、液体与固体): p-T 图上过原点的直线 = 等容线 (p/T = 常数 ⇔ V 不变); 同理 V-T 图过原点直线 = 等压线
  • 微观三量桥接 (锚: 分子动理论):
    • 分子数密度 n = N/V — 由体积决定 (质量一定时)
    • 分子平均动能 — 只由温度 T 决定 (T 是平均动能的标志)
    • 宏观压强 p ∝ 碰撞频率 × 每次碰撞平均冲力 — 两个微观因素共同决定
  • 跨板块综合: 先用气体图像判过程类型, 再翻译成分子动理论微观语言
🧭 ② 解题思路与方法
  • 拆题: 已知 = p-T 图过原点直线段 A→B (升温) + "刚性瓶"; 求 = 宏观/微观各量变化判断
  • 选路径: 三步翻译法 — ① 图像定过程(等容) → ② 宏观三量 p, V, T 定变化 → ③ 微观三量逐项翻译
  • 双重印证: 题面"刚性金属瓶"与图像"过原点直线"互相印证等容, 任一信息即可锁定
📝 ③ 详细解题步骤
1 判过程: p-T 图上 A、B 在同一条过原点直线上 → p/T = 常数 → 等容过程 (与"刚性瓶"一致). T 升 → p 升, V 不变.
规则: p-T 图过原点直线 = 等容线
2 查 A: 等容过程 V 不变 → "体积增大" ❌
3 查 B: 质量一定 + V 不变 → 分子数密度 n = N/V 不变 → "减小" ❌
4 查 C: T 升高 → 分子平均动能增大 (T 是分子平均动能的标志) → ✓
5 查 D: T 升 → 分子平均速率增大 → 单位时间撞壁次数增多 ✓, 但每次碰撞的平均冲力也增大(分子动量更大) → "平均冲力不变" ❌. 正是"次数增多 + 冲力增大"两因素共同使 p 升高.
最终答案: C
✓ 验证: 等容升温下 p 升高的微观机制 = 碰撞更频繁 × 每次冲力更大, 与宏观查理定律 p/T 恒定自洽. 仅 C 正确.
⚠️ ④ 易错点
易错 1: 把 p-T 过原点直线当成等压线
❌ 错答: 看到"直线"就以为是等压过程, 判体积增大
✓ 正答: p-T 图过原点直线是等容线; 等压线在 V-T 图上才是过原点直线. 先看坐标轴是什么再下判断
易错 2: 认为 p 升高只因碰撞次数增多
❌ 错答: 选 D, 觉得"次数多了压强自然大"
✓ 正答: p 由两个微观因素共同决定 — 升温后分子更快, 撞得更频繁且每次撞得更重, 两者都增大
易错 3: 把分子平均动能与压强混为一谈
❌ 错答: "p 增大所以平均动能增大"——方向碰对但逻辑错
✓ 正答: 平均动能只看 T. 等温压缩时 p 增大而平均动能不变, 可见 p 与平均动能没有必然的同向关系
📖 锚回: 气体、液体与固体 task-cards (图像判别) · 分子动理论 task-cards (微观三量)

第一定律单过程辨析 · 恒温水池中上升的气泡
P1.1
基础
题目
某恒温养殖水池水温处处相同且保持不变. 池底增氧管释放出一个小气泡, 气泡缓慢上升, 上升过程中泡内气体温度始终与水温相同, 泡内气体可视为一定质量的理想气体. 对气泡从池底升到水面的过程, 下列说法正确的是 ( )

A. 泡内气体内能增大
B. 泡内气体放出热量
C. 泡内气体吸收的热量等于它对外界做的功
D. 水的热量自发地从低温处传向高温处, 维持气泡温度不变
📚 ① 知识点梳理
  • 第一定律 (锚: 热力学第一定律): ΔU = Q − W, 约定吸热 Q 为正, 气体对外做功 W 为正
  • 等温特例: 理想气体内能只由 T 决定 → 等温过程 ΔU = 0 → Q = W (吸热全部用于对外做功)
  • 液下压强 (锚: 气体、液体与固体): 越靠近水面, 水深越浅, 气泡内压强越小 → 玻意耳定律 pV 恒定 → V 增大
  • 方向性辨析 (锚: 第二定律 · 基础): 热量不会自发从低温物体传向高温物体 — 用于排除干扰选项
🧭 ② 解题思路与方法
  • 拆题: 已知 = 恒温(等温过程) + 上升(p 减小); 求 = ΔU / Q / W 的判断
  • 选路径: 单过程判断三步 — ① 定过程类型(等温) → ② 定 W 的正负(V 增大 → 对外做功 W > 0) → ③ 用 ΔU = Q − W 推出 Q
  • 关键意识: "恒温"是题眼, 等温过程 ΔU = 0 把三个量的关系锁死为 Q = W
📝 ③ 详细解题步骤
1 定过程: 水温处处相同且不变 → 泡内气体温度不变 → 等温过程 → ΔU = 0 → A "内能增大" ❌
规则: 理想气体 ΔU 只由 T 决定
2 定 W: 上升 → 水深变浅 → 泡内压强 p 减小 → 由玻意耳定律 pV 恒定 → V 增大 → 气泡对外界做功, W > 0
3 推 Q: ΔU = Q − W → 0 = Q − W → Q = W > 0, 气体吸热且吸收的热量恰好等于对外做的功 → B ❌, C ✓
4 查 D: 气泡与水温度相同(或气泡因膨胀略微吸热时温度趋于略低), 热量从水传向气泡是从不低于气泡温度的一方传来, 不存在"自发地从低温传向高温" → 表述违反热传导方向性 → ❌
最终答案: C
✓ 验证: 等温膨胀链条 — ΔU = 0, W > 0, Q = W > 0 — 三量自洽; 能量账目: 吸进多少热, 就对外做多少功, 内能分文不动.
⚠️ ④ 易错点
易错 1: 看到"膨胀"就以为内能减小或"升温"
❌ 错答: 气泡变大 → 觉得气体"变活跃"了 → 内能增大(或做功消耗内能 → 减小)
✓ 正答: 理想气体内能只看 T. 等温过程无论 V 怎么变, ΔU = 0; 对外做功消耗的能量由吸热补上
易错 2: W 的符号约定混乱
❌ 错答: 把"气体对外做功"代成负值后又用 Q = ΔU − W, 正负号绕晕
✓ 正答: 先写明约定再代入. 本文统一 ΔU = Q − W (W = 气体对外做功). 膨胀 → W > 0; 被压缩 → W < 0
易错 3: 误以为"温度不变 → 不吸热也不放热"
❌ 错答: ΔU = 0 → Q = 0
✓ 正答: ΔU = 0 只能推出 Q = W. 等温膨胀必须边吸热边做功才能保持温度不变 — "温度不变"不等于"无热交换"

热机循环逐段分析 · 太阳能空气热机演示装置
P1.3
中等
题目
科技社团的太阳能空气热机演示装置中, 一定质量的理想气体经历如下循环 (p-V 图上为闭合回路 A→B→C→A):
· A→B: 等容加热, 压强由 p0 升为 2p0 (体积保持 V0);
· B→C: 等温膨胀, 体积由 V0 增大到 2V0;
· C→A: 等压压缩, 体积由 2V0 回到 V0, 回到初态 A.

已知初态 A 的温度为 T0. 下列说法正确的是 (多选) ( )

A. 状态 B 与状态 C 的温度均为 2T0
B. A→B 过程气体吸热, 且吸收的热量全部用于增加内能
C. B→C 过程气体不吸热也不放热
D. C→A 过程气体放出的热量大于外界对气体做的功
E. 整个循环一周, 气体内能变化为零

📊 p-V 闭合循环 · A→B 等容(垂直) · B→C 等温(双曲线) · C→A 等压(水平). 一周 ΔU = 0

📚 ① 知识点梳理
  • 状态方程定各态温度 (锚: 气体、液体与固体): pV/T = 常数, 先把 A、B、C 三态的 (p, V, T) 全部定出再谈能量
  • 四特例过程的能量特征 (锚: 热力学第一定律):
    • 等容: W = 0 → Q = ΔU (吸热全变内能)
    • 等温: ΔU = 0 → Q = W (吸热全做功, 但不是不换热)
    • 等压: W = pΔV, Q = ΔU + W 两项都有
  • 循环铁律: 内能是状态量 → 循环一周回到初态, ΔU(一周) = 0
🧭 ② 解题思路与方法
  • 拆题: 已知 = 三段过程类型 + 初态 T0; 求 = 各态温度 + 各段吸放热判断
  • 选路径: 循环题两轮扫描 — 第一轮用状态方程把 A(p0, V0, T0) / B / C 三态参量填满; 第二轮逐段套 ΔU = Q − W 判能量
  • 判 ΔU 的捷径: 比较两端温度即可 — T 升则 ΔU > 0, T 降则 ΔU < 0, T 不变则 ΔU = 0
📝 ③ 详细解题步骤
1 第一轮定状态:
· A: (p0, V0, T0)
· A→B 等容: p0/T0 = 2p0/TB → TB = 2T0 → B: (2p0, V0, 2T0)
· B→C 等温: TC = TB = 2T0; pV 恒定 → pC = 2p0·V0/2V0 = p0 → C: (p0, 2V0, 2T0)
· 核对 C→A 等压: pC = pA = p0 ✓, 2V0/2T0 = V0/T0 ✓ → A 正确
规则: 先填满状态参量表, 再谈能量 — 循环题第一铁律
2 A→B 等容升温: W = 0, T 升 → ΔU > 0 → Q = ΔU > 0, 吸热且全部用于增加内能 → B 正确
3 B→C 等温膨胀: ΔU = 0, V 增大 → 对外做功 W > 0 → Q = W > 0, 必须吸热 → C "不吸热也不放热" ❌
4 C→A 等压压缩: T 由 2T0 降到 T0 → ΔU < 0; 体积减小 → 外界对气体做功 (W < 0). ΔU = Q − W → Q = ΔU + W < 0 放热, 且 |Q| = |ΔU| + |W| 大于外界做的功 |W| → D 正确
5 循环一周回到 A: 内能是状态量 → ΔU(一周) = 0 → E 正确
最终答案: ABDE
✓ 验证: 能量总账 — 一周 ΔU = 0 → Q(净) = W(净); p-V 图上闭合回路沿顺时针 (A→B→C→A 包围面积为正), 气体对外净做功, 对应净吸热, 是一台"吸热做功"的热机, 与"演示热机"情境自洽.
⚠️ ④ 易错点
易错 1: 把"等温"当成"绝热"
❌ 错答: 选 C — "温度不变所以不换热"
✓ 正答: 等温是 ΔU = 0, 绝热才是 Q = 0. 等温膨胀必须持续吸热来补偿对外做功, 两个概念是循环题最常考的偷换
易错 2: 等压段只算一头账
❌ 错答: C→A 放热 |Q| 与 |W| 比大小时, 忘了 ΔU 也在减小, 答"放热等于外界做的功"
✓ 正答: 等压过程 Q 与 ΔU、W 三项并存: 放出的热 = 内能减少量 + 外界做功量, 必然大于其中任何一项
易错 3: 不先定状态就直接判能量
❌ 错答: 跳过第一轮, 凭感觉判 B→C "温度升高"
✓ 正答: 循环题先用 pV/T 把每个状态点的三个参量填满, 能量判断全部以参量表为准, 不凭感觉

装置计算 · 卡销限位活塞加热台
P2+P3
较难
题目
某标本恒温加热台的核心部件是一只竖直放置、开口向上的汽缸: 缸内封有一定质量的理想气体, 上方活塞质量 m = 2 kg、横截面积 S = 1×10-3 m2, 可沿缸壁无摩擦滑动且不漏气. 缸壁上装有一对卡销, 初始时活塞被卡销托住, 缸内气柱长 L1 = 20 cm, 气体压强 p1 = 1.0×105 Pa, 温度 T1 = 300 K. 大气压强 p0 = 1.0×105 Pa, g 取 10 m/s2.

(1) 求初始时卡销对活塞的支持力;
(2) 缓慢加热气体, 求活塞恰好离开卡销时的温度 T2;
(3) 继续缓慢加热, 使气柱长度变为 L3 = 25 cm, 求此时温度 T3.

🔧 卡销活塞装置剖面 · 加热气体 → 卡销支持力减为 0 时活塞恰好离开 → 之后等压上升

📚 ① 知识点梳理
  • 活塞受力平衡定压强 (锚: 气体、液体与固体): 对活塞列平衡方程, 把"看不见的"气体压强用"看得见的"重力 / 大气压 / 卡销力表达出来
  • 卡销的角色: 卡销存在时活塞位置被锁定 → 等容过程; "恰好离开卡销" = 卡销支持力减为零的临界时刻; 离开后活塞自由 → 受力平衡压强恒定 → 等压过程
  • 分段选定律: 等容段用查理定律 (p/T 恒定); 等压段用盖-吕萨克定律 (V/T 恒定)
  • 缓慢加热 = 准静态过程, 每一刻都可视为平衡态
🧭 ② 解题思路与方法
  • 拆题: 装置 = 竖直缸 + 活塞 + 卡销; 过程 = 等容升压(卡销托住) → 临界点(卡销力归零) → 等压升温(活塞上移)
  • 选路径: 三步链 — ① 受力平衡定各阶段压强 → ② 找临界条件 → ③ 分段套气体定律
  • 临界条件翻译: "恰好离开卡销" ⇔ N = 0 ⇔ 气体压强恰好能独立托起活塞 ⇔ p2 = p0 + mg/S
📝 ③ 详细解题步骤
1 初始对活塞受力分析 (设卡销支持力 N 竖直向上):
向上: 气体压力 p1S + 卡销 N; 向下: 重力 mg + 大气压力 p0S
平衡: N = mg + p0S − p1S = 2×10 + (1.0×105 − 1.0×105)×1×10-3 = 20 N
规则: 受力平衡定压强 — 三步链第一步
2 临界压强: 活塞恰好离开卡销时 N = 0, 活塞自身平衡:
p2S = p0S + mg → p2 = p0 + mg/S = 1.0×105 + 20/(1×10-3) = 1.2×105 Pa
3 第一阶段 (卡销托住 → 恰好离开): 活塞不动 → 等容, 用查理定律:
p1/T1 = p2/T2 → T2 = T1·p2/p1 = 300 × 1.2/1.0 = 360 K
4 第二阶段 (活塞离开卡销后): 活塞自由且平衡 → 压强恒为 p2 → 等压, 用盖-吕萨克定律 (同一缸, V ∝ L):
L1/T2 = L3/T3 → T3 = T2·L3/L1 = 360 × 25/20 = 450 K
最终答案: (1) N = 20 N; (2) T2 = 360 K; (3) T3 = 450 K
✓ 验证: 全程也可一步核对 — p1V1/T1 = p3V3/T3: (1.0×20)/300 = (1.2×25)/450 → 0.0667 = 0.0667 ✓. 分段结果与总状态方程自洽.
⚠️ ④ 易错点
易错 1: 漏掉大气压力这股力
❌ 错答: N = mg − p1S, 算出负值后强行取绝对值
✓ 正答: 开口向上的活塞上表面受大气向下压. 受力四件套: 重力 + 大气压力 (向下) / 气体压力 + 卡销力 (向上), 一件不能少
易错 2: 全程套一个定律
❌ 错答: 从 300 K 直接用等压 (或等容) 算到 L = 25 cm 的状态
✓ 正答: 卡销离开前后过程性质突变(等容 → 等压), 必须以"恰好离开"为分界分段处理, 各段各选各的定律
易错 3: 临界条件写错
❌ 错答: 把"恰好离开卡销"当成 p2 = p0 (忘了活塞重力)
✓ 正答: 临界 = N = 0 且活塞平衡 → p2 = p0 + mg/S. 凡"恰好"二字, 先翻译成某个力恰好为零, 再列平衡式
易错 4: 温度用摄氏度代入气体定律
❌ 错答: 若题给 27 ℃, 直接用 27 代入比例式
✓ 正答: 气体定律一律用热力学温度 K (T = t + 273), 摄氏温度只在升温差值 ΔT 上与 K 等价
📖 锚回: 气体、液体与固体 task-cards (查理 / 盖-吕萨克 / 受力平衡定压强)
5
变质量问题 · 救援充气筏气瓶 (等效定质量法)
P4
较难
题目
水上救援队为容积 V = 10 L 的刚性储气瓶充气, 用于快速充开救生筏. 充气筒每打一次, 可将压强 1.0×105 Pa、体积 2 L 的空气压入瓶内. 瓶内初始有压强 p1 = 1.0×105 Pa 的空气, 充气过程温度保持 27 ℃ 不变, 空气视为理想气体.

(1) 要使瓶内压强达到 p2 = 3.0×105 Pa, 需要打气多少次?
(2) 充好气的瓶子从 27 ℃ 的库房移到 −3 ℃ 的户外救援现场 (瓶内气体质量不再变化), 求瓶内气体压强 p3.

🔧 变质量充气示意 · 关键: 把"瓶内原有 + n 次充入"打包成定质量, 在共同 (p₁ = 1×10⁵ Pa) 下总体积 V + n·ΔV → 末态全挤进 V

📚 ① 知识点梳理
  • 等效定质量法 (本题核心方法, 先吃透): 气体定律只对一定质量的气体成立. 充气 / 放气时瓶内气体质量在变, 不能直接套定律. 化解办法 — 重新圈定研究对象:
    • 充气: 把"瓶内原有气体 + 即将充入的全部气体"在初态下打包成一个大对象 (总体积 = V + n·ΔV, 同压强), 末态 = 全部挤进瓶内 (体积 V). 变质量 → 定质量
    • 放气: 把"末态留在瓶内的 + 已放出的气体"在末态下合并看待, 倒推初态
  • 等温用玻意耳定律 (锚: 气体、液体与固体): 打包后的大对象 p·V(总) = 恒定
  • 变温定容用查理定律: 第 (2) 问气体质量已锁定且瓶刚性 → p/T 恒定
🧭 ② 解题思路与方法
  • 拆题: 第 (1) 问 = 等温充气 (变质量 → 等效打包); 第 (2) 问 = 定质量定容降温 (常规查理)
  • 选路径: 变质量问题四步 — ① 识别场景 (充/放/抽/灌) → ② 选打包时刻 (充气在初态打包, 放气在末态合并) → ③ 写出打包后的初末态参量 → ④ 套玻意耳 (等温) 或状态方程 (变温)
  • 核心一句话: 定律不变, 研究对象变 — 把"会变的质量"圈成"不变的总量"
📝 ③ 详细解题步骤
1 设需打气 n 次. 打包研究对象 = 瓶内原有气体 + n 筒待充气体, 初态 (同温同压 1.0×105 Pa) 总体积:
V(总) = V + n·ΔV = 10 + 2n (单位: L)
规则: 充气问题在初态打包 — 等效定质量法
2 末态: 这包气体全部位于瓶内, 体积 V = 10 L, 压强 p2 = 3.0×105 Pa. 等温 → 玻意耳定律:
p1(V + n·ΔV) = p2V
1.0×105 × (10 + 2n) = 3.0×105 × 10
3 解出: 10 + 2n = 30 → n = 10 次
4 第 (2) 问: 质量锁定 + 刚性瓶 (等容), 降温 T2 = 300 K → T3 = 270 K, 查理定律:
p2/T2 = p3/T3 → p3 = 3.0×105 × 270/300 = 2.7×105 Pa
最终答案: (1) 打气 10 次; (2) p3 = 2.7×105 Pa
✓ 验证: 第 (1) 问用"分子数守恒"复核 — 初态 10 L×1 + 充入 20 L×1 = 30 (L·105 Pa) = 末态 10 L×3 ✓; 第 (2) 问降温压强应下降, 2.7 < 3.0 方向正确.
⚠️ ④ 易错点
易错 1: 对变质量气体直接套定律
❌ 错答: 对"瓶内气体"写 p1V = p2V, 得出压强不可能变的荒谬结论
✓ 正答: 瓶内气体质量在增加, 不满足定律前提. 必须先打包成定质量对象再套定律
易错 2: 打包时漏算瓶内原有气体
❌ 错答: 只算 n 筒气: 1.0×105×2n = 3.0×105×10 → n = 15 次
✓ 正答: 末态瓶内 3.0×105 Pa 的气体里包含原有的那一份, 初态总体积是 10 + 2n 而不是 2n
易错 3: 充气筒体积与瓶体积单位混用
❌ 错答: 一边用 L 一边换算成 m3, 比例式两边不齐
✓ 正答: 比例式中同一物理量单位统一即可 (全用 L 或全用 m3); 涉及做功 W = pΔV 时才必须换国际单位制
易错 4: 第 (2) 问又把充气次数搅进来
❌ 错答: 降温段还想用"打包对象"分析
✓ 正答: 充气结束后质量已锁定, 第 (2) 问就是普通的定质量等容降温, 切换回常规查理定律即可 — 识别"哪一段变质量、哪一段定质量"是本题型的分水岭
📖 锚回: 理想气体的状态方程 task-cards (玻意耳 / 查理定律 · 变质量等效定质量法详见该板块方法卡 M)
6
第一定律综合大题 · 电热顶升演示缸 (ΔU = CΔT)
P1.4
综合压轴
题目
创新实验社团制作了一台"电热顶升"演示装置: 竖直放置、开口向上的绝热汽缸内封有一定质量的理想气体, 缸内装有小电热丝. 活塞质量 m = 1 kg、横截面积 S = 5×10-4 m2, 绝热且可无摩擦滑动、不漏气. 初始气柱长 L1 = 30 cm, 温度 T1 = 300 K. 大气压强 p0 = 1.0×105 Pa, g 取 10 m/s2.

接通电热丝缓慢加热, 活塞缓慢上升 6 cm 后断电. 已知该气体内能变化与温度变化满足 ΔU = CΔT, 其中 C = 0.5 J/K.

(1) 求加热结束时气体的温度 T2;
(2) 求该过程中电热丝放出的热量 Q (即气体吸收的热量).

🔧 绝热缸 + 自由活塞 + 电热丝 · 活塞缓慢上升 6 cm = 等压过程. ΔU = CΔT, W = pΔV, Q = ΔU + W

📚 ① 知识点梳理
  • 三步链全链条 (本题型骨架): ① 受力平衡定压强 → ② 气体定律定末态 → ③ ΔU = Q − W 算热量 — 物理热学综合大题的标准流水线
  • 等压判定 (锚: 气体、液体与固体): 活塞自由 + 缓慢移动 → 每一刻都受力平衡 → 气体压强恒定 → 等压过程, 用盖-吕萨克定律
  • 等压做功 (锚: 热力学第一定律): W = pΔV = pS·ΔL (气体对外做功取正)
  • ΔU = CΔT 桥: 题给常数 C 把"温度变化"直接换算成"内能变化", 是命题人递给你的内能算尺 (其根据见进阶板块: 理想气体内能的微观推导)
🧭 ② 解题思路与方法
  • 拆题: 已知 = 等压升温 (活塞自由) + ΔL + C; 求 = T2 与 Q
  • 选路径: 严格走三步链 — 平衡定 p → 盖-吕萨克定 T2 → 先算 ΔU 和 W 两块, 最后 Q = ΔU + W 收口
  • 能量账本意识: 电热丝放出的热, 一部分存进内能 (ΔU), 一部分被气体用来顶活塞对外做功 (W), 两本账加起来才是 Q
📝 ③ 详细解题步骤
1 受力平衡定压强 (活塞自由, 全程平衡):
p = p0 + mg/S = 1.0×105 + 10/(5×10-4) = 1.0×105 + 0.2×105 = 1.2×105 Pa (全程恒定 → 等压)
三步链第一步: 力平衡定压强
2 盖-吕萨克定律定末温 (V ∝ L, L2 = 30 + 6 = 36 cm):
L1/T1 = L2/T2 → T2 = 300 × 36/30 = 360 K
三步链第二步: 气体定律定状态
3 算内能变化: ΔU = CΔT = 0.5 × (360 − 300) = 30 J
4 算气体对外做功 (等压):
W = pS·ΔL = 1.2×105 × 5×10-4 × 0.06 = 3.6 J
注意: ΔL 必须换成 0.06 m — 做功计算用国际单位制
5 第一定律收口: ΔU = Q − W → Q = ΔU + W = 30 + 3.6 = 33.6 J
三步链第三步: ΔU = Q − W 算热量
最终答案: (1) T2 = 360 K; (2) Q = 33.6 J
✓ 验证: 能量账本 — 吸热 33.6 J = 存入内能 30 J + 顶升做功 3.6 J, 账目平衡; 缸与活塞绝热 → 热量只能来自电热丝, Q 全归气体, 无旁漏 ✓.
⚠️ ④ 易错点
易错 1: 看到"绝热汽缸"就当绝热过程 (Q = 0)
❌ 错答: Q = 0 → "气体不吸热", 第 (2) 问无从下手
✓ 正答: 缸壁绝热只是说热不外漏, 缸内电热丝照样给气体供热. "装置绝热"≠"过程绝热", 要看热源在系统内还是系统外
易错 2: 做功计算单位不换
❌ 错答: W = 1.2×105 × 5×10-4 × 6 = 360 J (ΔL 用了 cm)
✓ 正答: W = pΔV 必须全用国际单位制: ΔL = 0.06 m → W = 3.6 J. 比例式可以不换单位, 乘积式必须换
易错 3: Q 只算一头
❌ 错答: Q = ΔU = 30 J (忘了做功) 或 Q = W = 3.6 J (忘了内能)
✓ 正答: 等压吸热两本账并存: Q = ΔU + W. 凡末态温度和体积都变的过程, 两项都不为零
易错 4: 把 ΔU = CΔT 的 ΔT 用错温标差
❌ 错答: 犹豫"360 K − 300 K 要不要换摄氏度"而改成错误数值
✓ 正答: 温差 ΔT 在 K 与 ℃ 下数值相同 (两温标只差平移 273), ΔT = 60 K = 60 ℃, 直接代入
