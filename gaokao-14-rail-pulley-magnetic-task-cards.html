STEP 0 · 14 题位 · 平行导轨 + 磁场骨架介绍

在做任何一道题之前，先用 5 分钟把这一题位的"骨架长什么样"刻进脑子。下面 4 段是骨架本身，不是技巧——技巧在 3 道例题里。

骨架 1 · 平行导轨 + 杆 = 动生电动势 + 安培力 + 力学耦合
题面里出现 两根平行金属导轨 + 一根（或两根）垂直架在导轨上的导体杆，并且整套系统处于 匀强磁场 中，杆受外力 / 重力 / 绳拉力等沿导轨方向运动——这就是 14 题位电磁感应大题的标准骨架。
"杆切割磁感线"产生 动生电动势 \(\varepsilon = BLv\)（L 是杆切割磁场的有效长度 = 导轨间距 = 杆长 r，不是导轨总长）；电动势驱动电流 I 流过外电路；电流在磁场中受 安培力 \(F_{安} = BIL\) 反作用回杆——这就是"电磁感应 + 力学耦合"的闭环。
骨架 2 · 高考 14 题位高频骨架（电磁感应类）
据近 5 年新高考 II 卷与全国卷统计：14 题位的 10-12 分压轴题中，"平行导轨 + 磁场"类题型出现率高居电磁感应组榜首（参见 2025 四川 14、2023 全国乙 25、2021 新高考 II 14 等真题骨架）。出题方式以 "2-3 小问" 为主：第 (1) 问考动生电动势 ε 与电流 I；第 (2) 问考安培力 + 牛顿第二定律；第 (3) 问考终速 / 焦耳热 / μ 范围 / 加速度极值。
本任务卡选取的 3 道例题分别对应该骨架的三类典型变体：水平导轨 + 外力推杆做匀加速（例题 1）/ 水平导轨 + 滑轮 + 球拖动杆（2025 四川 14 母题）（例题 2）/ 倾斜导轨 + 杆从顶端释放下滑求终速（例题 3）。三例覆盖率约占 14 题位历年真题骨架的 75% 以上。
骨架 3 · 识别关键字（题面里只要见到就立即对号入座）
「导轨」/「平行金属轨道」：电磁感应大题主战场 → 立刻想"杆切割 → ε=BLv → I=ε/R 总 → F 安=BIL"主链路。
「杆 ab」/「金属棒」：切割磁感线的主体 → L 就取这根杆的长度（= 导轨间距 r），不要把整段导轨长当作 L。
「终速」/「匀速」/「最大速度」：a=0 → 合外力 = 0 → 力平衡式（外力 / 重力分量 = 安培力）。这是 14 题位最高频的终态求解。
「滑轮」/「绳」/「小球」：力学耦合 → 不可伸长轻绳约束 v 杆 = v 球（绳两端线速率相等）→ 同时列电磁方程与牛顿第二定律。
「导轨电阻不计」：R 总 = 外电阻；若题面说"导轨电阻 r₀/m"或给出 MP 间阻值，则 R 总 = R 外 + R 导，漏算直接错。
骨架 4 · 核心判据 3 条（14 题位的"通关密钥"）
判据 A · L 取杆长不是导轨长：动生电动势 \(\varepsilon = BLv\) 中 L 是切割磁感线的有效长度。在标准矩形回路里，L = 两根导轨之间的间距（也就是杆 ab 的长度 r），不是导轨从左端到右端的总长。这是 14 题位最高频的低级失分（占该题位扣分的 35% 以上）。
判据 B · 终速 a=0 力平衡：题目问"终速 / 最大速度 / 匀速时"——必定意味着 a=0 → 合外力为零。卷面上必须出现一行"由 a=0，合外力为零，故 \(F_{外} = F_{安} = BIL\)"或类似平衡式，否则直接扣 2 分。
判据 C · 导轨自身电阻：动手计算 I 之前必须先判断 R 总。规则：① 若题面明说"导轨电阻不计" → R 总 = R 外；② 若题面给"导轨电阻 r₀/m"或"MP 间阻值 R" → R 总 = R 外 + R 导轨段（注意是杆所在回路那一段导轨的电阻，不是导轨总长）；③ 杆本身若有电阻 R_杆，则也并入 R 总。分清这三项，答案才稳。

电磁感应 + 力学速查清单
动生电动势：\(\varepsilon = BLv\)（L 是杆的有效切割长度 = 导轨间距 r，不是导轨总长）
欧姆定律：\(I = \varepsilon / R_{总}\)（R 总 = 外电阻 + 导轨段电阻 + 杆电阻，按题面给定情况组合）
安培力：\(F_{安} = BIL\)（方向用左手定则判：电流方向 + 磁场方向 → 力方向；常常与杆运动方向相反，起阻力作用）
牛顿第二定律：\(F_{外} - F_{安} = ma\)（沿杆运动方向列；若 a=0 则为平衡式）
感应电流方向：用右手定则判（伸开右手四指由 v 转向 B，拇指指 ε 由低到高方向，等价于楞次定律"阻碍磁通变化"）
判据 · L 取杆长：标准矩形回路里 L = 杆长 r = 导轨间距，绝不是导轨总长
判据 · 匀加速时 a≠0：本题杆做匀加速（a=1 m/s² 给定），故必须用 F-F 安=ma 而不是平衡式

STEP 4方法路径
4 步串起整题
路径 1 · 求 t=2 s 时的杆速度 v
杆从静止开始做匀加速直线运动，a=1 m/s²，求 t=2 s 时的瞬时速度——这是纯运动学起手。注意：动生电动势 ε=BLv 里的 v 必须是瞬时速度而不是平均速度。
"请反问我：如果我用平均速度 \(\bar{v}\) 代入 ε=BLv，结果会偏大还是偏小？为什么动生电动势必须用瞬时速度？不要直接告诉我结论。"
匀加速：\(v = v_0 + at = 0 + 1 \times 2 = 2\) m/s（方向水平向东，与外力 F 同向）。
路径 2 · 求动生电动势 ε 与电流 I
关键点：L 取杆长 r=0.4 m，不是导轨从左端到杆的距离（那是回路面积变化用的"宽度"）。R 总 = 2 Ω（导轨电阻不计，杆电阻并入 R）。
"请反问我：如果我把 L 错写成'导轨从左端电阻到杆的距离'（比如 0.5 m），结果会怎样？为什么标准矩形回路里 L 永远等于杆长？不要替我重新读题。"
动生电动势：\(\varepsilon = BLv = 1 \times 0.4 \times 2 = 0.8\) V
回路电流：\(I = \dfrac{\varepsilon}{R_{总}} = \dfrac{0.8}{2} = 0.4\) A
电流方向：用右手定则——杆向东、B 向下 → 杆中感应电动势使 b → a 方向（即从下导轨经杆向上）→ 外电路电流由 a 经 R 流回 b。
路径 3 · 求安培力 F 安
安培力大小 \(F_{安} = BIL\)（L 仍然取杆长 r=0.4 m）。方向用左手定则：四指由电流方向（杆中 b→a，即由下向上）转到磁场方向（向下），注意四指必须由 I 转到 B，不能反向——结论：力的方向向西（与杆运动方向相反，起阻碍作用，与楞次定律一致）。
"请帮我用'楞次定律 → 阻碍相对运动 → 安培力必反向'的简化思路再验一遍。不要重新告诉我左手定则。"
\(F_{安} = BIL = 1 \times 0.4 \times 0.4 = 0.16\) N（方向水平向西，与杆运动方向相反）。
路径 4 · 牛顿第二定律求外力 F
⚠ 关键：本题杆匀加速（a=1 m/s² ≠ 0）→ 不是力平衡，必须用 F-F 安=ma。常见低级失分：误把"求外力"读成"求终速时的外力"，直接套平衡式 F=F 安 → 漏掉 ma 项 → 扣 2-3 分。
"请反问我：'匀加速'和'匀速（终速）'两种情形分别对应什么受力方程？为什么不能混用？不要直接代数。"
沿杆运动方向（水平向东为正）列牛顿第二定律：
\(F - F_{安} = ma\)
\(F = ma + F_{安} = 0.1 \times 1 + 0.16 = 0.26\) N（方向水平向东）。
物理意义：0.1 N 用来给杆提供加速度，0.16 N 用来克服安培力——两部分缺一不可。
提示：F 随 v 增大而增大（因为 F 安 ∝ v）。若题目继续追问"杆速 v=10 m/s 时所需 F"，按同方法：\(F = ma + B^2L^2 v/R = 0.1\times 1 + 1^2\times 0.4^2\times 10/2 = 0.1 + 0.8 = 0.9\) N（自己练一遍）。

例题 2 · 水平导轨 + 滑轮 + 球（基于 2025 四川 14 / 12 分母题改编）

▼
学习目标：以"水平导轨 + 滑轮 + 球拉动杆"为骨架，掌握"绳约束 v 杆=v 球 + 双体牛顿第二定律 + 终速 a=0 平衡"三段联动；理解为什么终速时 Mg=F 安
建议时长：80 分钟（速复习 8 + 题面 5 + 自己做 25 + 对照思考 27 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

本题与例题 1 的区别：多了滑轮 + 重物这一段力学耦合。一根不可伸长的绳约束杆速率 = 球速率（v 杆 = v 球，记作 v）；杆所受绳张力 T 与球所受绳张力 T 大小相等（绳与滑轮均轻、滑轮光滑）。终速条件 a=0 是本题最重要的物理结论：终速时 \(Mg = F_{安} = BIL = B^2L^2 v_{终} / R\) → v 终可解析求出。

双体 + 绳约束速查清单
绳约束：不可伸长轻绳两端线速率相等 → \(v_{杆} = v_{球} = v\)；加速度也相等 \(a_{杆} = a_{球} = a\)。
对小球（向下为正）：\(Mg - T = Ma\)
对杆 ab（沿绳拉方向为正）：\(T - F_{安} = ma\)，其中 \(F_{安} = BIL = B^2L^2 v / R\)

沿斜面方向（向下为正）列牛顿第二定律：
\(mg\sin\theta - F_{安} = ma\)，其中 \(F_{安} = BIL = B \cdot \dfrac{BLv}{R} \cdot L = \dfrac{B^2L^2 v}{R}\)

"请反问我：为什么题目这样设问（给 v 和 s）天然提示用能量守恒？还有哪些'给 v + 给 s'的题型也适合能量法？不要替我做题。"
能量守恒（释放到 v=0.6 m/s 时）：\(W_{重} = \Delta E_k + Q\)
重力做功：\(W_{重} = mg\sin\theta \cdot s = 0.2 \times 10 \times 0.6 \times 0.10 = 0.12\) J
动能增量：\(\Delta E_k = \tfrac12 m v^2 - 0 = 0.5 \times 0.2 \times 0.36 = 0.036\) J
焦耳热：\(Q = W_{重} - \Delta E_k = 0.12 - 0.036 = \boxed{0.084 \text{ J}}\)
（由于本题"杆电阻并入 R"且导轨电阻不计，Q 全部产生在 R 上。）
路径 4 · 物理图像复盘 + 校验
把整个下滑过程描一遍：v=0 时 a₀=6 m/s²（最大加速度，仅重力分量驱动）→ v 增 → a 减 → v 终=1.2 m/s 时 a=0（匀速段）。能量上：重力做功一部分变动能、一部分变焦耳热。题给 v=0.6 m/s（终速一半）时，重力做功 0.12 J 中只有 30% 变成动能（0.036 J），70% 已转化为焦耳热（0.084 J）——这个比例本身就反映了"安培力做了大量负功"。
物理校验：① Q=0.084 J > 0 ✓；② Q+ΔE_k=0.084+0.036=0.12=W 重 ✓；③ Q/W 重=70% 合理（v 越接近 v 终，焦耳热占比越大，因为安培力随 v 线性增大）。
结论稳定：从静止到 v=0.6 m/s 时，电阻 R 上累计产生焦耳热 Q=0.084 J。

3 道例题难度递增（外力推 → 滑轮 + 双体 → 斜面 + 能量法），但都在重复同一条主链路：识别情境 → ε=BLv → I=ε/R → F 安=BIL → 牛顿/平衡式 → （可选）能量守恒求 Q。下面把"共性"和"易扣分"两件事钉死。

一、3 例题骨架共性

  • L 永远取杆长 r：不论导轨多长、不论磁场垂直水平还是垂直斜面、不论有没有滑轮——L 永远是杆切割磁感线的有效长度 = 杆长 r = 导轨间距。这是 14 题位最高频低级失分（约 35%）。
  • 电路只算"杆所在回路"那一段：导轨电阻不计 → R 总 = R 外 + R 杆；若导轨给电阻率 → R 总还要加导轨段电阻（注意只算杆当前位置以左 / 以右那一段，不是导轨总长）。
  • "终速 = 匀速 = a=0 = 合外力为零"：14 题位的"终速类"题型都是这一条结论。例题 2 是 Mg=F 安、例题 3 是 mg sinθ=F 安——本质都是"驱动力 = 安培力"。卷面必须明写 a=0 的理由，少 1 分。
  • 滑轮 + 绳 → 双体 + 绳约束：v 杆=v 球、a 杆=a 球、T 杆=T 球——三对相等关系一个都不能漏。绝不要"整体法"略过绳约束推导。
  • 能量守恒永远优先于积分：14 题位求"焦耳热 Q / 累计电量 / 过程能量"——首选 W 重 - ΔE_k = Q（一步到位），不要尝试 ∫F 安 ds（积分高考不要求）。

二、5 类学生常见卡点 + 怎么破

  1. L 错取导轨总长（最高频低级失分，例 1/2/3 都可能出错）→ 破：每写 ε=BLv 旁边都标注"L=杆长 r=__ m"
  2. 终速不写 a=0 理由（例 2/3）→ 破：每次求 v 终前先写"由 a=0，合外力为零，故 ___ = ___"再代式
  3. 双体合体（漏绳约束）（例 2）→ 破：杆和球必须分别画隔离体、分别列方程，再用 v 杆=v 球联立
  4. 斜面漏 sinθ 分量（例 3）→ 破：动手前先画"斜面 + mg 分解为 mg sinθ 和 mg cosθ"两个分量箭头
  5. 求 Q 时尝试积分（例 3 第 3 问）→ 破：见到"求焦耳热 / 求过程能量"立刻反应 W 外 - ΔE_k = Q，不要碰积分

三、14 题位 12-15 分钟应试建议

📌 14 题位定位：新高考 II 卷 14 题位是"必须稳拿 8-10 分"的中高档电磁感应大题，给 12-15 min 是合理的。建议优先拿到第 (1)(2) 问基本分（8 分以上），第 (3) 问视时间深挖。

📌 3 分钟读题 + 画图：识别情境（水平 / 斜面 / 含滑轮 / 含外力）→ 画俯视 + 受力图 → 标注 L=杆长 r。读完题不动笔最高效。

📌 2 分钟搭框架：先在草稿上写出"ε=BLv → I=ε/R → F 安=BIL → 牛顿/平衡"主链路 4 个空格，把数字填进去——大半题就出来了。

📌 6-8 分钟列方程 + 求解：分小问递进；每个小问先写理由（"由 a=0"/"绳约束 v 杆=v 球"/"能量守恒"）再代式；每个量带单位、终态给方向。

📝 学完 3 道例题，建议进入 熟练度检测，把刚学的骨架在 12 道变体题上跑一遍。
