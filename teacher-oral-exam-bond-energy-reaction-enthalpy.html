📋 听点总表（全部 22 锚 · ⭐ = 核心听点 · 4 项板块边界守门）
类别听点学生说到位时，话里应包含什么
键能定义⭐⭐ 键能定义25 °C / 1 atm 下气态分子中 1 mol 化学键完全断裂到原子态所吸收的能量；ΔH键 > 0。
⭐ 4 个限定词气态 / 1 mol 键 / 完全断到原子态 / 始终吸热（ΔH > 0）。
⭐ 气态限定的理由液固相会混入氢键 / 范德华 / 离子结合能，不属于化学键本身。
完全断到原子态的理由原子态是绝对最高参考点，所有键能在统一参考下可加和。
⭐⭐ 断键吸热 / 成键放热断键始终吸热（克服稳定结合）、成键始终放热；符号永不倒置。
⭐ 离解能 vs 平均键能H₂O 第一 O-H 离解能 498、第二 428、平均 463；多原子分子里逐根键不等。
⭐ 典型键能数据H-H 436 / Cl-Cl 243 / H-Cl 431 / O=O 498 / N≡N 945 / C-C 348 / C=C 614 / C-H 413 / O-H 463 / C=O 745。
键能算法⭐⭐ 算法核心公式ΔH ≈ Σ(反应物键能) − Σ(产物键能)；逐项指认每一项符号。
⭐ 盖斯特例算法 = 盖斯定律在「全部原子化虚拟路径」上的特殊情形。
三层能级图顶层气态原子态 + 中层反应物 + 底层产物；红箭头上行（断键吸热）+ 绿箭头下行（成键放热）+ 蓝虚线 ΔH。
精度 ±5-10%用平均键能 → 误差累积但部分抵消；实验 / ΔHf° 精度 ±1%。
⭐ 4 个使用限制① 全气态 ② 平均键能误差 ±5-10 ③ 不适用离子 / 强极性反应 ④ 必须完整断键+完整成键。
⭐ H₂+Cl₂→2HCl 算例Σ断 = 436 + 243 = 679；Σ成 = 2 × 431 = 862；ΔH ≈ −183 kJ/mol（实测 −185 验证）。
⭐ CH₄ 燃烧算例Σ断 = 2648 / Σ成 = 3342 ⇒ ΔH ≈ −694；与燃烧热 −890（液态）差 196 = 液化放热 88 + 键能误差 108。
水气态 vs 液态差额差额 = 2 × 蒸发热（每 mol 水 +44 kJ/mol 蒸发吸热 / 液化放热）。
三方法对照⭐⭐ 三方法对照实验测量 ±1%（弹式 / 简易量热计）/ 盖斯+ΔHf° ±1%（查表）/ 键能算法 ±5-10%（全气态共价）。
⭐ 决策树有实验数据? → 实验测量；ΔHf° 全查得到? → 盖斯+ΔHf°；全气态共价? → 键能算法。
⭐ H₂+½O₂→H₂O(g) 三方法案例键能 −241 / ΔHf° −241.8 / 实验 −241.8 三方法一致（本反应「运气好」误差 < 1%）。
⭐ 互补不替代实验测得准 / 盖斯算得广 / 键能估得快；分工不是替代。
思维方式反应能量的微观本质反应放热 = 产物键比反应物键更稳定；吸热反过来。
工程估算 vs 精确推导键能（±5-10%）vs 实验+ΔHf°（±1%）是两种风格，按精度需求 + 数据可用性选。
平均误差部分抵消反应物 / 产物两侧的「高估」「低估」相减时部分抵消 → 最终 ΔH 误差控制在 ±5-10%。

📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 ΔH ≈ Σ反应物键能 − Σ产物键能 并逐项指认」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
