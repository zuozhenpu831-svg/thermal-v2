关于深度口试

口试是热力学第一定律板块的真实通关环节：先把学习任务卡学完、基础检测和熟练度检测练熟，然后预约口试。本页是 完整题库，公开给所有学生 —— 你可以提前准备每一条主问题。

口试由教师当面进行，从下方 6 条主问题中抽 3 条。每条主问题先开放讲，再由教师根据你的回答追问。讲思路比说出唯一正确答案重要 —— 教师听的是因果重建，不是背诵。

📌 通关条件：抽中的 3 条里，至少 2 条把核心要点讲清楚并能回应一层「为什么」，且整场至少一次在纸上写出 ΔU = Q − W 并指着式子说清每一项含义。

口试流程 总时长 15-20 分钟

① 暖场 · 自述
0-3 分钟
你先讲：「这个板块学到的最重要的一件事是什么？为什么重要？」—— 不卡时间，讲到能切入主问题就行。
② 主问题 ×3
3-15 分钟
从下方 6 条主问题中由教师抽 3 条，每条约 3-4 分钟。先开放讲，再回答追问。
③ 错题联动
15-18 分钟
带 1-2 道自己曾经做错的题（基础检测或熟练度检测里的），现场讲「当时为什么错、现在怎么想」。
④ 判定 · 签字
18-20 分钟
通过 = 当场签字；暂缓 = 给补学方向 + 预约二试（换未用过的主问题）；不通过 = 回到学习阶段重学。
主问题 1

主问题 1 · 跨 盖斯定律 + 燃烧热与中和热 · 用燃烧热数据反算未知反应

约 6-7 分钟
🎤 开放主问：工业上常需要 CH₄ + ½O₂ → CO + 2H₂(g) 这个反应的 ΔH，但实验室直接测它很难（不完全燃烧产物难控）。现在给你三个燃烧热数据：CH₄(g) -890.3、C(石墨) -393.5、H₂(g) -285.8 kJ/mol（H₂O 全是液态）。请你在纸上把这道综合题做出来 —— 关键是讲清楚两件事：① 用哪条原理把『测不到的反应』搭到『测得到的反应』上？② 三个组合系数怎么定？做完，把数值算到底。
👂 教师在听这些要点
  • ⭐⭐ ⭐⭐ 盖斯定律：ΔH 只看始末态、与路径无关 —— 这是搭桥的原理
  • ⭐⭐ ⭐⭐ 把目标反应写成已知燃烧热反应的代数组合：目标 = (CH₄ 燃烧) − (C 石墨燃烧) − 2×(H₂ 燃烧)
  • ⭐⭐ ⭐⭐ 燃烧热定义：1 mol 可燃物完全燃烧 → 稳定氧化物（H₂O 液态）、ΔH<0
  • ⭐ ⭐ 三种已知反应都是「→ CO₂(g) + H₂O(l) 完全燃烧式」，所以三个端点对齐才能消
  • ⭐ 反向规则：反向取 ΔH 相反数（学生说出 C 燃烧、H₂ 燃烧两条要反向）
  • ⭐ 倍乘规则：H₂ 燃烧 × 2（因目标有 2H₂）
  • ⭐ 净计量数法定系数（CH₄=+1、C=−1、H₂=−2 经物质核算）
  • ⭐ 连接性：实验测的燃烧热数据 → 用盖斯算法 → 算出测不到的反应（数据入口 → 算法出口）
  • 答案 ≈ −35.7 kJ/mol；用消元法核对方程左右物质对齐
主问题 2

主问题 2 · 跨 键能与反应焓变 + 盖斯定律 · 键能法 vs ΔHf° 法同题双算

约 6-7 分钟
🎤 开放主问：对反应 H₂(g) + ½O₂(g) → H₂O(g)，给你两套数据：(α) 键能 —— H-H 436、O=O 498、O-H 463 kJ/mol；(β) 标准生成焓 —— ΔHf°(H₂O, g) = −241.8 kJ/mol（稳定单质生成焓 = 0）。请你用两套方法分别算出这个反应的 ΔH —— 在纸上写过程。算完，回答三个问题：① 两次结果接近还是恰好一样？② 它们一致或近似一致的『根源』是什么？③ 如果不一致，能简单地说『有一个方法错了』吗？
👂 教师在听这些要点
  • ⭐⭐ ⭐⭐ 键能法：ΔH = Σ反应物键能 − Σ产物键能 = (436 + ½×498) − 2×463 = −241 kJ/mol
  • ⭐⭐ ⭐⭐ 盖斯+ΔHf° 法：ΔH = Σν·ΔHf°(产) − Σν·ΔHf°(反) = −241.8 − 0 = −241.8 kJ/mol
  • ⭐⭐ ⭐⭐ 两次结果几乎相同（差 < 1 kJ/mol）—— 这道题键能法精度好；一般键能法 ±5-10%、ΔHf° 法 ±1%
  • ⭐ ⭐ 一致性根源：键能法 = 盖斯定律应用到「全部断键到原子态」的虚拟路径，本质是同一个状态量原理
  • ⭐⭐ ⭐⭐ 连接性：所有 ΔH 算法的统一根源 = 焓是状态函数 / ΔH 路径无关
  • ⭐ 误差讨论：平均键能在多键反应里高低估部分抵消，所以最终 ΔH 误差被压缩
  • 若不一致：可能因为题里 O-H 用了平均键能，不是 H₂O 第一/第二离解能
  • 选谁的决策：有 ΔHf° → 优先 ΔHf°；全气态共价 + 缺数据 → 用键能估算
主问题 3

主问题 3 · 跨 燃烧热与中和热 + 盖斯定律 · 葡萄糖代谢综合题

约 5-6 分钟
🎤 开放主问：生物课问到：一颗中等苹果含约 27 g 可代谢的葡萄糖，全部氧化能放多少能量？反应是 C₆H₁₂O₆(s) + 6O₂(g) → 6CO₂(g) + 6H₂O(l)。已知 ΔHf°(葡萄糖) = −1273.3、ΔHf°(CO₂) = −393.5、ΔHf°(H₂O, l) = −285.8、ΔHf°(O₂) = 0 kJ/mol。请在纸上：① 算反应 ΔH；② 算 27 g 葡萄糖释放的能量（M = 180 g/mol）；③ 顺带回答：这个反应的 ΔH 算不算是『葡萄糖的燃烧热』？为什么？
👂 教师在听这些要点
  • ⭐⭐ ⭐⭐ ΔH = Σν·ΔHf°(产) − Σν·ΔHf°(反) = [6×(−393.5) + 6×(−285.8)] − [(−1273.3) + 0] = −2802.5 kJ/mol
  • ⭐⭐ ⭐⭐ 燃烧热定义 4 约定：1 mol 可燃物 / 完全燃烧 / 稳定氧化物（H₂O 液）/ 标准条件
  • ⭐ ⭐ 葡萄糖系数恰为 1 mol → 满足『1 mol 可燃物』归一
  • ⭐ ⭐ 产物是 6CO₂(g) + 6H₂O(l) → 满足『稳定氧化物』+ 液态水约定
  • ⭐ ⭐ 「两步法」物理本质：先把葡萄糖分解为稳定单质（反向 ΔHf°）+ 再让单质合成 CO₂ + H₂O —— 这就是 ΔHf° 公式『末减初』的来源
  • ⭐ 稳定单质 O₂ 的 ΔHf° = 0（约定 / 零点参考）
  • 为何选稳定单质做零点：因为反应物 / 产物都可拆为元素
  • ⭐ 连接性：ΔHf° 公式本身就是盖斯定律在『以稳定单质为零点』情形下的代数化
  • 连接性：本题的 ΔHf° 数据本身来源 = 弹式量热计实验测量
主问题 4

主问题 4 · 跨 吉布斯基础 + 键能 / 盖斯 · 合成氨综合判据

约 6-7 分钟
🎤 开放主问：工业合成氨：N₂(g) + 3H₂(g) → 2NH₃(g)，ΔH = −92 kJ/mol、ΔS = −198 J/(mol·K)。请在纸上完成这道综合题：① 用键能数据（N≡N = 945、H-H = 436、N-H = 391 kJ/mol）独立验算这个 ΔH ≈ −92 是否合理；② 写出 ΔG = ΔH − TΔS，算出转变温度 T_转 = ΔH/ΔS；③ 室温 298 K 下 ΔG 是正还是负？理论上自发吗？④ 工业合成氨为什么实际选在 ~450 ℃ + 高压 + 催化剂？这跟 ΔG 怎么协调？
👂 教师在听这些要点
  • ⭐⭐ ⭐⭐ 键能验算：ΔH = (945 + 3×436) − 6×391 = 2253 − 2346 = −93 kJ/mol ≈ −92 ✓
  • ⭐⭐ ⭐⭐ ΔG = ΔH − TΔS（本板块核心成果 / capstone 关键综合）
  • ⭐⭐ ⭐⭐ T_转 = ΔH/ΔS = (−92000)/(−198) ≈ 465 K ≈ 192 ℃
  • ⭐⭐ ⭐⭐ (ΔH<0, ΔS<0) 组合 → 低温自发（T < T_转）/ 高温不自发
  • ⭐ ⭐ ΔG < 0 自发 / = 0 平衡 / > 0 不自发（室温 ΔG = −92 + 298×0.198 ≈ −33 < 0 → 理论自发）
  • ⭐⭐ ⭐⭐ 「自发」≠「马上发生」：室温 ΔG<0 但 N≡N 键 945 kJ/mol → 动力学障碍极高 → 室温几乎不进行
  • ⭐ ⭐ 450 ℃：升温提速率（动力学）但牺牲一些 ΔG（热力学打折）+ 催化剂降活化能 + 高压利用气体摩尔减少推产率 —— 三重妥协
  • ⭐ 案例数值：合成氨 ΔH = −92.4 / ΔS = −198 / T_转 ≈ 467 K
  • ⭐ 三定律三件套：第一定律守恒、第二定律方向、ΔG 综合判据 —— 同一道题用了三个
  • ⭐ 连接性：键能算 ΔH 本身依赖盖斯定律 + 焓状态量 —— ΔG 的 ΔH 输入是盖斯算法体系的产物
主问题 5

主问题 5 · 跨 盖斯定律 + 吉布斯基础 + 燃烧热（讲思路）· 催化剂 vs 自发 vs ΔH 状态量

约 5-6 分钟
🎤 开放主问：工业 CO₂ 加氢制甲醇分两步：① CO₂ + H₂ → CO + H₂O(g) ΔH₁ = +41.2；② CO + 2H₂ → CH₃OH(g) ΔH₂ = −90.8 kJ/mol。请你不动笔（先讲思路），回答 4 个问题：① 总反应 ΔH 怎么算？根据是什么原理？② 如果换一种 Cu/ZnO 新型催化剂，让中间过渡态能量降低 30 kJ，总反应 ΔH 会变吗？为什么？③ 这个反应总 ΔH ≈ −50 < 0 是放热，那它是不是「肯定自发」？为什么？④ 第 1 步 ΔH₁ > 0 吸热、第 2 步放热 —— 中间体 CO 的存在，对工业实施有什么提示？
👂 教师在听这些要点
  • ⭐⭐ ⭐⭐ 总 ΔH = ΔH₁ + ΔH₂ = +41.2 + (−90.8) = −49.6 kJ/mol —— 盖斯定律加规则
  • ⭐⭐ ⭐⭐ 盖斯定律核心：ΔH 路径无关 / 焓是状态函数 ⇐ 反复用在两问里
  • ⭐⭐ ⭐⭐ 路径无关在化学反应中：测不到 / 难直测的反应可借能测的算出（第 ① 问的工具根基）
  • ⭐⭐ ⭐⭐ ΔG = ΔH − TΔS：判断自发要看 ΔG 不是只看 ΔH
  • ⭐ ⭐ 「放热不等于自发」（金刚石→石墨反例 / 冰融化反例）
  • ⭐ ⭐ 显式说出「ΔH < 0 ≠ 自发」是盖斯定律板块明确留给吉布斯解决的边界
  • ⭐ ⭐ 催化剂改变路径 → 改动力学 / 不改始末态 → 不改 ΔH（盖斯定律推论）
  • ⭐ ⭐ ΔH = H_末 − H_初 凡始末态固定 → ΔH 固定（催化剂不改始末态）
  • 盖斯代数 4 规则之加规则（① + ② = 总）
  • ⭐ 盖斯板块的边界：自发性判据的『另一半』是熵 → 吉布斯回答
  • 中间体 CO 提示：第 1 步吸热 → 需供热 / 催化剂选择性；高温吸热平衡偏右 + 低温放热平衡偏右 —— 两步温度难协同

📝 怎么准备

  • 不要背口诀。教师追问的第一刀往往就是「为什么」—— 把每个核心结论的来源 + 一句话理由想清楚（比如：等温 ΔU = 0 的根据是「理想气体 U 只依赖 T」，这条结论来自分子动理论板块）。
  • 带纸和笔。主问题 1、2、3、5、6 都涉及在纸上写公式或画图，画完再讲思路。
  • 提前挑 1-2 道自己错过的题（基础检测、熟练度检测、或纸笔检测里的），准备讲「当时为什么错、现在怎么想」。这是流程第 ③ 步。
  • 边界声明给你「松绑」：本板块有些「更深层」的概念不在要求内，没学过也不扣分；如果主动谈到是亮点，教师会简短肯定但不展开（这些深入内容在后续板块或进阶板块里学）。
  • 讲思路 ＞ 算对答案。教师听的是因果重建：你能不能把「为什么这样、不那样」讲清楚，比说出唯一正确数值更重要。
