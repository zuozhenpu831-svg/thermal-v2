📋 听点总表（全部 23 锚 · ⭐ = 核心听点 · 6 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义⭐ 分子是物质构成的基本微粒高中口径下「分子」既可指单原子（惰性气体），也可指多原子结合体；不是「只含多原子」。
⭐ 分子大小 ~10⁻¹⁰ m典型 0.3-0.5 nm；与原子核（10⁻¹⁵ m）/ 病毒（10⁻⁸ m）能区分量级。
⭐ 阿伏伽德罗常数 Nₐ = 6.02×10²³ /mol1 mol 任意物质粒子数；是宏观-微观的「汇率」，能算 ρ、M 到 d 的链。
关键结论 / 公式油膜法 d = V/S单分子层假设；油酸要稀释；油膜面积 ≠ 水面面积。
⭐ 球模型直径 d = ∛(6M/πρNₐ)链式：V摩 = M/ρ → V单 = V摩/Nₐ → 套球体积公式开 1/3 次方。
分子数密度 n数 = ρNₐ/M单位体积分子数；与摩尔数密度区分；量纲检验。
⭐ 布朗运动是热运动的间接证据5 步论证链（看见花粉无规则 → 必有外力 → 唯一来自水分子碰撞 → 碰撞不均 → 分子永不停止热运动）；花粉 ≠ 分子。
扩散现象是另一证据不需搅拌 → 分子自己会扩散 → 分子在运动；与对流区分。
⭐ F-r 曲线（r₀ 处合力 = 0）短距斥力主导 / 长距引力主导 / r₀ ~ 10⁻¹⁰ m；固体难压难拉、气体作用力可忽略。
⭐⭐ 温度 T ∝ 分子平均平动动能同温下不同气体的平均平动动能相同；本板块核心结论。
⭐ 同 T 下 v ∝ 1/√m例：vH/vO = √(mO/mH) = 4；不漏开方。
⭐ 温度是宏观统计平均量单个分子谈温度无意义；温度只反映平均、与 N 无关。
⭐⭐ 内能 U = 分子总动能 + 总势能E动 ∝ N·T（依赖 N 和 T）；E势 取决于分子间距（三态结构）；本板块核心结论。
⭐ U 是状态量只依赖当前 N、T、V、三态，与历史路径无关 ⇒ ΔU = U末 − U初。
⭐ 温度大 ≠ 内能多一杯热水（小 N + 高 T）vs 一桶冷水（大 N + 低 T），后者内能可能多得多；温度强度量、内能广延量。
⭐ 潜热的微观本质是分子势能差汽化吸热 / 熔化吸热 = 补足分子势能差（U气 > U液 > U固）；T 不变 E动 不变。
思维方式间接观察方法看不见 ≠ 不存在；找放大器（花粉 / 染色 / 香味）让看得见的现象告诉我们看不见的事。
多途径独立验证油膜 + 阿伏伽德罗链 + 布朗运动数学理论三种独立方法测同一个量结果一致。
宏观-微观桥接Nₐ 是「汇率」—— 宏观可测的 ρ、M 通过 Nₐ 算出微观的 d、V单、平均动能。
统计平均思维温度、内能都是大量分子的统计平均量；单分子谈不上温度。
边界守门
不展开		不推 P = (1/3)nmv² 压强公式留给进阶微观推导板块。
不推 ε̄ = (3/2)kT 严格关系本板块只到「T ∝ 平均动能」定性论断；严格留给进阶。
不引麦克斯韦速率分布解析式只用速率分布定性图像；解析式留给大学物化。
不谈量子零点能高中口径绝对零度 = 热运动停止；零点能不在范围。
不引分子间势能函数 U(r)伦纳德-琼斯势解析形式留给进阶 / 大学物化。
不引自由度均分定理动能只算平动，不分平动 / 转动 / 振动；留给大学物化。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上画出 F-r 曲线并指认 r₀、斥力区、引力区」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
