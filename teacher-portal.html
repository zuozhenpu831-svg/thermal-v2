深度口试 · 教师工具页 题库 + 追问树 + rubric + 记录区

📍 当前主题：热力学。每一页 = 一个板块的口试教师工具页（含追问树 / 误区 / rubric / 记录区）。学生从自己页面底部「🎤 深度口试」入口看到的仅是题库。其他主题接入待 starmap 后端。

口试操作流程速查 15-20 分钟（板块级）/ 20-25 分钟（capstone）

  1. 预约 + 暖场（0-3 分钟）：让学生先讲「这个板块最重要的一件事」，听切入点。
  2. 主问题 ×3（3-15 分钟）：在教师工具页用选题器抽 3 条（二试必须换未用过的主线），按学生反应走追问树。
  3. 错题联动（15-18 分钟）：学生带 1-2 道自己的错题（基础检测 / 熟练度检测里的），听因果重建。
  4. 判定 + 签字（18-20 分钟）：对照三档 rubric —— 通过当场签字；暂缓给补学清单 + 预约二试；不通过约定重学路径。

设计原则速查 公开 vs 私有 边界

  • 学生题库页（公开 · basic-oral-exam.html）：开放主问 + 考查锚点 + 流程说明 + 准备建议。学生可以从板块底部 nav「🎤 深度口试」直接打开。
  • 教师工具页（私有 · teacher-oral-exam-{slug}.html）：以上全部 + 追问树 (深追 L1/L2 vs 台阶 1/2) + 误区信号 + 救场话术 + 边界守门 + 三档 rubric + 补学清单 + 记录区。学生面 不链接本类页面。
  • 红线（教师页也遵守）：防投屏 —— 禁用「高考 / 备考 / 课标 / 必修 X / 选修 / 人教版」措辞。
  • 核心规则：教师页面 ≠ 答案册。题库公开不影响口试效力，因为追问 / rubric / 判定都在教师手里。
