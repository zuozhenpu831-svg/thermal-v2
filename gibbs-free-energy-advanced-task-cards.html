吉布斯自由能 · 进阶

G 的严格推导 · 最大有用功 · 平衡常数桥

焓与焓变的意义（进阶）（必读 —— H = U + PV 的严格定义 + ΔH = Qp 等压热定理是 G = H − TS 引入的前提）
盖斯定律（基础）（必读 —— ΔH 的计算与 ΔG 输入紧密结合）
吉布斯自由能 · 基础（必读 —— ΔG = ΔH − TΔS 公式 + 4 组合 + 应用是本进阶板块的对照基线）
⚠️ 跨主题前置：平衡常数 K（来自化学反应速率与化学平衡大主题 —— 本板块 Card 3 涉及 ΔG° = -RT ln Keq 平衡常数桥, 需要先理解"反应商 Q / 平衡常数 Keq"的化学定义。严格学习见该大主题相关板块）
所需工具AI 工具、纸笔、计算器；化学手册或 NIST WebBook 上的标准生成焓 ΔHf° / 标准熵 S° 数据；对气体常数 R = 8.314 J/(mol·K) 与 自然对数 ln 的熟练度

学习目标

  1. 能严格证明 G ≡ H − TS 是状态函数（由 H、T、S 均为状态量推出），并解释引入这个组合的物理动机
  2. 能由熵增定律 \(\Delta S_{universe} \geq 0\) 严格推导等温等压下 \(\Delta G \leq 0\) 是自发判据（含"乘 −T 反转不等号"的代数与物理含义）
  3. 能解释 G 的物理意义 —— 在等温等压条件下，ΔG 的负值 = 系统能提取的最大非膨胀有用功（\(\Delta G = -W_{max,非膨胀}\)），并推导其原型 dG = −δW非膨胀,可逆
  4. 能推导电化学桥公式 \(\Delta G = -nFE\)（n 为转移电子数 / F 为法拉第常数 / E 为电池电动势），并理解这是 G 作为"有用功"的电化学具体形式
  5. 能推导 \(\Delta G = \Delta G^\circ + RT \ln Q\)（反应商公式），并由平衡条件 ΔG = 0 推出平衡常数桥 \(\Delta G^\circ = -RT \ln K_{eq}\)
  6. 能用 \(\Delta G^\circ = -RT \ln K\) 算具体反应的平衡常数（如合成氨 \(N_2 + 3H_2 \rightleftharpoons 2NH_3\)），并解释 K 与 ΔG° 符号的对应（\(K>1 \Leftrightarrow \Delta G^\circ < 0\)）
  7. 能用范特霍夫方程（简化版）\(\frac{d \ln K}{dT} = \frac{\Delta H^\circ}{RT^2}\) 定性判断温度对 K 的影响：吸热反应 T↑ → K↑，放热反应 T↑ → K↓
驱动问题

基础板块给了 ΔG = ΔH − TΔS + 4 个组合的自发判据。进阶板块要回答 4 个更深的问题 ——

  • ΔG ≤ 0 自发判据从哪里来？"宇宙总是朝熵增方向变化"(ΔSuniverse ≥ 0) 是热力学第二定律的核心。这两个不等号方向相反，是怎么转换的？基础板块直接给了 ΔG ≤ 0，但没说推导过程。
  • G 的物理意义是什么？"自由能" 这个名字暗示了"自由的能量"。但什么意义上"自由"？为什么不直接用 ΔH 或 ΔS 而要引入这个新组合 G = H − TS？
  • ΔG 和电池电动势有什么关系？电化学告诉我们"电池能做电功 = nFE"。但 nFE 与热力学 ΔG 有什么定量关系？这两个看似不同的"功"是同一物理量吗？
  • ΔG 和化学平衡常数 K 有什么关系？"平衡常数 K" 在化学课上常作为经验量直接给出。但 K 与 ΔG° 有什么深层的热力学联系？为什么 K 随温度变化的方向由 ΔH° 的符号决定？

本板块用 3 个 Card 严格回答这 4 个问题，并把 G 整合成热力学与化学平衡 / 电化学的核心桥梁。

（提示：基础板块给了"工程化判据"; 进阶板块给"严格推导 + 物理意义 + 桥梁连接"。两者互补，进阶不是"重新讲一遍"。）

1

G 是状态函数 + 由 ΔSuniverse ≥ 0 严格推 ΔG ≤ 0

▼
学习目标: 严格证明 G ≡ H − TS 是状态函数 → 由熵增定律严格推 ΔG ≤ 0 自发判据 → 理解"乘 −T 反转不等号"的代数与物理含义
建议时长: 50-60 分钟
STEP 1先自己想（5-8 分钟）

不查笔记，先答 ——

  • 基础板块给 ΔG = ΔH − TΔS, 你能从更基本的物理原理(如熵增定律)推出这个公式吗？
  • "乘 −T" 在代数上 = "把不等号反向 + 乘正数"。这步操作对应什么物理? 为什么 T 必须是正的(绝对温度)?
  • G = H − TS 这种"焓减温度乘熵"的组合形式能保证 G 是状态函数吗? 为什么?
先回顾（核心理论根基 · 4 个前置板块）：
前置 熵的物理意义（进阶） 板块讲过 —— 熵增定律: 任何自发过程中 ΔSuniverse = ΔSsys + ΔS环 ≥ 0(等号成立于可逆过程)。这是第二定律的最严格表述。
前置 焓与焓变的意义（进阶） 板块讲过 —— 焓 H ≡ U + PV 是状态函数(因 U, P, V 都是状态量)，且等压条件下 Qp = ΔH(等压热定理)。
前置 盖斯定律（基础） 板块讲过 —— ΔH 的计算工具(代数加减 + ΔHf° 表)是本板块"求 ΔG"的输入。
前置 吉布斯自由能 · 基础 板块讲过 —— 公式 ΔG = ΔH − TΔS + 4 组合(ΔH × ΔS 的符号组合)+ 应用判据 ΔG ≤ 0 自发。本进阶板块要严格推导这个公式的来源。
STEP 2G ≡ H − TS 的引入动机 + 状态函数证明
吉布斯自由能 G 的严格定义 ⭐⭐

\[ G \equiv H - TS \]
其中 H 是焓, T 是绝对温度, S 是熵。所有量都是状态函数。

为什么是 G = H − TS 而不是别的组合？

G 引入的物理动机
动机 ①: 我们想找一个量, 在等温等压条件下(化学反应最常见的条件), 该量的变化方向 = 反应自发方向。
动机 ②: 已知 ΔSuniverse ≥ 0 是自发判据, 但 ΔSuniverse = ΔSsys + ΔSenv 涉及环境, 实际算不方便(环境太大)。我们想把它转化成只涉及系统的量。
关键观察: 等压条件下, 环境从系统吸热 = -Qp,sys = -ΔHsys。环境是大热库(温度恒定 T), 它的熵变 = ΔSenv = -Qp,sys / T = -ΔHsys / T。
代入熵增定律: ΔSuniverse = ΔSsys + ΔSenv = ΔSsys − ΔHsys / T ≥ 0
两端乘 −T (T > 0, 不等号反转): −T·ΔSuniverse = −T·ΔSsys + ΔHsys ≤ 0
定义: ΔG ≡ ΔHsys − T·ΔSsys = -T·ΔSuniverse → ΔG ≤ 0 ⟺ ΔSuniverse ≥ 0 ✓
核心收获: G = H − TS 不是"凭空定义", 它是把熵增定律从"宇宙视角"翻译成"系统视角"的桥梁。两者完全等价(等温等压下), 但 ΔG 只涉及系统, 实际算更方便。
G 是状态函数 — 严格证明
已知: H 是状态函数(由前置 焓与焓变进阶); T 是状态量(温度由系统当前态决定); S 是状态函数(由前置 熵进阶, Clausius 不等式定义)。
原理: 状态量的组合(代数运算: 加减乘除)得到的量也是状态量, 因为状态量只依赖系统当前态。
结论: G ≡ H − TS = (状态量) − (状态量)·(状态量) = 状态量 ✓
推论: ΔG = G末 − G初 只依赖始末态, 与路径无关。这意味着可以用盖斯定律的精神处理 G —— 任何反应的 ΔG 可分步算。
STEP 3由 ΔSuniverse ≥ 0 严格推 ΔG ≤ 0 的完整链
图 1 · 由 ΔSuniverse ≥ 0 严格推 ΔG ≤ 0 的 4 步推导链。① 熵增定律(橙色, 起点); ② 等压条件下环境熵变 = −Qp / T = −ΔH / T (蓝色); ③ 代入 ① 得只涉及系统的不等式 ΔSsys − ΔH/T ≥ 0 (蓝色); ④ 两端乘 −T(T > 0, 不等号反转), 得 ΔG = ΔH − T·ΔS ≤ 0(红色, 终点)。底部蓝色结论: 两个不等式完全等价, ΔG 是 ΔSuniverse 的"系统视角翻译"。

关键观察 ⭐:

  • 不等号方向反转: ΔSuniverse ≥ 0 和 ΔG ≤ 0 一个朝上一个朝下, 这是"乘 −T(负数)"导致的代数反转, 不是物理矛盾。
  • T > 0 是必要条件: 推导中"两端乘 −T 反转不等号"成立要求 T 必须是正数。绝对温度 T 始终 > 0(零度 = 0 K 不可达, 第三定律), 所以这步永远合法。
  • "自由"的物理意义: G 的"自由" = 自由朝负方向变化的能量(在等温等压下); ΔG 越负, 反应越自发, 系统能"提取"的有用功越多(Card 2 推)。
思考点 1 · "乘 −T 反转不等号"的物理含义是什么？
推导中"两端乘 −T(T>0) 反转不等号"是关键代数步。它的物理含义是什么？为什么这个步骤把"宇宙朝熵增方向"翻译成了"系统朝 G 减小方向"?
物理上理解 ——
· "宇宙熵增" = ΔSuniverse ↑ = 自发方向
· "−T·ΔSuniverse" = -T × ↑ = ↓ (T > 0 不改变趋势, 但符号反了) = ΔG ↓ = 自发方向
· 同一物理事件("宇宙熵增")用两个等价语言描述: "宇宙视角"是 ΔSuniverse 升, "系统视角"是 ΔG 降。

为什么用"系统视角"? 因为算"宇宙熵变"需要算"环境熵变", 但环境是无限大的, 不可能直接测。而 ΔG = ΔH − TΔS 只涉及系统(系统的焓变和熵变都能测), 所以更实用。

类比: 像 "海平面上升" vs "陆地下降" — 两个等价描述, 视参考系而定。物理事件是同一个(海陆相对距离变化), 只是参考系换了。"乘 −T" 就是"换参考系"的操作, 不改变物理。
在纸上: ① 写"宇宙视角"和"系统视角"两个不等式; ② 用"海平面 vs 陆地"类比解释参考系切换; ③ 一句话总结"为什么 G 更实用"。
思考点 2 · 严格证明 ΔG 与路径无关 + 推论(任何反应 ΔG 可拆步算)
G ≡ H − TS 是状态函数, 所以 ΔG 路径无关。具体说: 对一个化学反应 R → P, ΔG = GP − GR, 不管反应经过什么中间步, 答案一样。
用这个性质, 你能想象一个"盖斯定律 for G"吗?
证明: H 是状态量(由 H = U + PV, U/P/V 都是状态量), T 是状态量, S 是状态量。由"状态量的代数组合仍是状态量"原理, G = H − TS 是状态量。所以 ΔG总 = ΔG步1 + ΔG步2 + ...

盖斯定律 for G: 任何反应的 ΔG 可以拆成已知反应 ΔG 的代数组合。
· 反应反向 → ΔG 变号
· 反应乘 k → ΔG 乘 k
· 反应叠加 → ΔG 相加

应用: 用标准生成自由能 ΔGf° 表(化学手册中)+ 盖斯定律 → 算任意反应的 ΔG°:
ΔG°_反应 = Σν·ΔGf°(产物) − Σν·ΔGf°(反应物)

这与基础板块 ΔG = ΔH − TΔS 是两种等价计算方法: 前者用 ΔGf° 表查, 后者用 ΔH + S 数据算。
在纸上: ① 证明 G 是状态量; ② 写"盖斯定律 for G"的 3 条规则; ③ 写"用 ΔGf° 表算 ΔG°"的公式。
ΔGf° 表是化学手册的另一个高频数据, 与 ΔHf° 表平行。用 ΔGf° 表能直接算反应 ΔG°, 不必先算 ΔH 和 ΔS。
STEP 4边界 — 本板块到哪里为止
  • 不引入: Helmholtz 自由能 A ≡ U − TS 的完整体系(等容等温的对应自由能 + 与 G 的对比)— 留给大学物理化学(广义热力学势的体系化讨论)
  • 不引入: 化学势 μ(偏导数 μ ≡ (∂G/∂n)T,P) 的完整定义 + 多组分系统的 Gibbs-Duhem 方程 — 留给大学物理化学
  • 不引入: 标准状态的严格定义(IUPAC 标准压力 1 bar / 标准浓度 1 mol/L / 标准态选择的依赖性) — 本板块用"约定标准态"即可
  • 本板块完成: G 严格定义 + 由熵增定律推 ΔG ≤ 0 + G 是状态函数 + 盖斯定律 for G
✓ Card 1 检查点

2

ΔG = Wmax,非膨胀 — 自由能的"有用功"意义

▼
学习目标: 推导 dG = -δW非膨胀,可逆 → 理解 ΔG 的物理意义 = 等温等压下系统能提取的最大非膨胀有用功 → 推出 ΔG = -nFE 电化学桥
建议时长: 50-60 分钟
STEP 1先自己想（5 分钟）

不查笔记，先答 ——

  • "自由能" 的名字暗示了"自由的能量"。你能猜什么意义上"自由"吗?
  • 系统的总能量分两部分: 等压条件下, 一部分变成 PV 膨胀功(体积变化), 一部分能做"其他用"(电功 / 力学功 / 等)。后者是"自由能"吗?
  • "可逆过程"和"不可逆过程"哪个能提取更多功? 为什么?
先回顾(第一定律 + 熵进阶):
前置 热力学第一定律 板块讲过 —— 功的分类: 总功 W 可分膨胀功 W膨 = ∫p外 dV(体积变化抵抗外压做的功)+ 非膨胀功 W非膨(其他形式的功, 如电功 / 表面张力做功 / 浓度梯度做功等)。
前置 熵的物理意义 板块讲过 —— 可逆过程提取最大功: 任何过程, 可逆路径能提取 / 输入的功 ≥ 不可逆路径(可逆是极限)。这是从 Clausius 不等式 + 第二定律推出的核心结论。
STEP 2推导 dG = -δW非膨胀,可逆
从第一 + 第二定律推 dG = -δW非膨胀,可逆
第一定律: dU = δQ − δW = δQ − δW膨 − δW非膨
可逆过程 + 第二定律: δQrev = T·dS
等压条件: δW膨 = p·dV
代入: dU = T·dS − p·dV − δW非膨,rev
焓定义 dH = dU + d(pV) = dU + p·dV + V·dp
等压 dp = 0: dH = dU + p·dV = (T·dS − p·dV − δW非膨,rev) + p·dV = T·dS − δW非膨,rev
G 定义 dG = dH − d(TS) = dH − T·dS − S·dT
等温 dT = 0: dG = dH − T·dS = (T·dS − δW非膨,rev) − T·dS = −δW非膨,rev ✓
积分: ΔG = −W非膨胀,可逆 = −Wmax,非膨胀(可逆给出最大功)
核心物理意义 ⭐⭐

在等温等压条件下:

\[ \Delta G = -W_{max, 非膨胀} \]
ΔG 的负值(若 ΔG < 0)= 系统能提取的最大非膨胀有用功(电功 / 表面张力做功 / 等其他形式)。

"自由"的含义: ΔG 是"可被自由用作有用功"的那部分焓变(其余 TΔS 部分被"熵增"消耗, 不能做有用功)。

关键观察:

  • 可逆 = 最大功: 不可逆过程提取的 W非膨 < |ΔG|(部分能量耗散为熵增)。只有可逆过程能等号成立提取 |ΔG|。
  • "等温等压"是约束: 这个 ΔG = -Wmax 的公式只在等温等压下严格成立。其他条件下(等容等温)用 Helmholtz 自由能 A 而不是 G(留给大学物化)。
  • "非膨胀"是约束: 膨胀功 p·dV 已经被"内置"在 ΔH 中(因为 dH = dU + p·dV 已经把 p·dV 加进去了), 所以 ΔG 只剩下"其他功"的潜力。
STEP 3电化学桥 ΔG = −nFE

最重要的"非膨胀功"应用 — 电化学电池。

电池的非膨胀功 = 电功 → ΔG = −nFE
电池可逆运行(理想电池, 无电阻发热): 当电池放电时, 通过外电路 n mol 电子(电荷量 = n·F, F = 96485 C/mol 是法拉第常数)。
外电路做功: 电功 W电 = (电荷)×(电压) = nF·E, 其中 E 是电池电动势(电池两极电势差, 单位 V)。
电池对外做的非膨胀功: W非膨 = W电 = nF·E
用 ΔG = -W非膨,可逆: ΔG = -nF·E ✓
电化学桥公式 ⭐⭐

\[ \Delta G = -nFE \]
n: 反应转移的电子摩尔数
F: 法拉第常数 = 96485 C/mol
E: 电池电动势(可逆条件下), 单位 V

意义: 这是热力学 ↔ 电化学的核心桥梁。可通过测电池电动势 E 直接推算反应 ΔG(精度高于其他方法), 也可通过 ΔG 数据预测电池电动势。
思考点 3 · 为什么"可逆 = 最大功"?
推导得 ΔG = -W非膨,可逆。不可逆过程提取的非膨胀功会比 |ΔG| 小。这背后的物理原因是什么? 用第二定律解释。
关键: 第二定律的 Clausius 不等式 δQ ≤ T·dS(等号成立于可逆过程)。
· 可逆过程: δQrev = T·dS, 能量"全部用尽" 不浪费
· 不可逆过程: δQirrev < T·dS, 部分能量被熵增"消耗掉"(转化为环境的低品质热能, 无法再做功)

在等温等压下, 这"被消耗的部分" = T·dS额外 = T(dSirrev − dSrev) — 是不可逆过程的"额外熵产生"乘以温度。

物理图像: 想象一个反应能放出 100 kJ 总能量(ΔG = -100 kJ)。
· 可逆电池: 慢慢放电, 几乎全部 100 kJ 变成电功(Wmax = 100 kJ)
· 不可逆电池(短路, 内阻发热): 部分能量变内阻热(不能做有用功), 比如只剩 70 kJ 变电功 + 30 kJ 变废热

这 30 kJ 就是"熵产生"的代价。可逆过程的"理想性"在于零熵产生, 100% 能量都能做有用功。
在纸上: ① 写 Clausius 不等式 δQ ≤ T·dS 及其等号条件; ② 用"可逆电池 vs 不可逆电池"的例子说明"额外熵产生"的物理意义; ③ 一句话总结"为什么可逆 = 最大功"。
思考点 4 · 用 ΔG = -nFE 算甲醇燃料电池的电动势
甲醇燃料电池反应: CH₃OH(l) + 3/2 O₂(g) → CO₂(g) + 2H₂O(l)
ΔG° ≈ -702 kJ/mol(25°C, 标准态)
反应转移电子数 n = 6 (CH₃OH 的 C 由 -2 价升到 +4 价, 转移 6 个电子)
F = 96485 C/mol

算电池的理论(可逆)电动势 E°。
用 ΔG = -nFE → E = -ΔG / (nF)

· E° = -ΔG° / (nF)
· = -(-702 × 1000 J/mol) / (6 × 96485 C/mol)
· = 702000 / 578910
· ≈ 1.21 V

这是甲醇燃料电池在可逆条件下的理论电动势。实际电池由于不可逆(内阻 + 电极极化 + 燃料未完全反应), 工作电压通常约 0.7-0.9 V, 能量转化效率约 60-75%。

核心收获: ΔG 不是抽象的"热力学量", 它直接决定一个反应能做的最大有用功。在燃料电池 / 电池 / 燃烧的"热机效率"分析中, ΔG 是关键参数。
在纸上: ① 写 ΔG = -nFE 公式; ② 代入数据算 E° = 1.21 V; ③ 说明实际工作电压低于此值的原因(不可逆); ④ 一句话总结"ΔG 与电池电动势的关系"。
这是"待建电化学大主题"的核心入口。本板块只到 ΔG = -nFE 公式的推导和单一电池电动势计算, Nernst 方程 / 浓差电池 / 复杂电极反应等留给电化学板块。

3

ΔG° + 化学平衡常数 K + 温度对 K 的影响

▼
学习目标: 推导 ΔG = ΔG° + RT ln Q 反应商公式 → 由平衡条件 ΔG = 0 推 ΔG° = -RT ln Keq 平衡常数桥 → 用范特霍夫方程定性判断温度对 K 的影响
建议时长: 50-60 分钟
STEP 1先自己想（5 分钟）

不查笔记，先答 ——

  • "化学平衡常数 K" 在化学课中常作为经验量给出。但 K 与热力学量 ΔG° 有什么定量关系?
  • 当反应"未到平衡"时, 反应有方向(向产物 or 反应物方向移动)。这个方向怎么由 ΔG 判断?
  • K 随温度变化的方向(吸热反应 T↑K↑, 放热反应 T↑K↓) — 这个经验规律的热力学根是什么?
⚠️ 跨主题前置 — 化学反应速率与化学平衡大主题:
本 Card 涉及的"反应商 Q / 平衡常数 Keq"属于化学反应速率与化学平衡大主题, 不在热力学主题内。严格定义见该大主题; 此处只给出最小自包含简介以使本 Card 能进行下去 ——
对一个一般反应 \(aA + bB \rightleftharpoons cC + dD\), 定义反应商 \(Q = \frac{[C]^c [D]^d}{[A]^a [B]^b}\)(气相用分压 / 溶液用浓度)。
反应达到平衡时, 反应商等于平衡常数 Keq: Keq = Qeq。
Keq 是反应的特征量, 只依赖温度(不依赖初始浓度)。
本板块直接使用此结论作为"跨主题输入"; 详细学习见化学反应速率与化学平衡大主题。
STEP 2推导 ΔG = ΔG° + RT ln Q + 平衡常数桥
反应商公式 ΔG = ΔG° + RT ln Q 的来源
背景: 在标准态(气体 1 bar / 溶液 1 mol/L) 下定义的 ΔG° 是反应的标准自由能变化。但实际反应物 / 产物浓度通常非标准, 需要"修正"。
关键事实(自包含): 理想气体 / 稀溶液的化学势 μi = μi° + RT·ln(ci/c°)(由统计力学推出, 本板块直接用)。对反应 aA + bB → cC + dD:
ΔG = Σν·μ产物 − Σν·μ反应物 = Σν·(μ° + RT·ln c) ⇒ ΔG = ΔG° + RT·(c·ln[C] + d·ln[D] − a·ln[A] − b·ln[B])
= ΔG° + RT·ln([C]^c [D]^d / [A]^a [B]^b) = ΔG° + RT·ln Q ✓
用法: 已知 ΔG°(查表 ΔGf°)和当前反应商 Q(从初始浓度算), 立即推出当前 ΔG。
· ΔG < 0 → 反应向产物方向自发(Q 太小 → 朝增大 Q 方向走 → 平衡时 Q = K)
· ΔG > 0 → 反应向反应物方向自发(Q 太大 → 朝减小 Q 方向走)
· ΔG = 0 → 达到平衡(Q = Keq)
由平衡条件推 ΔG° = -RT·ln Keq
平衡条件: ΔG = 0, Q = Keq
代入 ΔG° + RT·ln Q = 0: 0 = ΔG° + RT·ln Keq
整理: ΔG° = -RT·ln Keq ✓
平衡常数桥公式 ⭐⭐

\[ \Delta G^\circ = -RT \ln K_{eq} \]
或等价地: \( K_{eq} = e^{-\Delta G^\circ / (RT)} \)

意义: 这是热力学 ↔ 化学平衡的核心桥梁。可通过 ΔG° 数据(化学手册查 ΔGf° 算)直接算 Keq, 也可通过测得 Keq 反推 ΔG°。

关键对应关系 ⭐:

ΔG° 符号Keq 值反应在标准态下的方向
ΔG° = 0Keq = 1反应物 ↔ 产物等量(标准态平衡)
ΔG° < 0Keq > 1产物主导(反应向右进行较多)
ΔG° > 0Keq < 1反应物主导(反应向右较难, 平衡时产物少)
ΔG° 越负Keq 越大(指数关系)反应越"彻底"地朝产物方向
STEP 3具体例题 — 算合成氨 Keq

反应: \(N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)\)

已知数据(25°C, 标准态 1 bar):

  • ΔHf°(NH₃, g) = -45.9 kJ/mol
  • S°(N₂, g) = 191.6 J/(mol·K), S°(H₂, g) = 130.7 J/(mol·K), S°(NH₃, g) = 192.8 J/(mol·K)
3 步算 Keq
① 算 ΔH°: ΔH° = 2 × ΔHf°(NH₃) − [ΔHf°(N₂) + 3 × ΔHf°(H₂)] = 2 × (-45.9) − [0 + 0] = -91.8 kJ/mol
② 算 ΔS°: ΔS° = 2 × S°(NH₃) − [S°(N₂) + 3 × S°(H₂)] = 2 × 192.8 − [191.6 + 3 × 130.7] = 385.6 − 583.7 = -198.1 J/(mol·K)
③ 算 ΔG° = ΔH° − T·ΔS°(T = 298 K): ΔG° = -91800 − 298 × (-198.1) = -91800 + 59034 = -32766 J/mol ≈ -32.8 kJ/mol
用 ΔG° = -RT·ln Keq 反解: ln Keq = -ΔG° / (RT) = -(-32766) / (8.314 × 298) = 32766 / 2477.6 ≈ 13.22
Keq = e^13.22 ≈ 5.5 × 10⁵

结论: 25°C 下合成氨反应的 Keq ≈ 5.5 × 10⁵, 远大于 1, 意味着在标准态下产物 NH₃ 占绝对主导。

但实际工业上为什么需要"高温高压"?
尽管 Keq 在 25°C 下很大(热力学上有利), 但反应速率太慢(N≡N 945 kJ/mol 键能太强, 活化能很高)。所以工业 Haber-Bosch 工艺用 400-500°C(加速)+ 200-400 atm(进一步推平衡向右)+ 铁基催化剂(降活化能)。但温度升高会让 Keq 下降(本反应是放热, Card 3 STEP 4 范特霍夫方程会解释)。所以高温是速率 - 平衡常数的折中。
STEP 4温度对 K 的影响 — 范特霍夫方程
范特霍夫方程简化版的来源
从 ΔG° = -RT·ln Keq 出发: ln Keq = -ΔG° / (RT) = -(ΔH° − T·ΔS°) / (RT) = -ΔH°/RT + ΔS°/R
对 T 求导(假设 ΔH° 与 ΔS° 不随 T 变, 这是近似): d(ln Keq) / dT = ΔH° / (R·T²)
这就是范特霍夫方程简化版: \(\frac{d \ln K_{eq}}{dT} = \frac{\Delta H^\circ}{RT^2}\)
定性结论 ⭐:
· ΔH° > 0(吸热反应): T↑ → d(ln Keq)/dT > 0 → Keq 升高(温度升高有利产物形成)
· ΔH° < 0(放热反应): T↑ → d(ln Keq)/dT < 0 → Keq 下降(温度升高不利产物形成)
用合成氨验证: ΔH° = -91.8 kJ/mol < 0(放热)→ T↑ K↓ ✓ 与上面"工业用高温降低 K 的折中"一致。
图 2 · 合成氨反应 (ΔH° = -91.8 kJ/mol, ΔS° = -198.1 J/(mol·K)) 的 ΔG° vs T 线性图。左侧主图: 红色实线为 ΔG° vs T 关系, 与 0 轴(灰色虚线)交于绿色 T转 点。右侧"图例"框列出参数 — 红线截距 ΔH° < 0(放热)、斜率 -ΔS° > 0(因 ΔS° < 0)、T转 ≈ 463 K 是 ΔG° = 0 反转处, 左侧低温区 ΔG° < 0 产物主导(K > 1), 右侧高温区 ΔG° > 0 反应物主导(K < 1)。这条直线的 x 截距(与 0 轴交点)即基础板块的 T转 = ΔH°/ΔS° 概念。
思考点 5 · 合成氨为什么要"低温高压"? 用 ΔG° + 范特霍夫综合解释
合成氨反应 N₂ + 3H₂ ⇌ 2NH₃ 是放热(ΔH° < 0) + 气体分子数减少(ΔS° < 0)。"低温有利平衡向右(K↑) + 高压推平衡向右"是经典工业经验。
用本板块的 ΔG° + 范特霍夫定量解释这两个条件 — 为什么是低温(不是高温)? 为什么是高压(不是常压)?
两个条件 ——

① 低温(对平衡常数 Keq 有利):
· 用范特霍夫: d(ln Keq) / dT = ΔH° / (R·T²)
· ΔH° = -91.8 kJ/mol < 0 → d(ln Keq) / dT < 0 → T 升高 Keq 下降
· 25°C(298 K): Keq ≈ 5.5 × 10⁵ (上面算过)
· 工业 450°C(723 K): Keq 由直线推 ≈ 0.5(粗估; ΔG° ≈ -91.8 + 0.198 × 723 = +51.4 kJ/mol → ln K = -51400/6010 = -8.55 → K ≈ 2 × 10⁻⁴, 实际值与此近, 仅 0.0001 量级)
· 所以从纯平衡角度看, 应该低温有利 K↑。

② 高压(推动平衡向右):
· Kp = p(NH₃)² / [p(N₂) × p(H₂)³] (反应商 Q 的平衡形式)
· 总压力增加 N 倍 → 各分压增加 N 倍
· 分母变 N × N³ = N⁴; 分子变 N²
· Q 变化 = (N²) / (N⁴) = 1/N²
· 即压力增加, Q < K, 反应向右移动直到 Q 重新等于 K → 平衡时 NH₃ 浓度增加
· 这是气体分子数减少的反应对压力敏感的本质。

实际工业折中: 低温利好 K, 但反应速率太慢(N≡N 945 kJ/mol 键能太强)。工业上选 400-500°C: 速率高 + 用铁催化剂降活化能, 同时高压(200-400 atm)弥补 K 在高温下的下降。这是热力学(K) vs 动力学(速率) 的经典权衡。

核心收获: 化学工业不仅看"反应是否自发(ΔG < 0)", 还要看"K 多大 + 速率多快 + 经济成本"。本板块给的工具(ΔG° = -RT ln K + 范特霍夫)是热力学侧的核心, 与动力学侧(活化能 + 催化剂)配合才能设计真实工艺。
在纸上: ① 用范特霍夫定量解释低温有利 K(d ln K/dT 符号 + 25°C vs 450°C 数值估算); ② 用 Q vs K 解释高压推平衡(N⁴ 分母 vs N² 分子); ③ 一句话总结"热力学(K) + 动力学(速率) + 经济(成本) 的工业折中"。
