📋 听点总表（全部 17 锚 · ⭐ = 核心听点 · 4 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义 / 模型⭐ 微观模型四大假设H1 大量粒子 / H2 随机运动各方向等概率 / H3 弹性碰壁 / H4 无分子间作用力 ⇒ 无势能。
立方体容器模型V = x³、A = x²；形状不影响结论，只是推导脚手架。
关键公式⭐ 单粒子碰壁链Δp = 2mv_x → Δt = 2x/v_x → F = mv_x²/x → P₁ = mv_x²/V，逐步指认假设。
⭐ 各向同性 1/3v² = v_x²+v_y²+v_z²；由 H2 ⇒ v̄_x² = v̄²/3。
⭐⭐ 压强微观公式P = Nmv̄²/(3V)；本板块核心枢纽公式。
平均动能含义½mv̄² = 单粒子平均（平动）动能；N·½mv̄² = 系统总动能。
⭐ U = (3/2)PV由 H4（无势能）⇒ 内能 = 总动能 ⇒ 改写 3PV = Nmv̄² 得 U = (3/2)PV。
⭐⭐ U = (3/2)nRT核心结论：联立 PV = nRT；能在纸上写出并指认 n、R、T。
ΔU = (3/2)nRΔT推论：ΔU 只与 ΔT 有关 ⇒ 等温 ΔU = 0；自由膨胀理想气体 T 不变。
符号约定N vs n大写 N = 粒子总数 / 小写 n = 物质的量（mol），严格区分。
v̄² 读法速度平方的平均，不是平均速度的平方。
思维方式⭐ U 只与 T 有关U 表达式不含 V、P 不是巧合，根在 H4（无分子间作用力 ⇒ 无势能）。
⭐ 温度的微观意义U = (3/2)nRT ⇒ 温度本质 = 平均动能的度量（对单原子理想气体）。
假设审查思维每步标注用到哪条 H1–H4；能讨论「去掉 H? ⇒ 结论怎么变」。
统计平均的桥梁从单粒子到 N 粒子靠统计平均（依赖 H1 + H2）。
边界守门
不展开		不展开双原子 / 多原子自由度(3/2) 系数只对单原子；双原子定量留给后续 / 大学。
不展开真实气体 / 范德瓦尔斯方向肯定但不推方程数值。
不写玻尔兹曼 (3/2)kT 形式本板块全程用 nR 宏观形式，单分子动能形式留给统计力学。
不展开麦克斯韦速率分布只用「均方速度 v̄²」一个平均量，分布形式留给统计力学。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 U = (3/2)nRT 或 (3/2)PV 并指认每项含义与适用前提」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
