焓与焓变的意义 · 深度口试 · 题库

关于深度口试

口试是热力学第一定律板块的真实通关环节：先把学习任务卡学完、基础检测和熟练度检测练熟，然后预约口试。本页是 完整题库，公开给所有学生 —— 你可以提前准备每一条主问题。

口试由教师当面进行，从下方 6 条主问题中抽 3 条。每条主问题先开放讲，再由教师根据你的回答追问。讲思路比说出唯一正确答案重要 —— 教师听的是因果重建，不是背诵。

📌 通关条件：抽中的 3 条里，至少 2 条把核心要点讲清楚并能回应一层「为什么」，且整场至少一次在纸上写出 ΔU = Q − W 并指着式子说清每一项含义。

口试流程 总时长 15-20 分钟

① 暖场 · 自述
0-3 分钟
你先讲：「这个板块学到的最重要的一件事是什么？为什么重要？」—— 不卡时间，讲到能切入主问题就行。
② 主问题 ×3
3-15 分钟
从下方 6 条主问题中由教师抽 3 条，每条约 3-4 分钟。先开放讲，再回答追问。
③ 错题联动
15-18 分钟
带 1-2 道自己曾经做错的题（基础检测或熟练度检测里的），现场讲「当时为什么错、现在怎么想」。
④ 判定 · 签字
18-20 分钟
通过 = 当场签字；暂缓 = 给补学方向 + 预约二试（换未用过的主问题）；不通过 = 回到学习阶段重学。
主问题 1

Q 不是状态函数 · 假想热含量反例

约 4-5 分钟
🎤 开放主问：想象有人这样定义：「一个系统的『热含量 HC』就是它累计吸收过的总热量。」请你设计一个具体的循环过程，从这个定义出发，把矛盾讲清楚 —— 我要听你用第一定律完成代入论证，不是只说『Q 不是状态函数』这句结论。
👂 教师在听这些要点
  • ⭐⭐ 构造循环：回到原状态但 Q ≠ 0 → HC「同一状态两个值」
  • ⭐ 用 ΔU = 0（循环）+ ΔU = Q − W 推出 Q = W ≠ 0
  • ⭐ 总结：物理上需要一个「描述热的状态函数」 —— 这就是引入 H 的动机
  • 复现：状态函数 = 只看当前状态；过程量 = 看路径
主问题 2

H ≡ U + PV · 为什么这个组合是状态函数

约 4-5 分钟
🎤 开放主问：焓的定义是 H ≡ U + PV。请告诉我两件事：第一，为什么仅凭这条等式就能断定 H 一定是状态函数（不需要做任何实验）？第二，把它写成 ΔH = ΔU + Δ(PV) 时，那个 Δ(PV) 和「Pₑₓₜ·ΔV」（外界压强乘体积变化）是不是一回事？
👂 教师在听这些要点
  • ⭐⭐ 复述定义 H ≡ U + PV（在任意状态下都成立，与是否恒压无关）
  • ⭐⭐ 论证：U、P、V 都是状态量 → 状态量的代数组合仍是状态量 → H 必是状态函数
  • ⭐ 分清 Δ(PV) = P₂V₂ − P₁V₁（状态差）vs P·ΔV（过程量，等压下才退化为前者）
  • 复现：状态函数的加减仍是状态函数（前置板块结论）
主问题 3

反向推理 · 恒压下 ΔH = Q_p 怎么逼出来的

约 5-6 分钟
🎤 开放主问：请你在纸上完成「反向推理」：从「我们想找一个量，让它在恒压下恰好等于 Q_p」这个需求出发，一步一步逼出 H = U + PV 的形式。整条推导必须用第一定律，且明确指出「恒压」这个条件用在哪一步。
👂 教师在听这些要点
  • ⭐ 起手就讲「这是从需求倒推」，不是直接套定义
  • ⭐ Q = ΔU + W_out（系统对外做功视角）作为起点
  • ⭐ 恒压下 W_out = P·ΔV
  • ⭐⭐ 恒压下 P·ΔV = Δ(PV)（因为 ΔP = 0，写开两项验证）
  • ⭐⭐ 代入得 Q_p = Δ(U + PV) = ΔH，即 ΔH ≡ Q_p（恒压下）
  • ⭐ 能精确指出「恒压」用在哪一步：把 P·ΔV 改写成 Δ(PV) 那步
主问题 4

符号双视角 · +74 kJ 与 ΔH = −74 kJ/mol

约 4-5 分钟
🎤 开放主问：甲烷燃烧反应有两种常见写法：化学方程式末尾写「+74 kJ」（环境视角）；热化学方程式写「ΔH = −74 kJ/mol」（系统视角）。请告诉我：这两个数为什么符号相反、物理上其实是同一件事？再讲一下，对应到能量阶梯图上，产物的焓位置在哪儿？
👂 教师在听这些要点
  • ⭐⭐ 讲清两种写法是两种视角（外界视角 vs 系统视角），不是两件事
  • ⭐ ΔH < 0 ↔ 放热（exothermic）；ΔH > 0 ↔ 吸热（endothermic）
  • ⭐ Q_p 流入系统为正；放热则 Q_p < 0；恒压下 ΔH = Q_p 同号
  • ⭐ 放热反应产物焓更低（能量阶梯图：产物在下）
  • 复现能量阶梯图：起点是反应物焓位置，箭头方向 = ΔH 符号
主问题 5

标准生成焓 · 海平面思路与可查表

约 4-5 分钟
🎤 开放主问：标准生成焓 ΔH_f° 的定义里有两个看似奇怪的约定：（一）以「标准条件（25°C、1 atm）由稳定单质合成 1 mol 化合物」为参考过程；（二）「稳定单质」按定义 ΔH_f° = 0。请你讲清楚：这两个约定为什么是合理的？类比「海平面」会帮上什么？
👂 教师在听这些要点
  • ⭐⭐ 复述定义：标准条件 + 稳定单质 + 合成 1 mol 化合物 + 单位 kJ/mol
  • ⭐⭐ 「稳定单质」按定义 ΔH_f° = 0（这是约定 / 参考零点，不是测量结果）
  • ⭐ 海平面类比：绝对高度不可测，但差值（海拔）可比较；H 同理 —— 我们造一个零点
  • 下标 f = formation；上标 ° = 标准条件；单位 kJ/mol
  • ⭐ 「可查表」是革命：每个化合物有一个 ΔH_f°，化学计算从「测」变成「查」
主问题 6

稳定性的相对性 · 边界守门

约 4-5 分钟
🎤 开放主问：「CH₄ 比 C + 2H₂ 稳定」—— 这句话用 ΔH_f° 怎么读出？「CH₄ 比 CO₂ + H₂O 不稳定」—— 这句话又怎么读出？最后一问：如果有人由此推出「燃烧反应一定自发」，错在哪里？
👂 教师在听这些要点
  • ⭐⭐ 「稳定性」是相对的（CH₄ 相对于 C+2H₂ 更低 / 相对于燃烧产物更高）
  • ⭐ 用「焓更低 = 能量上更稳定」读出第一句
  • ⭐⭐ 边界声明：「稳定 ≠ 自发」 —— 自发还涉及熵 / 温度，本板块不展开
  • 辅助：ΔH_逆 = −ΔH_正（H 为状态函数推论）

📝 怎么准备

  • 不要背口诀。教师追问的第一刀往往就是「为什么」—— 把每个核心结论的来源 + 一句话理由想清楚（比如：等温 ΔU = 0 的根据是「理想气体 U 只依赖 T」，这条结论来自分子动理论板块）。
  • 带纸和笔。主问题 1、2、3、5、6 都涉及在纸上写公式或画图，画完再讲思路。
  • 提前挑 1-2 道自己错过的题（基础检测、熟练度检测、或纸笔检测里的），准备讲「当时为什么错、现在怎么想」。这是流程第 ③ 步。
  • 边界声明给你「松绑」：本板块有些「更深层」的概念不在要求内，没学过也不扣分；如果主动谈到是亮点，教师会简短肯定但不展开（这些深入内容在后续板块或进阶板块里学）。
  • 讲思路 ＞ 算对答案。教师听的是因果重建：你能不能把「为什么这样、不那样」讲清楚，比说出唯一正确数值更重要。
