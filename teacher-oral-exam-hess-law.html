📋 听点总表（全部锚 · ⭐ = 核心听点 · 4 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义⭐ 盖斯定律的表述「任何反应，无论一步还是多步完成，总焓变相同」；等价形式 ΔH总 = Σ ΔH各步；前提是首尾相接、始末态对齐。
⭐ 盖斯定律是"命名"而非新原理本质上是「H 是状态函数」在化学反应场景下的代数推论（5 行代数）；不是凭空多出的新公理。
关键结论⭐⭐ 反应焓变查表公式能在纸上写 ΔH反应 = Σν·ΔHf°(产物) − Σν·ΔHf°(反应物)，并指着说出每一项：差结构 = 末减初；ν 是化学计量数；稳定单质 ΔHf° = 0。
⭐ 路径无关性的代数形式ΔH直接 = ΔH1 + ΔH2 + … + ΔHN；根据是 H 状态函数 + ΔH = H末 − H初。
⭐ 两步法（借稳定单质中介）任意反应 R → P 拆成 R → 稳定单质 → P，让所有反应都能用同一张 ΔHf° 表查到。
双重验证（代数组合 vs 查表）同一反应用代数组合法和查表代入法算出的 ΔH 必然一致；两种方法同源于 H 是状态函数。
代数 4 规则⭐ 加法 ① + ②两反应首尾相接 ⇒ ΔH总 = ΔH1 + ΔH2。
⭐ 减法 ① − ②① − ② = ① + 反向②；ΔH差 = ΔH1 − ΔH2。
⭐ 反向规则方程左右翻 ⇒ ΔH反向 = −ΔH1；根据是 ΔH = H末 − H初 在始末态对换时取负。
⭐ 乘 k 倍k × ① ⇒ ΔHk·1 = k · ΔH1；ΔH 是「mol 反应」单位的代数量。
⭐ 4 规则统一根据差结构 ΔH = H末 − H初 ⇐ H 是状态函数；一个根基支撑 4 条规则。
工程方法净计量数法对每物质 X 算 ν净(X) = 右系数 − 左系数；只出现在一个方程的物质直接定系数，其余用来验证。
3 反应组合判反向共同物质两边都是右 ⇒ 反其中一个；两边都是左 ⇒ 反其中一个；一左一右 ⇒ 都不反直接叠加。
思维方式零点参考是约定稳定单质 ΔHf° = 0 是约定，不是物理事实；选稳定单质做零点是因为所有反应物 / 产物都可由元素拆出，使查表能 cover 所有反应。
路径无关从物理延伸到化学路径无关（U / S 在物理板块）延伸到化学反应：测不到的反应可借能测的反应算出。
边界引子⭐ 放热 ≠ 自发反例：金刚石 → 石墨 ΔH ≈ −1.9 kJ/mol（放热）常温稳定数百万年；冰融化 ΔH>0（吸热）却在 0 °C 以上自发。
稳定性是相对的CH4 比 C + 2H2 稳定，但比 CO2 + 2H2O 不稳定 —— 需要重新审视「稳定」含义。
边界守门
不展开		不引熵 S / 自发性判据留给吉布斯自由能板块。
不推 H = U + PV 的严格定义本板块只用 H 状态函数的代数结论，严格定义留给焓与焓变进阶板块。
不展开反应动力学本板块只谈「反应若发生 ΔH 多少」，不谈「反应发生得多快」，留给化学动力学板块。
不引吉布斯自由能 G统一判据「会不会发生」留给吉布斯自由能板块。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 ΔH = Σν·ΔHf°(产物) − Σν·ΔHf°(反应物) 并逐项指认」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
