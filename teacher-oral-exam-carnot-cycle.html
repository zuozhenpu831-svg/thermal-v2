📋 听点总表（全部 17 锚 · ⭐ = 核心听点 · 3 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义 / 模型⭐ 理想热机模型剥离零件细节，只保留「T₁ 热源 → 热机 → T₂ 冷源 + W 输出」主干结构；卡诺为问「上限在哪」造的模型。
⭐ 卡诺循环四步结构等温膨胀（T₁ 吸热）→ 绝热膨胀（降温）→ 等温压缩（T₂ 放热）→ 绝热压缩（升温）；等温管热进出 / 绝热管 T 切换。
无限大热源 / 冷源热容无穷大 ⇒ T₁、T₂ 恒定，作为常数写进公式。
等温与绝热段⭐ 等温功 W = nRT ln(V₂/V₁)由 PV=nRT → ∫P dV → nRT ln 比值；T 是常数提到积分外。
⭐ 等温 ΔU=0 ⇒ Q=W根据：理想气体 U 只依赖 T；吸的热全变功。
绝热 TV^(γ−1)=const理想气体绝热关系；联立两段绝热 ⇒ V_B/V_A = V_C/V_D。
净功⭐ 净功 = 闭环面积 / W_net = Q₁ − Q₂P-V 图闭合曲线面积；循环 ΔU=0 给出净功公式。
Q_h、Q_c 取正值本板块约定：Q 取流动的量，方向用流入 / 流出区分。
效率公式⭐⭐ 卡诺效率 η = 1 − T₂/T₁纸上写出 + 逐项指认；T 必须用绝对温度（K）；T₁>T₂ 高在前。本板块核心成果。
⭐ Q₂/T₂ = Q₁/T₁效率公式等价变形；比值守恒指向后续「熵」。
找效率上限的思维方式问「一切热机的效率上限」而不是「这一台还能不能更好」—— 问题层级提升本身是物理学方法。
卡诺定理 / 边界⭐ 卡诺定理 · 工质无关给定 T₁、T₂，一切可逆热机效率 = 卡诺效率；不可逆机严格更小；与工质无关。
第二定律两种表述引入克劳修斯（热不能自发冷→热）+ 开尔文（单一热源不可能全转化为功）；本板块只初步引入。
角标约定 T₁>T₂1=高温，2=低温；化学语境也写 T_h、T_c。
边界守门
不展开		不引熵的定义 ΔS=Q_rev/T留给状态函数与非状态函数板块。
不推第二定律完整内容克劳修斯不等式、不可逆性量化留给熵的物理意义板块。
不讲卡诺过程的微观解释统计力学视角留给熵的物理意义板块。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 η = 1 − T₂/T₁ + 声明 T 为 K + 逐项指认」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
