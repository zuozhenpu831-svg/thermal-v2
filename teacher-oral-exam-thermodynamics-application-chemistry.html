📋 听点总表（全部 23 锚 · ⭐ = 核心听点 · 6 项板块边界守门）
类别听点学生说到位时，话里应包含什么
定义系统 + 外界模型把研究对象框出来当「系统」，框外是「外界」；能量交换只有热 Q 与做功 W 两条通道。
⭐ 热量 Q 符号约定系统吸热 Q＞0 / 放热 Q＜0；正负是「替系统记账」，不是由温度高低直接决定。
⭐ 功 W 符号约定本板块统一约定 W = 系统对外做功为正；被压缩（外界做功）时 W＜0。
关键结论⭐⭐ 第一定律 ΔU = Q − W能在纸上写出，并指着每一项说出含义与正方向；本板块核心成果。
能量守恒的具体形式第一定律不是新原理，是能量守恒把「热」包含进来后的热力学形式。
⭐ 第一类永动机不可能能做代入论证：Q = 0 且 ΔU = 0 ⇒ W = 0，与「持续对外做功」矛盾。
⭐ 状态量 vs 过程量U 状态量 / Q、W 过程量；同始末态 ΔU 必同、Q W 可不同；循环 ΔU = 0。
⭐ 热 vs 做功的微观差异热传导 = 分子无序碰撞传能；做功 = 分子有序定向位移传能。
气体做功 W = p·ΔV等压下由 F = pA、W = FΔx 推出；几何上 = P-V 图过程线下方面积。
4 过程变形⭐ 等容：ΔU = QV 不变 ⇒ W = 0，吸热全部变内能。
⭐ 等压：ΔU = Q − p·ΔV吸热一部分变内能、一部分对外做功。
等温：ΔU = 0，Q = W根据：理想气体 U 只依赖 T。
绝热：Q = 0，ΔU = −W做功完全来自内能消耗 ⇒ 绝热膨胀温度下降。
绝热 P-V 线比等温更陡双重作用论证：等温只有 V↑→P↓；绝热 V↑ 的同时 T↓，P 被双重压低。
复合过程复合过程 3 步法拆段 → 每段 ΔUi = Qi − Wi → 累加。
思维方式宏观对称 · 微观不对称Q、W 地位对称（都改 ΔU），微观图像截然不同（无序 vs 有序）。
第二定律的引子「从单一热源吸热全变功」不违反守恒但仍不行 —— 留给下个板块。

📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」（capstone：跨板块连接性听点至少命中 1 条）。
  • 整场至少完成一道综合题的纸面解答步骤（顶部 📝 勾选；可来自教师抽题时指定的综合题）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
