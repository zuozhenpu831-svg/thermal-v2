热力学第二定律 · 基础

自然过程为什么是单向的 · 从能量"数量"到能量"方向"的关键飞跃
模块名称热力学第二定律 · 基础（基础模块）
所属大主题热力学 · 物理化学融合主题
预计学习时间2-2.5 小时（按自己的节奏）

所需工具AI 工具、纸笔；对第一定律"能量守恒"与"\(Q, W\) 过程量"的熟练度

学习目标

  1. 能陈述第二定律的两种等价形式: 克劳修斯陈述（热不能自发从低温物体流向高温物体）+ 开尔文陈述（不可能从单一热源吸热使之全部转化为有用的功）
  2. 能用第二定律解释第二类永动机不可能（与第一定律对比：第二类不违反能量守恒，但违反"方向"）
  3. 能用"自发方向"描述自然过程：热从高 → 低 / 气体扩散 / 摩擦消耗机械能 / 油墨入水 等都是单向的
  4. 能定性论证热机效率 \(\eta < 1\) 永远不可达 100% —— 任何热机都必须向低温源排放部分热量
  5. 能用"无序度"定性介绍熵 \(S\) —— 自发过程熵增加（本板块不引入 \(dS = dQ/T\) 公式 / 玻尔兹曼公式）
  6. 能解释第二定律是独立于第一定律的新原理 —— 第一定律管"数量"（能不能守恒），第二定律管"方向"（会不会发生）。两者独立而互补
  7. 知道本板块到定性陈述为止：严格推导 + 数学形式 + 微观统计解释全部留给进阶板块的卡诺循环 / 状态函数 / 熵的物理意义
驱动问题

三个让"第一定律"也答不上来的问题 ——

  • 一杯热水放在桌上，一会儿凉了。第一定律完全允许它反向发生（室温水自发把环境的能量吸进来再变热）—— 但实际从未观察到。为什么自然过程是单向的？
  • 摩擦生热可以发生（功 → 热）。那"反过来"呢 —— 桌上一块铁突然变冷，释放出的热量推动桌子向前滑动？第一定律完全允许（能量守恒）。为什么从未发生？
  • 燃料烧的总能量都满足能量守恒，那为什么热机的效率不能 100%？为什么必须有"废热"排出？

这三个问题的共同点：第一定律完全允许，自然却从不发生。这告诉我们 —— 第一定律不够了。需要一个新的独立原理，专门管"方向" —— 这就是热力学第二定律。

（提示：第一定律 = 能量"数量"判据（能不能守恒）。第二定律 = 能量"方向"判据（会不会自发发生）。两个判据缺一不可。）

本板块只到定性陈述为止。如果你已掌握以下 3 个进阶板块（它们之间存在前置链 ——卡诺循环 → 状态函数 → 熵的物理意义），建议按顺序学完再回来读本板块，你会从"宏观陈述"层升到"微观原理"层：

1

第二定律的两种陈述 + 自发方向

▼
学习目标：从"反向过程从不发生"的现象切入 → 引出第二定律的两种历史陈述（克劳修斯 + 开尔文）→ 抽象出"自发方向 / 不可逆性"的概念 → 明确第二定律是独立于第一定律的新原理
建议时长：50-60 分钟
STEP 1先自己想（5-8 分钟）

列举几个"反向过程从不发生"的例子 ——

  • 热水放凉了。从未看到室温水自发变热 —— 即使第一定律允许（吸收环境的能量）
  • 一滴墨水滴入清水，慢慢均匀染色。从未看到均匀染色的水自发把墨水重新汇聚成一滴
  • 气体进入抽真空的容器，会均匀充满。从未看到均匀分布的气体自发挤回原来那一角
  • 用力推一辆小车（做功）→ 摩擦让小车减速并发热。从未看到桌面自发变冷、把热量"还给"小车让它再次加速

这些都是 "第一定律允许、但实际从不发生" 的现象。请自己想：

  • 这些现象有什么共同点？
  • 能不能用一个统一的判据描述"什么过程能自发发生 / 什么不能"？
先回顾：热力学第一定律 板块讲过 \(\Delta U = Q - W\)，以及 Card 1 思考点 2 留了一个引子 —— "第二类永动机不违反第一定律但仍不可能"。本板块就要把这个引子展开，给出第二定律的完整陈述。
STEP 2克劳修斯陈述（1850 年）

1850 年，德国物理学家克劳修斯（Rudolf Clausius）研究热机和制冷机的工作过程后，提出 ——

克劳修斯陈述 ⭐: 不可能把热量从低温物体传到高温物体而不引起其他变化。

等价说法：热不能自发地从低温流向高温。

（"自发"= 不借助外力、不引起其他变化）
图 1 · 克劳修斯陈述示意。红色方框为高温物体（温度 Th），蓝色方框为低温物体（温度 Tc）。从高温物体到低温物体的红色实线箭头表示自发方向（自动发生 / 不需外力 / 标注 "✓ 自发"）。从低温物体到高温物体的灰色虚线箭头表示不自发方向（标注 "✗ 非自发"；要发生需要消耗外功，如制冷机 / 空调，但那已经"引起了其他变化"）。两条箭头共用同一对热源，方向相反，对比鲜明。

关键澄清：

  • "自发"的严格意义 = 不引起其他变化。空调 / 冰箱能把热从低温流到高温，但消耗了电能并改变了电网状态 → 不算"自发"
  • 这里说的"自发"不意味着"必然能在短时间内完成" — 把"会自然发生"和"何时发生"是两件事
  • 克劳修斯陈述不否认"低温向高温的热流" — 它只说这种过程不会自发发生，需要"付出代价"（外功 / 其他变化）
STEP 3开尔文陈述（1851 年）

同年，开尔文勋爵（William Thomson, Lord Kelvin）从研究热机效率的角度给出另一种陈述 ——

开尔文陈述 ⭐: 不可能从单一热源吸取热量，全部用来做功而不引起其他变化。

等价说法：不能制造"只从一个热源吸热、全部转化为有用功"的循环热机。

开尔文陈述说的是"完美热机"的不可能 ——

  • 第一定律允许 \(Q = W\)（吸的热量全部变成功）—— 这满足能量守恒
  • 但开尔文说：这种"100% 转化"的热机不可能存在 —— 必须有一部分热量"浪费" 给低温物体（变成废热）
  • 这就是"第二类永动机不可能"的物理含义（Card 2 详细分析）
思考点 1 · 第一定律允许的为什么实际不发生？
驱动问题里的三个"反向过程"（热水自发变热 / 桌子自发被热推动 / 100% 效率热机）都满足第一定律的能量守恒。为什么不发生？用第二定律的两种陈述逐一分析。
① 热水自发变热 = 低温环境自发把热流到高温热水 → 直接违反克劳修斯陈述（热不能自发从低温流向高温）。
② 桌面自发变冷推桌子 = 单一热源（桌面）吸热全部变功（推桌子的机械能） → 直接违反开尔文陈述（不能单一热源全部转功）。
③ 100% 效率热机 = 单一高温热源吸热全部变功，无废热 → 直接违反开尔文陈述。
所以这三个"第一定律允许的反向过程"都被第二定律"挡住"了 —— 这正是为什么第二定律是必需的、独立的。
在纸上：① 写每个反向过程违反哪一条陈述；② 一句话总结"第一定律不够 + 第二定律补上"。
STEP 4两陈述等价 + 自发方向 + 不可逆性

两种陈述看似无关（一个讲"热流向"，一个讲"热机效率"），实际上逻辑等价 —— 由任意一条可以推出另一条（严格证明留给进阶 卡诺循环 板块）。

从两种陈述抽象出更深的核心概念 ——

自发方向 ⭐: 自然界的过程都有单向性。有些方向"自动发生"（自发），反方向"不自动发生"（非自发）。
举例: 热从高 → 低 / 气体从高密度 → 均匀分布 / 油墨从一滴 → 均匀染色 / 机械能 → 内能（摩擦生热）。所有这些反向过程都不自发。
不可逆性 ⭐: 一个自发过程的"反向"不会自发发生 —— 这个过程是不可逆的。
"不可逆"不是说"绝对做不到反向"，而是说"反向不会自发发生"。要走反向必须付出代价（消耗外功 / 引起其他变化）。

不可逆性的常见例子 ——

自发过程反向（不自发）
热水放凉（热 → 低温环境）凉水自发变热
气体扩散均匀气体自发聚回一角
油墨入水均匀染色染色水自发汇聚墨水
摩擦消耗机械能 → 热热自发推动物体加速
鸡蛋摔碎碎片自发重组为完整鸡蛋
思考点 2 · 自发方向的判据（定性）
能不能从上述例子里"抽象出"一个统一的判据 —— 看一个过程，能不能立刻判断它是"自发"还是"非自发"？
观察上面的例子，自发方向都对应系统从"有序" → "无序":
· 热从高 → 低 = 温度差变小（从"有差别" → "更均匀"）
· 气体扩散 = 分子分布从"聚集" → "均匀"
· 油墨入水 = 分子分布从"集中一滴" → "均匀分散"
· 摩擦生热 = 机械能（有序的整体动能）→ 内能（无序的分子热运动）
· 鸡蛋摔碎 = "有序结构" → "无序碎片"
这个"无序度"的概念，物理上正式名字叫熵 \(S\)（Card 3 详细讲）。
自发判据（定性）⭐: 自发过程总是让系统"无序度增加"（孤立系统熵增）。这就是第二定律的"统一判据"，统一了克劳修斯陈述、开尔文陈述、以及所有的不可逆性观察。
在纸上：① 把 5 个例子各自标注"从 _ 到 _"，看看是不是都对应"有序 → 无序"；② 写下"自发判据 = 无序度增加 = 熵增"；③ 一句话总结"为什么第二定律的核心是'方向'问题"。
熵增原理的严格形式留给进阶熵的物理意义板块。本板块定性掌握"自发 = 无序度增加"足以解释所有第二定律现象。
✓ Card 1 检查点
素养字段：物理判据 · 概念抽象 · 自然观（单向性与时间箭头）
2

第二类永动机不可能 + 热机效率定性

▼
学习目标：用开尔文陈述论证"第二类永动机不可能" → 分析实际热机的能量流向 → 定性论证 \(\eta < 1\) 的不可避免性
建议时长：30-40 分钟
STEP 1第二类永动机 — 第一定律允许、第二定律否决
第一类 vs 第二类永动机:
· 第一类: "不消耗任何能量却能持续对外做功" → 违反能量守恒（第一定律）
· 第二类: 从单一热源吸取热量，全部转化为功 → 满足能量守恒（\(Q = W\)）
两类永动机都不可能，但理由不同：
  • 第一类违反第一定律
  • 第二类不违反第一定律，但违反开尔文陈述（第二定律）
这两类永动机的不可能同时展示了第一定律和第二定律的独立作用。
思考点 3 · 第二类永动机的"诱惑"
第二类永动机听起来非常美好：海洋里有无穷的热量，按理只要"吸海水的热全部变功"，就能解决人类能源问题。第一定律不反对（能量守恒满足）。但实际不行 —— 为什么？
关键在"全部"两个字。开尔文陈述明确说：不能"全部"。任何热机都必须有一部分热量"漏" 给低温热源（即废热）。
如果你"只用海洋这一个热源"工作：你的热机吸海水的热 \(Q\)，必须排放 \(Q_c\) 给低温源（如大气 / 深海）才能继续运行，自身做的功只是 \(W = Q - Q_c < Q\)，不可能 100%。
"排废热"不是工程局限，是物理原则上不可避免的（开尔文陈述的直接结论）。所以"海洋发电厂"这种想法在物理原理上就不成立 — 这是为什么我们要找"温差"（如海洋温差发电、地热）而不是"单一恒温热源"。
在纸上：① 写"第二类永动机违反开尔文陈述"的逻辑；② 解释为什么"温差发电"行而"恒温源发电"不行；③ 一句话总结"热机必须有温差"。
STEP 2实际热机的能量流向

所有实际工作的热机（蒸汽机 / 内燃机 / 燃气轮机 / 火力发电厂）都遵循同一个能量流向 ——

图 2 · 热机能量流向。红色高温热源（Th）将热量 Qh（红色向下箭头）传给中间的"热机"（蓝色圆）；热机将其中一部分 W（绿色向右箭头）作为有用功输出（如推动汽车 / 发电），其余部分 Qc（蓝色向下箭头）作为废热排放给低温热源（Tc）。能量守恒（第一定律）：Qh = W + Qc。效率 η = W / Qh = 1 − Qc / Qh。开尔文陈述要求 Qc > 0（必须有废热）→ η < 1（不可达 100%）。

关键论证（开尔文陈述 → η < 1）:

  • 能量守恒（第一定律）: \(Q_h = W + Q_c\)，所以 \(W = Q_h - Q_c\)
  • 效率定义: \(\eta = W / Q_h = (Q_h - Q_c) / Q_h = 1 - Q_c / Q_h\)
  • 开尔文陈述: 不能有 \(Q_c = 0\)（否则就是"从单一热源吸热全部转功"）
  • 所以 \(Q_c > 0\) → \(\eta < 1\) ⭐ —— 任何热机的效率都不可能达到 100%
思考点 4 · 实际热机效率示例
为什么实际热机的效率都很低？查一下：① 蒸汽机大约多少；② 内燃机大约多少；③ 火力发电厂大约多少。这些值都远低于 100% —— 不只是工程局限，更是物理原则上的限制。
① 蒸汽机 ~ 10-20%（早期）/ ~ 30% （现代）。② 汽油内燃机 ~ 25-30%，柴油 ~ 35-45%。③ 火力发电 ~ 40-50%。
这些远低于 100% 的第一原因是物理原则上的限制：开尔文陈述 → 必须有废热。
第二原因是工程局限：摩擦、漏热、不可逆等。
对应的卡诺定理给出一个定量上限: \(\eta_\text{卡诺} = 1 - T_c / T_h\)（仅依赖两个热源温度，不依赖工质 / 工程细节）—— 严格推导留给卡诺循环（进阶）。本板块只需定性知道"\(\eta < 1\) 是物理原则上的"。
在纸上：① 查 3 类热机的典型效率；② 用 \(Q_h = W + Q_c\) 和 \(\eta = W/Q_h < 1\) 解释为什么不能 100%；③ 一句话总结"热机效率有物理上限，不只是工程局限"。
温差越大（Th - Tc 越大）热机效率上限越高 —— 这是为什么火力发电要"高压高温"蒸汽（提高 Th）+ "低温冷却"（降低 Tc）。
✓ Card 2 检查点
素养字段：物理建模 · 工程实例 · 物理原则 vs 工程局限
3

熵的定性概念 + 边界

▼
学习目标：用"无序度"定性介绍熵 \(S\) → 陈述熵增原理（孤立系统 \(\Delta S \geq 0\)）→ 明确本板块的边界（不引 \(dS = dQ/T\) / 不引玻尔兹曼公式）
建议时长：30-40 分钟
STEP 1"无序"的现象（5 分钟）

Card 1 思考点 2 已经发现一个统一线索 —— 自发过程都对应"从有序到无序"。本卡把"无序度"这个概念形式化为熵。

不查资料，列举几个"无序度变化明显"的过程 ——

  • 气体扩散：分子从"集中一角"（有序）→ "均匀分布"（无序）
  • 油墨入水：分子从"集中一滴"（有序）→ "均匀染色"（无序）
  • 冰融化：水分子从"规则点阵"（有序）→ "无规则液态"（无序）
  • 水蒸发：水分子从"液态束缚"（半有序）→ "气态自由"（更无序）
  • 鸡蛋摔碎：分子从"有结构的蛋"（有序）→ "无结构的碎片"（无序）

这些过程都是"无序度增加"。在物理上，这个"无序度"叫熵 \(S\)。

STEP 2熵 — 系统"无序度"的度量
熵的定性定义 ⭐: 熵 \(S\) 是系统"无序度"的度量。无序度越大 → 熵 \(S\) 越大。
· 注意：本板块只到"定性"——不引入 \(S\) 的计算公式（\(dS = dQ_\text{可逆}/T\)）
· 不引入玻尔兹曼公式 \(S = k_B \ln \Omega\)（用微观态数定义熵的统计形式）
· 这些数学形式留给状态函数（进阶）和熵的物理意义（进阶）板块。
图 3 · 熵增示意：气体扩散。左侧"初态"中红色小圆代表气体分子全部集中在左半盒（虚线为分隔），右半盒空（真空）—— 这是"有序"分布，熵 \(S\) 小。中间黑色箭头表示这是自发方向（自动发生 / 不需外力）。右侧"末态"中红色小圆均匀填充整个盒子 —— 这是"无序"分布，熵 \(S\) 大。反向（自发把均匀分布的气体挤回左半）不发生。
熵增原理（定性形式）⭐: 对一个孤立系统（不与外界交换能量 / 物质），熵不会自发减少 ——
· 自发过程: \(\Delta S > 0\)（熵增加，对应不可逆）
· 可逆过程: \(\Delta S = 0\)（熵不变，理想化的极限）
· 不可能: \(\Delta S < 0\)（孤立系统熵自发减少 → 违反第二定律）

注意"孤立"是关键 —— 局部系统熵可以减少（如冰箱里水结冰，水的熵减），但需要外界做功，整个"水 + 冰箱 + 电网"的孤立系统熵必定增加。
思考点 5 · 哪些日常过程是熵增？
列举 5 个日常过程，分析每个是熵增（自发）还是熵减（非自发或需外力）。
举例: ① 冰融化(熵增 · 规则点阵 → 液态无序). ② 水蒸发(熵增 · 液态 → 气态更无序). ③ 气体被压缩(熵减 · 分子更密集 → 但需要做功, 是"非自发" — 外界做功强迫). ④ 水自然结冰(熵减 · 液态 → 规则点阵 — 但要在 0°C 以下才发生, 因为环境吸收水的热, 环境熵增 > 水熵减, 总熵增). ⑤ 鸡蛋摔碎(熵增 · 有结构 → 无结构). 关键观察: 局部熵减不违反第二定律, 只要"局部 + 环境"总系统熵增就行。
在纸上：① 5 个例子的熵变方向；② 解释"为什么冰可以结冰"（局部熵减但总熵增）；③ 一句话总结"判定熵增看孤立系统 / 总系统，不看局部"。

思考点 6 · 熵和内能的关系
同一个系统能不能同时内能增加 + 熵增加？内能不变 + 熵增加？内能减少 + 熵减少？分类别想想，并体会"U 和 S 是独立变化的"。
① U ↑ + S ↑: 加热气体（U 升, 同时分子运动更剧烈 / 速率分布更宽 → S 升）。最常见情形。
② U 不变 + S ↑: 理想气体绝热自由膨胀（Joule 实验）—— 绝热 Q=0 + 自由膨胀对外不做功 W=0 → ΔU = Q−W = 0 → U 不变（对理想气体 T 也不变）；但气体从"集中一边" → "均匀分布全盒" → S 升。
③ U ↓ + S ↓: 水在 0°C 以下结冰（系统视角）—— 水放热给冰箱 / 寒冷环境 → U 降；水分子从无规则液态 → 规则点阵冰 → S 降。但这不违反第二定律: 局部水 ΔS<0, 但环境吸热 ΔS环境>0, "水+环境" 总系统 ΔS总>0 才能让结冰自发。
第四种组合 U ↑ + S ↓ 在自然界中很难找简单例子—— 加热使分子运动更剧烈, 通常导致 S↑而不是 S↓。这反映了物理世界的一种"对应趋势"（温度变化与无序度变化常同方向）。
关键: U 和 S 是独立变化的两个量。第一定律单独不能预测 S 的变化, 第二定律单独不能预测 U 的变化。两者互补而不替代。
在纸上：① 3 个能找到例子的组合各画一个具体例子；② 解释为什么"U↑+S↓" 难找例子（温度升通常伴随无序度升）；③ 一句话总结"U 和 S 是独立的，需要两个独立定律分别管它们"。
