📋 听点总表（27 锚 · ⭐ = 核心听点 · 6 项板块边界守门）
类别听点学生说到位时，话里应包含什么
G 定义与状态函数⭐⭐ G ≡ H − TS 严格定义纸上写出 G ≡ H − TS；指认 H、T、S；T 是系统温度；等温等压前提。
⭐⭐ G 引入的物理动机把 ΔS宇宙 ≥ 0 翻译成系统视角以方便计算；不是熵增判据不对。
⭐ G 是状态函数H、T、S 均状态量；代数组合仍状态量 ⇒ ΔG 路径无关。
盖斯定律 for GΔG 可拆步算；ΔG° = Σν·ΔGf°(产物) − Σν·ΔGf°(反应物)。
ΔG ≤ 0 严格推导⭐⭐ 4 步推导链① ΔS宇宙 = ΔS系 + ΔS环 ≥ 0 → ② 等压代入 ΔS环 = −ΔH系/T → ③ 整理 → ④ 乘 −T 反转 ⇒ ΔG ≤ 0。
⭐ 环境熵变 ΔS环 = −Qp/T等压 + 大热库 T 不变；负号来自系统↔环境视角切换。
⭐ 乘 −T 反转不等号T > 0 ⇒ 乘 −T 必反转（代数事实）+ 宇宙↔系统视角等价（物理含义）。
⭐⭐ ΔG ≤ 0 ⟺ ΔS宇宙 ≥ 0等温等压下完全等价；翻译而非取代。
最大非膨胀功⭐⭐ ΔG = −Wmax,非膨胀等温等压下，ΔG = 可提取的最大非膨胀有用功；可逆才取等。
⭐ 可逆 = 最大功不可逆过程的额外熵产生 T·dS额外 被以热放掉，吃掉了做功份额。
「自由」的含义ΔG = 可"自由用作有用功"的份额；TΔS 必须放热维持总熵不减。
dG = −δW非膨胀,可逆 推导第一 + 第二定律 + dH = dU + d(pV) + dG = dH − TdS − SdT 出发的 8 步链。
电化学桥⭐⭐ ΔG = −nFEn 转移电子数；F = 96485 C/mol；E 电池电动势（V）；ΔG 单位 J。
单位与符号双校验kJ → J 必换算；自发 ⇔ E > 0 ⇔ ΔG < 0。
理论 vs 实测电压E° 是可逆上限；实测低 ≠ 公式错；E° 提供"上限 + 基准 + 诊断"三重价值。
反应商 + 平衡桥⭐⭐ ΔG° = −RT ln Keq热力学 ↔ 化学平衡的核心桥；R = 8.314 J/(mol·K)；ΔG° 是标准态。
⭐ ΔG = ΔG° + RT ln Q反应商公式；ΔG° 固定参数（只随 T 变），ΔG 才随浓度变。
⭐ ΔG 符号 → 方向ΔG < 0 向产物 / > 0 向反应物 / = 0 平衡；平衡本质 Q 追上 K。
反应商 Q 定义产物在上 / 反应物在下 + 化学计量数次幂；气相分压 / 溶液浓度 / 纯固液不入式。
Keq 与 ΔG° 符号对应K > 1 ⟺ ΔG° < 0；K 只随 T 变。
温度对 K 的影响⭐ 范特霍夫方程（微分）d ln K / dT = ΔH° / (RT2)；ΔH° 视作不随 T 变。
放热 / 吸热 K 走向放热 ΔH° < 0 ⇒ T↑则 K↓；吸热反之；从导数符号推理。
合成氨工业折中热力学(低温利 K) vs 动力学(高温利速率) vs 经济(成本) —— 多维妥协。
ΔG° vs T 直线截距 ΔH° / 斜率 −ΔS° / x 截距 T转 = ΔH°/ΔS°。
思维方式一个量统一三学科G 统一热力学 / 化学平衡 / 电化学；物化"统一性"。
严格推导 vs 工程化进阶视角与基础视角互补 —— 不是"讲两遍"是"两个深度"。
可逆 = 理想极限可逆 = 最大有用功 = 零熵产生；物化核心范式。
边界守门
不展开		不引 Helmholtz 自由能 A等容等温对应自由能 A ≡ U − TS，留给大学物化（广义热力学势）。
不引化学势 μμ ≡ (∂G/∂n)T,P + Gibbs-Duhem，留给大学物化。
不严格标准态 IUPAC 定义留给大学物化（标准态选择依赖性）。
不引 Nernst 方程E = E° − (RT/nF)·ln Q + 电极电势体系，留给待建电化学板块。
不引活度系数 γ非理想体系修正，留给大学物化。
不引范特霍夫积分形式ln(K2/K1) = −(ΔH°/R)(1/T2 − 1/T1) 留给大学物化；本板块只用微分形式定性。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 G ≡ H − TS 并逐项指认」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
