📋 听点总表（全部 22 锚 · ⭐ = 核心听点 · 6 项板块边界守门）
类别听点学生说到位时，话里应包含什么
燃烧热⭐⭐ 燃烧热定义 4 限定25°C / 1 atm + 1 mol 可燃物 + 完全燃烧 + 稳定氧化物，ΔH燃 < 0，单位 kJ/mol。
⭐ 完全燃烧 + 稳定氧化物C → CO₂(不是 CO) / H → H₂O(l) / S → SO₂(不是 SO₃)。
⭐ 1 mol 可燃物归一化归一对象是「1 mol 可燃物」，不是 1 mol O₂；因 O₂ 是共有反应物。
H₂O 液态约定 (HHV)燃烧热约定 H₂O 液态（高位热值）；液气差 ≈ +44 kJ/mol。
典型燃烧热数据CH₄ -890 / H₂ -286 / C(石墨) -393 / C₂H₅OH(l) -1367 量级感。
中和热⭐⭐ 中和热定义 + 离子方程式25°C / 1 atm + 强酸强碱稀溶液 + 生成 1 mol H₂O(l)；本质 H⁺ + OH⁻ → H₂O(l)，ΔH ≈ -57.3 kJ/mol。
⭐ 强酸强碱中和热相同的离子本质所有强酸强碱中和到位都是同一离子反应，所以数值基本相同。
⭐ 1 mol H₂O 归一化归一对象是「生成 1 mol 液态水」，不是 1 mol 酸 / 碱；多元酸碱要换算 H⁺ 摩尔数。
⭐ 弱酸弱碱中和热偏低弱电解质需先电离（吸热）→ 总放热绝对值 < 57.3 kJ/mol。
弱酸两步走分析① 弱酸电离微吸热 ② H⁺ + OH⁻ → H₂O 放热 -57.3 → 总和 ≈ -55 ~ -57 kJ/mol。
量热计实验⭐⭐ 简易量热计核心算式 Q = m·c·ΔT能在纸上写出并指认 m=溶液总质量、c≈4.18 J/(g·K)、ΔT=测得温升。
⭐ 弹式量热计构造钢弹耐压容器（放可燃物 + 过量 O₂）+ 水浴 + 电火花引燃 + 搅拌器 + 保温外壳。
弹式算式 Q = m·c·ΔT + C弹·ΔT水吸热 + 钢弹吸热两项缺一不可。
简易量热计构造保温杯 + 温度计 + 搅拌器；适用于溶液反应（中和、溶解）。
量热计实验误差源热损失 / 仪器吸热 / 反应不完全 / 温度延迟 / 溶液比热近似为水。
综合应用⭐ 燃烧热 / 中和热 ↔ 盖斯定律实验测 ΔH 的特殊形式 + 盖斯定律算 ΔH 的输入数据；不是对立而是互补。
实验测 ΔH 的 2 板斧弹式（燃烧反应）+ 简易（中和反应）= 实验直接测 ΔH 的标准两种入口。
用燃烧热数据反算其它 ΔH3 个燃烧热数据 + 盖斯定律可反算任意反应 ΔH（实验入口 → 算法 → 应用）。
思维方式定义中的限定词不是冗余「1 mol」「标准条件」「完全燃烧」「生成 1 mol 水」是规约消歧义。
实验 vs 算法两个入口本板块（实验测）+ 盖斯定律（算法算）是 ΔH 体系的两个独立入口，互补不替代。
边界守门
不展开		不引 QV 与 Qp 修正ΔH − ΔU = RTΔng 留给进阶焓与焓变。
不引量热计工程优化水当量校准 / 卡氏热值仪留给大学物化实验。
不定量电离平衡电离常数 Ka 留给水溶液化学板块。
不引比热容随温度变化留给大学物化。
不引非绝热定量修正热损失定量分析留给大学物化实验。
不区分 HHV vs LHV 工程规范仅给约定方向（H₂O 液态 = HHV），工程使用规范留给工程热力学。
📊 判定 rubric 三档详细

✅ 通过 → 当场签字

  • 抽中 3 条主问题里 ≥ 2 条「⭐ 核心听点全中 + 教师勾选状态为「✅ 通过」」。
  • 整场至少一次完成「在纸上写出 Q = m·c·ΔT 并逐项指认 m / c / ΔT」（顶部 📝 勾选）。
  • 无未化解的致命误区：背诵型信号 ≤ 1 条主问题触发。

⏸ 暂缓 → 补学 + 预约二试

  • 恰 1 条主问题核心听点缺失，或某条主问题两级台阶后仍卡壳。
  • 按系统列出的补学清单交给学生，预约二试。
  • 二试必须换未用过的主问题（抽题区已自动标记）。

❌ 不通过 → 重学

  • ≥ 2 条主问题核心听点严重缺失。
  • 背诵型信号在 ≥ 2 条主问题贯穿（系统会自动提示）。
  • 不签字，约定重学路径完成后再约口试。
