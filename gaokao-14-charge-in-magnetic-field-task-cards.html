STEP 0 · 14 题位 · 带电粒子磁场骨架介绍

在做任何一道题之前，先用 5 分钟把这一题位的"骨架长什么样"刻进脑子。下面 4 段是骨架本身，不是技巧——技巧在 3 道例题里。

骨架 1 · 带电粒子在磁场 = 洛伦兹力 + 圆周运动 + 弦切几何
题面里出现 带电粒子（质量 m、电量 q、初速 v）进入 匀强磁场区域（B 给定方向），并且没有电场或重力影响（或题面明示忽略）——这就是 14 题位的标准骨架。
洛伦兹力公式 \(F = qvB\)（v ⊥ B 时），方向由左手定则确定（正电荷直接读 F 方向；负电荷反向）。这个力始终垂直于速度 → 大小恒定 → 粒子做匀速圆周运动。半径由牛顿第二定律 \(qvB = mv^2/r\) 解出 \(r = mv/(qB)\)。
圆周运动一旦确立，剩下的问题几乎全是几何题：圆心在哪、半径与边界如何相交、入射点与出射点之间的弦长是多少、对应圆心角多大。卷面 60% 的分都在几何作图上。
骨架 2 · 高考 14 题位高频骨架（带电粒子磁场类）
据近 5 年全国卷与新高考统计：14 题位的 12-15 分压轴题中，"带电粒子在磁场中运动"出现率名列前茅（参见 2024 重庆 14、2023 山东 14、2022 全国乙 25 等真题骨架）。出题方式以 "2-3 小问" 为主：第 (1) 问考半径或周期（套公式即可），第 (2) 问考几何边界（圆与边界交点 / 弦长）+ (3) 问考多粒子源 / 收集装置范围（一组粒子的极端入射方向）。
本任务卡选取的 3 道例题分别对应该骨架的三类典型变体：矩形区域单向入射（例题 1，热身）/ 半无界区域 + 多方向粒子源 + 收集装置（例题 2，基于 2024 重庆 14 改编的"电荷收集器"骨架）/ 双磁场区拼接（例题 3，自创变式）。三例覆盖率约占 14 题位历年真题骨架的 75% 以上。
骨架 3 · 识别关键字（题面里只要见到就立即对号入座）
「带电粒子 / 质子 / α 粒子」+「匀强磁场」+「垂直纸面向外（·）/ 向里（×）」 → 立即写出半径 \(r = mv/(qB)\) 与周期 \(T = 2\pi m/(qB)\)。
「从某点沿某方向射入」：标出入射点 + 入射速度箭头 → 用左手定则定洛伦兹力方向 → 洛伦兹力指向 = 圆心方向 → 沿入射点向"洛伦兹力方向"画长度 = r 的线段，端点即圆心。
「连续向各方向均匀发射」/「各方向粒子」：电荷收集器骨架 → 不同入射方向 = 不同圆心位置（所有圆心都在以发射点为圆心、半径 = r 的圆上）→ 收集装置就是"所有可能出射点"的几何包络。
「打到 x 轴」/「从某边界射出」：典型几何约束 → 圆与直线的交点求解 → 用勾股定理或半径在弦中的垂直平分关系。
骨架 4 · 核心判据 3 条（带电粒子磁场的"通关密钥"）
判据 A · 洛伦兹力方向（左手定则）：四指指 v 方向，磁感线穿过手心（B 向外则手心朝外 / B 向里则手心朝里），大拇指方向 = 正电荷的 F 方向（即圆心指向）。负电荷必须取反。这个力始终指向圆心——这就是"圆心定位"的物理依据。
判据 B · 半径公式与圆心位置：\(r = mv/(qB)\) 给出半径大小；圆心位置由"入射点 + 沿洛伦兹力方向走 r"确定。等价描述：入射速度方向 ⊥ 入射点处的"圆心方向"——这是"入射点处速度沿切线、半径垂直于切线"的几何刻画。
判据 C · 弦切几何（弦长与圆心角）：粒子从入射点到出射点的轨迹是一段圆弧，对应的弦（连接入射点与出射点的直线段）长度 \(L_{\text{弦}} = 2r \sin(\alpha/2)\)，其中 α 是圆心角。等价的弦的垂直平分线必过圆心——这是"两个已知点求圆心"的几何主路。14 题位 50% 以上的几何题都靠这条判据破题。

读完以上 4 段，先合上页面，在草稿上把"识别关键字 → 定 F 方向 → 找圆心 → 算半径 → 求弦切几何"这条链路默写一遍，并独立画一张"正电荷以速度 v 向右进入向外磁场"的圆周示意图。能默 + 能画，再进入例题 1；不行，再回上面读一次。

STEP 1核心知识速复习
5-8 分钟

下面 4 条公式 + 2 条判据是例题 1 用到的全部工具。先合上页面回忆，再对照。

洛伦兹力与圆周运动速查清单
洛伦兹力大小：\(F = qvB\)（v ⊥ B 时），方向由左手定则定（正电直接读、负电反向）。
半径公式：由 \(qvB = mv^2/r\) 得 \(r = mv/(qB)\)。速度越大、半径越大；磁场越强、半径越小。
周期公式：\(T = 2\pi r/v = 2\pi m/(qB)\)。T 与速率 v 无关——这是磁约束的关键特征。
弧长与时间：粒子在圆弧上扫过的圆心角 α 对应时间 \(t = \alpha/(2\pi) \cdot T = \alpha m/(qB)\)。半圆 α = π → t = T/2。
判据 · 圆心定位=入射点 + 沿洛伦兹力方向走 r，端点即圆心。
判据 · 出射判定=圆周与边界的交点即出射点；若圆周不与边界相交，则粒子在区域内做闭合圆周（始终不出）。

STEP 4方法路径
5 步串起整题
路径 1 · 半径与周期（第 (1) 问，公式直出）
第 (1) 问几乎是"白送分"——但前提是你能把公式从原理推一遍，而不是死记。原理：洛伦兹力提供向心力 → \(qvB = mv^2/r\) → 解出 r。周期由 \(T = 2\pi r/v\) 替换 r 得到。
"请反问我：为什么洛伦兹力可以提供向心力？它做不做功？不要直接告诉我结论。"
由 \(qvB = mv^2/r\) → \(\boxed{r = \dfrac{mv}{qB}}\)
由 \(T = \dfrac{2\pi r}{v} = \dfrac{2\pi m}{qB}\) → \(\boxed{T = \dfrac{2\pi m}{qB}}\)
物理意义：周期 T 仅由 m、q、B 决定，与 v 无关（这是磁约束的关键特征）。
路径 2 · 定洛伦兹力方向（左手定则）
入射时刻：v 沿 +y，B 沿 +z（向外）。正电荷的 F = qv × B。左手定则：四指顺 v（指向 +y），磁感线从手心穿出（即手心朝 +z），大拇指方向即 F 方向 → 大拇指指向 +x。
"请帮我验证：若把粒子换成负电粒子（电荷 −q），洛伦兹力方向会不会反？为什么？不要替我做这一步。"
入射点 O(0, 0)，洛伦兹力沿 +x。关键：F 始终指向圆心 → 圆心一定在入射点的 +x 方向上、距 O 为 r 处。
当 r = L/2 时，圆心 O' 坐标 = (L/2, 0)。
路径 3 · 写出圆方程 + 找与边界交点（第 (2) 问几何）
圆心 (L/2, 0)，半径 L/2 → 圆方程 \((x - L/2)^2 + y^2 = (L/2)^2\)。粒子在 y ≥ 0 半平面内运动（因为矩形 y ∈ [0, 2L]），所以轨迹是该圆位于 y ≥ 0 上的上半圆。圆与矩形哪条边界相交？把每条边代入：
代入 y = 0：\((x - L/2)^2 = (L/2)^2\) → x = 0 或 x = L。两根分别是入射点 (0, 0) 与新点 (L, 0)。
代入 y = 2L：\((x - L/2)^2 = (L/2)^2 - 4L^2 < 0\)，无解 → 圆不触上边界。
代入 x = ±L：\(y^2 = (L/2)^2 - (L/2 \pm L)^2\)，亦不实根 → 圆不触左右边界（内部圆）。
故粒子沿上半圆走完整一圈半圆，从 (0, 0) 经最高点 (L/2, L/2) 到达 (L, 0) 离开。
路径 4 · 圆心角 → 时间
轨迹是从 (0,0) 经 (L/2, L/2) 到 (L, 0) 的上半圆 → 圆心角 = π → 用时 = 周期的一半。这里只需写 \(t = T/2\)，不用代具体数。
\(t = \dfrac{T}{2} = \dfrac{\pi m}{qB}\)
物理意义：粒子在磁场内走完半圈就离开了磁场区域，时间为周期的一半。
路径 5 · 出射点坐标 + 速度方向校验
出射点 (L, 0) 已由路径 3 几何求出。再校核速度方向：粒子在出射点处的速度沿圆的切线，方向应满足"沿圆周运动方向继续延伸"。已知粒子在 (L/2, L/2) 处速度沿 +x（圆心 (L/2, 0) 在它的 -y 方向 → F 指向 -y → 此时 v 必沿 +x，与圆切线一致 ✓）。继续到 (L, 0) 时，圆心在它的 -x 方向 → F 沿 -x → 速度沿 -y（即指向矩形外部 y < 0 区域）。
出射点：(L, 0)
出射速度方向：沿 -y（向下射出磁场区域）
校验：从 (0, 0) 经半圆到 (L, 0)，入射速度 +y、出射速度 -y，速度方向旋转 180° = π，与"半圆"一致 ✓。
提示：本题 r = L/2 是"刚好走半圆"的甜蜜参数。若 r > L/2，圆会越过右边界 x = L 而提前从右边出射；若 r < L/2，圆会被矩形完全容纳，但题面"离开磁场区域"必然要求圆周与边界相交（这种情况若题面未额外限定边界，需讨论 r 的取值范围）。本题取 r = L/2 是为了让你把"恰好半圆"的标杆情况吃透。

本题与例题 1 的区别：不止一个粒子方向，是一族粒子从同一点向 360° 各方向均匀发射。处理这种题的关键是"圆心轨迹"——所有圆心组成一条曲线（本题里是一个圆），落点构成这条曲线的几何包络。

多方向粒子源速查清单
圆心轨迹：每个粒子的圆心 C 距粒子源 P 都恰好为 r（因为洛伦兹力沿圆心方向，C 在入射点的 F 方向上、距入射点 r）。故所有圆心组成以 P 为圆心、半径为 r 的圆。
两点共圆几何判据：给定两点 P、Q，是否存在过 P 与 Q 且半径为 r 的圆？充要条件：\(|PQ| \le 2r\)（即两点距离不超过直径）。\(|PQ| < 2r\) 时有两个这样的圆；\(|PQ| = 2r\) 时两点为直径两端、圆唯一；\(|PQ| > 2r\) 时无解。
落点存在性：x 轴上某点 Q 是否可能成为落点 = 是否存在某个粒子的圆周同时过 P 与 Q = \(|PQ| \le 2r\)。
判据 · 半无界磁场出射条件=粒子的圆周与边界 x 轴第一次相交即出射；本题中粒子"第一次到 x 轴即被吸收"。
对称性=粒子初速方向均匀分布在 360° → 整个系统关于 y 轴对称 → 落点集合关于 y 轴对称。
几何前置 · 两点共圆：给定平面上两点 P、Q，所有"过 P 与 Q 的圆"的圆心都在线段 PQ 的垂直平分线上。若进一步要求该圆半径 = r，则圆心还必须满足"到 P（或 Q）的距离 = r"——两条约束相交即圆心位置。由勾股，若 d = |PQ|/2，圆心到 PQ 中点的距离 = \(\sqrt{r^2 - d^2}\)，故 d ≤ r、即 |PQ| ≤ 2r 时有解。
STEP 2题面 · 电荷收集器
3 小问
xy 平面 \(y > 0\) 半平面内存在匀强磁场 \(B\) 垂直纸面向里。坐标点 \(P(0, h)\) 处有一粒子源，连续向 360° 各方向均匀发射相同电量 \(q\)、质量 \(m\)、速率 \(v\) 的带正电粒子（\(h > 0\)）。设半径 \(r = mv/(qB)\)，且已知 \(r > h\)。x 轴是磁场区域的下边界（\(y < 0\) 无磁场），任何粒子第一次到达 x 轴时即被该处的收集装置吸收。

(1) 求 P 正下方释放（初速沿 \(-y\)）的粒子在 x 轴上的落点坐标 \(x_0\)。[约 5 分]
(2) 求所有粒子能落到 x 轴上的 x 坐标取值范围 \([x_{\min}, x_{\max}]\)。[约 7 分]
(3) 若要在 x 轴 \(x \ge 0\) 一侧设置一段连续的收集装置，使其能收集所有能落到 x 轴 \(x \ge 0\) 一侧的粒子，求该装置应覆盖的最短范围 \([x_1, x_2]\) 与所需最短长度 \(d_{\min}\)。[约 3 分]
SVG 2 · 粒子源 P(0, h) 向各方向均匀发射正电粒子；磁场 B 向里。紫色虚圆 = 所有粒子圆心组成的轨迹（以 P 为中心、半径 r）；蓝色虚圆 = "P 正下方释放"那一颗的轨迹（圆心 C₁ = (r, h)）；x 轴上橙点 = 该颗落点 x₀；绿色端点 = 全体落点的左右边界。

STEP 4方法路径
6 步串起整题
路径 1 · 第 (1) 问 · 定 F 方向 + 圆心
P 正下方释放：v 沿 -y。B 沿 -z（向里）。正电荷的 F = qv × B。代入坐标：v = (0, -v, 0)，B = (0, 0, -B)，则 v × B = ((-v)·(-B), 0, 0) = (vB, 0, 0)，F = q(vB, 0, 0) → F 沿 +x。
"请反问我：若 P 释放方向改为 +x（向右），F 方向会不会还是 +x？不要直接告诉我答案。"
F 沿 +x → 圆心在 P 的 +x 方向、距 P 为 r 处 → 圆心 \(C_1 = (r, h)\)。
圆方程：\((x - r)^2 + (y - h)^2 = r^2\)。
路径 2 · 第 (1) 问 · 圆与 x 轴交点
在圆方程中代 y = 0：\((x - r)^2 + h^2 = r^2\) → \((x - r)^2 = r^2 - h^2\)。由于题设 r > h，\(r^2 - h^2 > 0\)，根式有意义。
解出 \(x = r \pm \sqrt{r^2 - h^2}\)。
粒子从 P(0, h) 顺时针沿圆走（因 F 沿 +x → 粒子向 +x 偏转 → 绕圆心顺时针）→ 第一次碰到 x 轴的是较小的 x 根：
\(\boxed{x_0 = r - \sqrt{r^2 - h^2}}\)
提示：若不放心"顺时针 vs 逆时针"，可代一个具体值（如 r = h，则 x_0 = h ± 0 = h，刚好相切；r 略大于 h，落点离 P 在 x 轴上的投影 0 较近 → 取小根更合理）。
路径 3 · 第 (2) 问 · 几何包络的核心洞察
不要陷入"对每个 θ 列方程再求并"——那是高数级别的工作量。换个角度：x 轴上某点 \(Q(x_Q, 0)\) 能成为落点 ⇔ 存在一颗粒子的圆周同时过 P(0, h) 和 Q(x_Q, 0)，且半径为 r。这等价于"两点共圆半径 r 存在性"问题。
"请反问我：'过 P 和 Q 的半径为 r 的圆是否存在'对 |PQ| 有什么充要条件？不要直接说。"
几何判据：\(|PQ| \le 2r\)（即两点距离 ≤ 直径）。
计算：\(|PQ| = \sqrt{x_Q^2 + h^2}\)。条件 \(\sqrt{x_Q^2 + h^2} \le 2r\) → \(x_Q^2 \le 4r^2 - h^2\) → \(|x_Q| \le \sqrt{4r^2 - h^2}\)。
路径 4 · 第 (2) 问 · 完整答案 + 严谨性补全
"几何上存在过 P、Q 的半径 r 圆"只保证存在这样的圆周，还需要确认对应的粒子初速方向真在题面允许的 360° 内（这里全方向都允许，故无额外限制），且粒子沿圆周走到 Q 而不是绕到对侧。由于初速方向连续覆盖 360°、对称性、且粒子顺时针走，所有"几何上可达"的 Q 都对应至少一种粒子能真到达。
\(\boxed{x \in \left[-\sqrt{4r^2 - h^2},\ \sqrt{4r^2 - h^2}\right]}\)
即 \(x_{\min} = -\sqrt{4r^2 - h^2},\ x_{\max} = \sqrt{4r^2 - h^2}\)。
落点段总长 = \(2\sqrt{4r^2 - h^2}\)。
路径 5 · 第 (3) 问 · 用对称性 + (2) 的结果
系统关于 y 轴对称（初速方向均匀分布、磁场对称、几何对称）→ 落点也关于 y 轴对称 → x ≥ 0 一侧的落点 = [0, √(4r²-h²)]（原点是对称的极限点，由连续性也在落点集中）。
\(x \ge 0\) 一侧落点范围：\([0,\ \sqrt{4r^2 - h^2}]\)
故收集装置应覆盖 \(\boxed{[x_1, x_2] = [0,\ \sqrt{4r^2 - h^2}],\ d_{\min} = \sqrt{4r^2 - h^2}}\)。
路径 6 · 复盘 · 把"两点共圆"这条几何判据钉在桌面上
本题的核心洞察"过 P、Q 的半径 r 圆存在性 ⇔ |PQ| ≤ 2r"是 14 题位电荷收集类题型的万能钥匙。一旦把这条几何性质内化，所有"多方向粒子源"题都从"对每个方向求落点"退化为"对每个目标点反问能否被达到"——后者只需算距离。
小结：(1) x₀ = r − √(r²−h²)；(2) 落点 ∈ [−√(4r²−h²), √(4r²−h²)]；(3) 正侧收集装置 [0, √(4r²−h²)]，长度 √(4r²−h²)。
三问的核心都靠"圆心轨迹 + 两点共圆 + 对称性"三件套。
提示：若题目把粒子改成负电（或 B 反向），落点的对称性不变（仍关于 y 轴对称）；改变的是单个粒子的"顺/逆时针"转向，从而 (1) 中 x₀ 可能取大根 r + √(r²−h²)。卷面上务必看清电荷符号 + B 方向。

例题 3 · 双磁场区拼接 · 回到原点（自创变式 · 多区分段升级）

▼
学习目标：以"两区拼接（同方向 + 强度比 2:1）+ 多次穿越 y 轴"为骨架，掌握"边界处速度方向连续 + 新区重新算半径与圆心 + 时间分段累加"的高阶处理；理解"左区圆心 = O"为何是回到原点的几何关键
建议时长：70 分钟（速复习 8 + 题面 5 + 自己做 20 + 对照思考 22 + 评分锚点 + 反思 15）
STEP 1核心知识速复习
5-8 分钟

本题与例题 1、2 的区别：磁场区域不止一个。粒子穿越分界（y 轴）时，速度方向连续（因为洛伦兹力是有限力，不会让速度方向跳变），但半径和圆心都重算（因为 B 变了）。每段都是新的圆周运动，时间分段累加。

多区拼接速查清单
边界连续性：粒子穿越 y 轴时，速度方向、速率均连续（即过渡瞬间 v 大小方向都不变）。变化的只是"接下来的圆心位置和半径"。
新区半径与圆心：每进入新区，重算 \(r = mv/(qB_{\text{新}})\)，并用左手定则定 F 方向 → 新圆心。
周期与速率无关：\(T = 2\pi m/(qB)\) 只依赖 B，不依赖 v 或 r。两区 B 不同 → 两区 T 不同；同一区内 T 始终不变。
圆心 vs y 轴的关系：若 y 轴恰好是圆的弦或切线，粒子在 y 轴上的两个交点之间就是该区内的轨迹。是否走半圆 / 全圆 / 一小段，取决于"圆心相对 y 轴的位置 + 入射点 + 出射条件"。
判据 · 何时再回到 y 轴=粒子的圆与 y 轴的第二个交点（不是入射点）即为出射点；若该圆与 y 轴只切（不相交于第二点），粒子将永远绕该圆不出。
STEP 2题面
双磁场区 + 回到原点
xy 平面被 y 轴分为左右两区：左区（\(x < 0\)）匀强磁场 \(B_1\) 方向垂直纸面向外（·），右区（\(x > 0\)）匀强磁场 \(B_2 = 2B_1\) 方向同样垂直纸面向外（·）——两区磁场方向相同但强度不同（右区是左区的 2 倍）。带正电粒子（电量 q、质量 m）从原点 O 沿 \(+x\) 方向射入右区，初速率 v。

(1) 求粒子在右区运动的半径 \(r_2\)。[约 3 分]
(2) 粒子离开右区进入左区后，求其在左区运动的半径 \(r_1\)。[约 3 分]
(3) 求粒子从射入到第一次回到原点 O 所用的总时间 \(t_{\text{总}}\)。[约 6 分]
SVG 3 · 左右两磁场区方向相同（都向外）但强度比 1:2（右区强）。正电粒子从 O 沿 +x 进入右区 → 顺时针半圆到 (0, −2r₂) → 进入左区 → 逆时针半圆绕原点到 (0, +2r₂) → 再进入右区 → 顺时针半圆回到 O。共三段半圆。

STEP 4方法路径
5 步串起整题
路径 1 · 第 (1)(2) 问 · 两区半径（公式直出）
两问都是套公式 \(r = mv/(qB)\)。注意 B₁ 是左区强度，B₂ = 2B₁ 是右区强度——别搞反下标。
\(\boxed{r_2 = \dfrac{mv}{qB_2} = \dfrac{mv}{2qB_1}}\)（右区半径）
\(\boxed{r_1 = \dfrac{mv}{qB_1} = 2 r_2}\)（左区半径，是右区的 2 倍）
路径 2 · 段 I（右区第一次） · 定 F 方向 + 圆心 + 半圆
入射点 O，v 沿 +x，B₂ 沿 +z（向外）。直接用 F = qv × B：
\(\mathbf{v} \times \mathbf{B} = (v, 0, 0) \times (0, 0, B_2)\)。叉乘公式 j 分量 = \(v_z b_x - v_x b_z\) = 0 − v·B₂ = −vB₂。故 F = q(0, −vB₂, 0) → F 沿 −y。
（用左手定则：四指顺 +x、磁感线从手心穿出（向外）、大拇指指向 −y ✓ 一致）。
"请反问我：粒子从 O 顺时针绕 \(C_{R1}\) 走，半圆后到哪里？速度方向变成什么？不要直接告诉我。"
圆心 \(C_{R1} = (0, -r_2)\)（在 O 的 −y 方向，距 \(r_2\)）。
粒子从 O 出发，初速 +x、F 朝 −y → 粒子向 +x 下方偏 → 顺时针绕 \(C_{R1}\)。
圆方程 \(x^2 + (y + r_2)^2 = r_2^2\)。y 轴另一交点：\((y + r_2)^2 = r_2^2\) → y = 0 或 y = −2r₂。
故粒子走半圆 O → \((0, -2r_2)\)，速度由 +x 变为 −x。
段 I 时间：\(t_1 = T_2/2 = \pi m/(qB_2) = \pi m/(2qB_1)\)。
路径 3 · 段 II（左区） · 圆心刚好是原点 O
进入左区瞬间：位置 \((0, -2r_2)\)，速度 −x（连续过 y 轴）。B₁ 沿 +z（向外，与右区方向一致）。
\(\mathbf{v} \times \mathbf{B} = (-v, 0, 0) \times (0, 0, B_1)\)：j 分量 = \(0 \cdot 0 - (-v) \cdot B_1 = vB_1\) > 0。F = q(0, vB_1, 0) → F 沿 +y。
圆心 = (0, −2r₂) 沿 F 方向 +y 移 r₁ = (0, −2r₂ + r₁) = (0, −2r₂ + 2r₂) = (0, 0) = O！
太巧——左区的圆心刚好是原点 O，半径 r₁ = 2r₂。
圆方程 \(x^2 + y^2 = (2r_2)^2\)。粒子在 (0, −2r₂) 出发，初速 −x、F 朝 +y → 向 −x 上方偏 → 逆时针绕 O。
圆与 y 轴另一交点：(0, +2r₂)。粒子走半圆从 (0, −2r₂) 经 (−2r₂, 0) 到 (0, +2r₂)，速度由 −x 变为 +x。
段 II 时间：\(t_2 = T_1/2 = \pi m/(qB_1)\)。
"左区圆心 = O"是本题最优雅的几何巧合——根源是：r₁ = 2r₂ 恰好等于段 I 结束时粒子距 y 轴的 |y_in| = 2r₂。设计参数 B₂ = 2B₁ 正是为了让这件事发生。
路径 4 · 段 III（右区第二次） · 顺时针半圆回 O
进入右区瞬间：位置 \((0, +2r_2)\)，速度 +x。B₂ 沿 +z。同段 I 的推导：F 沿 −y → 圆心 \(C_{R2} = (0, 2r_2) + (0, -r_2) = (0, r_2)\)。半径 r₂。
圆方程 \(x^2 + (y - r_2)^2 = r_2^2\)。y 轴另一交点：y = 0 或 y = 2r₂。
粒子顺时针绕 \(C_{R2}\)，从 (0, 2r₂) 经 (r₂, r₂) 走半圆到 \((0, 0) = O\)！速度由 +x 变为 −x。
段 III 时间：\(t_3 = T_2/2 = \pi m/(2qB_1)\)。

总时间：
\(\boxed{t_{\text{总}} = t_1 + t_2 + t_3 = \dfrac{\pi m}{2qB_1} + \dfrac{\pi m}{qB_1} + \dfrac{\pi m}{2qB_1} = \dfrac{2\pi m}{qB_1}}\)
物理意义：粒子在右区走了一个完整的 T₂ 周期（两段半圆累计），在左区走了 T₁/2，合计 T₂ + T₁/2 = T₁/2 + T₁/2 = T₁ = 2πm/(qB₁)。
提示：注意粒子最终在 O 处速度变为 −x（不是回到初速 +x）。"第一次回到原点"是题面问的，没问"回到原点且速度复原"——若问后者，需走两个完整循环 = 4πm/(qB₁)。卷面务必分清"回到原点"和"恢复初态"。
路径 5 · 物理图像复盘 · 三段半圆 + 关于 x 轴对称
复盘三段轨迹：① 右区下方半圆（圆心 (0, −r₂)，r₂）；② 左区半圆绕原点（半径 2r₂）；③ 右区上方半圆（圆心 (0, r₂)，r₂）。整条轨迹关于 x 轴对称——从 O 出发往下、绕到左侧、再绕回 O。所有圆心都在 y 轴上。
物理意义：① 粒子在 B 大的右区半径小（r₂ = r₁/2）、绕得"急"；② 在 B 小的左区半径大（r₁ = 2r₂）、绕得"舒"。
② 三段时间不等：右区两段每段 T₂/2 = πm/(2qB₁)；左区一段 T₁/2 = πm/(qB₁) = 右区两段之和。三段累加恰等于一个 T₁。
③ "回到原点"的几何关键 = 左区圆心 = O，这要求 r₁ = 2r₂（即 B₂ = 2B₁）——若 B₂ ≠ 2B₁，左区圆心就不在 O，粒子不能简单地"半圆回原点"。

综合反思 · 三例打通这条骨架

3 道例题难度递增，但都在重复同一条主链路：识别情境 → 定洛伦兹力方向 → 找圆心 → 算半径 → 用弦切几何或边界相交求出射 → 算时间。下面把"共性"和"易扣分"两件事钉死。

一、3 例题骨架共性

  • 圆心永远在"入射点的洛伦兹力方向上、距离 r"：这是 14 题位所有几何工作的起点。无论是例 1 单方向、例 2 多方向、例 3 多区，每段轨迹的圆心位置都靠这条规则定。
  • 半径与周期都来自牛顿第二定律：\(r = mv/(qB),\ T = 2\pi m/(qB)\)。r 依赖 v 与 B，T 只依赖 B。同一区内 T 不变；跨区时 T 变。
  • 几何工具优于代数硬算：例 1 用"圆与边界相交"求出射点；例 2 用"两点共圆半径 r 存在性 ⇔ |PQ| ≤ 2r"破多方向题；例 3 用"圆心 = 原点"识别闭合条件。所有 14 题位都先画图后算式。
  • 速度方向 = 圆周切线方向 ⊥ 圆心方向：在任何一点上，速度沿圆周切线，半径（圆心连线）垂直于切线。卷面上常用此性质回推圆心或确认转向。
  • 电荷符号 + B 方向 = 转向决定：正电与负电在同一 B 中转向相反；同电荷在 B 反向中转向也相反。卷面每段开头都要明确写"电荷为正"+"B 沿 X 方向"+"用左手定则得 F 沿 Y"。

二、5 类学生常见卡点 + 怎么破

  1. F 方向定反（左右手搞反、电荷符号漏看、B 方向看反）→ 破：每次定 F 都写"四指顺 v、手心顺 B、大拇指出 F"逐字勾对一遍
  2. 圆心位置画错（沿 F 方向距 r，但画到 v 方向上）→ 破：先标"F⊥v"双向箭头，再沿 F 方向标距 r 的圆心点
  3. 圆与边界交点求错（漏代某条边、把切点算成交点）→ 破：把所有边界方程都代入圆方程一遍，逐个判别 Δ ≥ 0
  4. 多方向题没用"圆心轨迹"概念（对每个 θ 暴力代）→ 破：见"360° 均匀发射"立即想"圆心组成以 P 为中心、半径 r 的圆"+"两点共圆判据"
  5. 多区拼接漏算某段 / 没认出闭合条件（如例 3 漏第三段、或没认出"左区圆心 = O"）→ 破：每段开头都写"位置 + 速度 + B 方向 + F + 圆心"五件套，最后画出整条轨迹再校验是否闭合

三、14 题位 12-15 分钟应试建议

📌 14 题位定位：高考最后压轴题，分值 12-15 分，给 12-15 min 是合理的。比 13 题位多 2-3 分钟，因为几何作图本身耗时。

📌 3 分钟读题 + 画原图：读完立刻在草稿上画出磁场区域 + 入射点 + 入射方向 + B 方向（×/·）。读完不画图最不效率。

📌 2 分钟定 F + 圆心：左手定则确认 F 方向、沿 F 方向标 r、画圆心点。这一步定错全题崩。

📌 3-5 分钟几何分析：圆方程 + 边界相交 / 弦切几何 / 圆心轨迹族（按题型选）。能画图绝不代数。

📌 3-5 分钟最终落式：写半径 / 周期 / 出射点坐标 / 时间公式。每个量记得带单位与方向。
