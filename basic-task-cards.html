热力学第一定律

能量守恒在热力学系统的形式 · 用 \(\Delta U = Q - W\) 把"热"和"功"算在一起
模块名称热力学第一定律（基础模块）
所属大主题热力学 · 物理化学融合主题
预计学习时间2.5-3 小时（按自己的节奏）
前置板块分子动理论（必读 —— 内能的微观定义 \(U =\) 总动能 + 总势能 来自前置板块）
气体液体固体（推荐 —— 理想气体方程 \(PV = nRT\) 在 4 过程分析中常用）
所需工具AI 工具、纸笔、计算器；对前置板块"内能微观定义""分子总动能 ∝ N·T"的熟练度

学习目标

  1. 能用一句话陈述热力学第一定律 \(\Delta U = Q - W\)，并说清符号约定（系统视角：\(Q\) 吸热为正、\(W\) 系统对外做功为正）
  2. 能解释为什么第一定律就是"能量守恒"在热力学系统的具体形式 —— 不是新原理，而是能量守恒的一种说法
  3. 能区分热传导 Q 与做功 W 两种"外界给系统能量"的方式，并从分子层面解释二者的微观图像差异（无序传能 vs 有序定向）
  4. 会用 \(\Delta U = Q - W\) 在 4 种典型过程下分析: 等容（\(V\) 不变）/ 等压（\(P\) 不变）/ 等温（\(T\) 不变）/ 绝热（\(Q = 0\)）
  5. 能在简单案例中（气体压缩 / 加热水 / 摩擦生热 / 自由膨胀）识别 \(Q\)、\(W\)、\(\Delta U\) 的符号与数值关系
  6. 能用第一定律论证第一类永动机不可能（凭空生能违反 \(\Delta U = Q - W\)）；同时知道本板块不涉及反应方向 / 可逆性 / 熵 —— 这些留给第二定律基础及后续进阶板块
驱动问题

三个看上去无关的现象 ——

  • 摩擦双手会发热；烧水也会发热。这两件事微观上做的是同一件事吗？还是两种不同的传能方式？
  • 一团气体被压缩（外界对它做功）+ 同时被加热（外界给它热量），最后它的内能变化怎么算？两个效应加在一起吗？
  • "永动机"为什么不可能？这和"能量守恒"到底有什么严格的逻辑关系？

本板块的任务，是用一个简洁的等式 \(\Delta U = Q - W\) 把"热"和"功"这两件看似不同的事情统一到内能变化这一件事上 —— 并由此理解为什么"能量守恒"在热力学场景里的具体形式就是它。

（提示：先承认"热"和"功"都是"能量传递的方式"，本身都不是"系统的某个状态"。内能 \(U\) 是状态，\(Q\) 与 \(W\) 是过程 —— 这个区分是本板块的关键。）

1

第一定律的陈述 —— \(\Delta U = Q - W\)

▼
学习目标：从能量守恒的角度引出第一定律 \(\Delta U = Q - W\) → 理解符号约定（系统视角）→ 把它看作能量守恒在热力学系统的具体形式 → 由此论证第一类永动机不可能
建议时长：50-60 分钟
STEP 1先自己想（5-8 分钟）

不查笔记，先答 ——

  • 一团气体（系统）和它周围的环境（外界）之间，能量可以通过哪些方式互相传递？
  • "内能" \(U\) 是系统当前状态的一个量。如果系统的状态变了（温度变了 / 体积变了），\(U\) 应该会怎么变？这个变化 \(\Delta U\) 应该来源于哪里？
  • 常识告诉我们"能量不会凭空产生 / 凭空消失"。把这句话翻译到"系统 + 外界"这个模型，应该怎么说？
先回顾（核心理论根基 ⭐⭐）：上一板块 分子动理论 Card 5 给出了内能的微观定义 ——
\(U = \) 分子总动能 \(E_\text{动}\) \(+\) 分子总势能 \(E_\text{势}\)（其中 \(E_\text{动} \propto N \cdot T\)，\(E_\text{势}\) 取决于分子间距 / 三态结构）
而且 \(U\) 是状态量（只依赖系统当前的 \(N, T, V, \) 三态等，与到达这个状态的路径无关）。
正因为 \(U\) 是状态量，\(\Delta U = U_\text{末} - U_\text{初}\) 的差才有意义 —— 本板块的核心追问就是 \(\Delta U\) 由什么决定。如果对内能的微观定义不熟，先翻回去复习 5 分钟再回来。
STEP 2第一定律的陈述 + 符号约定

把上一步的直觉用一个等式精确化 ——

第一定律 — 从"能量守恒"到 ΔU = Q - W
系统视角：把研究对象（如一团气体）框出来当作"系统"，框外面的一切叫"外界"。
外界给系统能量的两种方式：① 热量 \(Q\)（通过温度差驱动的能量流）；② 做功 \(W\)（通过有方向的力作用驱动的能量流）。
能量守恒：外界给系统的总能量 \(=\) 系统内能的变化 \(\Delta U\)。
但约定 \(W\) 是系统对外做的功（系统输出 → 外界），所以外界给系统的功是 \(-W\)。
合起来: 内能变化 = 外界给的热 + 外界给的功 = \(Q + (-W) = Q - W\)。
\(\boxed{\,\Delta U = Q - W\,}\) ⭐⭐ — 这就是热力学第一定律。

符号约定（系统视角，本板块统一）:

量正方向（"+"）负方向（"−"）
\(Q\) 热量系统吸热（外界 → 系统）系统放热（系统 → 外界）
\(W\) 做功系统对外做功（系统 → 外界）
例：气体膨胀推活塞		外界对系统做功（外界 → 系统）
例：活塞压缩气体
\(\Delta U\) 内能变化系统内能增大（\(U_\text{末} \gt U_\text{初}\)）系统内能减小（\(U_\text{末} \lt U_\text{初}\)）
关于另一种符号约定：有的资料用 \(\Delta U = Q + W\)（约定 \(W\) 为外界对系统做的功）。两种约定在物理上完全等价，只是"\(W\)"指代的对象不同。本板块统一用 \(\Delta U = Q - W\)，\(W\) 是系统对外做的功 —— 后续板块全部沿用。换约定时切记换 \(W\) 的符号。
图 1 · 第一定律的能量流向示意。中间蓝色方框为"系统"（内能为 U）；红色箭头表示外界传给系统的热量 Q（吸热为正）；蓝色箭头表示系统对外做功 W（系统对外做功为正）。三者由等式 ΔU = Q − W 连接。
思考点 1 · 从能量守恒推 ΔU = Q − W
"能量不会凭空产生 / 凭空消失"是一句很泛的话。用"系统 + 外界"的模型把它精确地写下来 —— 它应该长什么样？为什么具体形式是 \(\Delta U = Q - W\) 而不是 \(\Delta U = Q + W\)？
能量守恒在系统视角下：外界传入系统的能量总和 = 系统内能增量。外界传入有 ① 热（量 \(Q\)）+ ② 做功（量 \(-W\)，因为我们约定 \(W\) 是系统对外做的功 → 外界对系统的功是 \(-W\)）。故 \(\Delta U = Q + (-W) = Q - W\)。如果换约定 \(W'\) 为外界对系统做的功，那等式就是 \(\Delta U = Q + W'\)。两种写法物理等价。
在纸上：① 画"系统 + 外界 + 两种能量流"的方框图；② 把"能量守恒"翻译成"外界给的总和 = 系统增量"；③ 代入 \(Q\) 和 \(-W\)（按本板块约定），推出 \(\Delta U = Q - W\)；④ 一句话总结"为什么 \(W\) 前面是减号"。
卡住时：想"如果 \(W\) 是系统给外界做的功，那这部分能量是离开系统的。系统内能要因此减少 —— 所以 \(W\) 前应该是负号。"
STEP 3第一定律 = 能量守恒 + 第一类永动机不可能

第一定律的真正深刻之处不是它"看上去是个公式"，而是它不是新原理 —— 它是能量守恒在热力学系统场景下的具体形式。

  • 力学已经有"能量守恒"（动能 + 势能 + ... = const），但这是不带热的形式
  • "热"是 19 世纪初新加入物理的能量传递方式（焦耳的热功当量实验：摩擦把功转成等量的热）
  • 热力学第一定律 = "能量守恒"把热也包含进来后的形式 —— \(\Delta U = Q - W\)
"第一类永动机"为什么不可能 —— 第一类永动机指"不消耗任何能量、却能持续对外做功"的机器。
按第一定律：系统对外做功 \(W \gt 0\)，又"不消耗能量"意味着 \(Q = 0\) 且系统内能不变 \(\Delta U = 0\)。
代入 \(\Delta U = Q - W\) → \(0 = 0 - W\) → \(W = 0\) —— 矛盾。
所以第一类永动机违反热力学第一定律，物理上不可能。⭐
思考点 2 · 永动机的两种"不可能"
第一类永动机违反第一定律。但还有"第二类永动机"，它不违反第一定律。它是什么？为什么也不可能？（不需要现在答全，留个引子）
第二类永动机：从单一热源吸热全部转化为功（满足能量守恒：\(Q = W, \Delta U = 0\)）。它满足第一定律 → 第一定律拦不住它。但它违反第二定律（不可逆性 / 自发方向），故也不可能。这正是为什么需要第二定律 —— 第一定律只关心"能不能在数量上守恒"，不关心"朝哪个方向走"。后续第二定律基础板块会展开。
在纸上：① 写"第一类永动机"违反第一定律的逻辑链；② 写"第二类永动机"满足第一定律但还是不可能 → 故需要第二定律；③ 一句话：第一定律负责能量数量，第二定律负责能量方向。
STEP 4批判性审查（5 分钟）
  • 第一定律里 \(U, Q, W\) 各是状态量还是过程量？为什么这个区分重要？
  • 如果一个系统的内能 \(U\) 不变（\(\Delta U = 0\)），是不是说明 \(Q = 0\) 且 \(W = 0\)？
  • 本板块只论"能量数量是否守恒"，不论"过程能否自发发生"。为什么这两件事不一样？
状态量 vs 过程量：
· \(U\) 是状态量：只依赖系统当前状态（\(N, T, V, \) 三态等），写法是 \(U\)（不写 \(\Delta U\) 也有意义）
· \(Q, W\) 是过程量：依赖"路径"（同样的始末态，走不同路径吸热和做功的量可以不同）。写法常用 \(Q, W\) 或 \(\delta Q, \delta W\)，不写 \(\Delta Q, \Delta W\)
这个区分在盖斯定律（化学反应焓变 \(\Delta H\) 路径无关）和进阶板块状态函数会再次强调。
✓ Card 1 检查点
素养字段：能量观 · 守恒律 · 系统建模 · 状态量 vs 过程量
2

热传导 vs 做功 —— 两种"给能量"方式的微观区分

▼
学习目标：宏观上 \(Q\) 和 \(W\) 都是"外界传能给系统"的方式，最终都体现为 \(\Delta U\)。但微观上这两种方式有本质区别 —— 本卡用分子图像把它讲清
建议时长：30-40 分钟
STEP 1先自己想（5 分钟）

两个场景，凭直觉答 ——

  • 烧水: 把一壶 20°C 的水放在火上，温度升到 80°C。外界给水的能量是怎么"进入"水分子的？
  • 摩擦生热: 双手反复搓动，手心从冷到温到热。外界给手的能量又是怎么"进入"皮肤分子的？
  • 这两种"传能"方式，微观图像看上去一样吗？

给一个核心猜测：热传导 \(Q\) = 大量分子的"无序"撞击传能；做功 \(W\) = 整体的"有序"位移传能。后面把这点讲清。

STEP 2微观对比 — 无序 vs 有序
图 2 · 热传导（左）与做功（右）的微观图像对比。左图用红色与蓝色小圆表示高温侧与低温侧的分子，每个分子带短箭头表示各方向无规则振动；高温分子振动剧烈，与低温分子相撞将能量"分散"传过去。右图用灰色矩形表示压缩气体的活塞，活塞向右推动后，气体中的分子普遍获得一个朝向同方向（向右）的额外速度；能量沿外力方向"定向"注入。

关键对比:

方面热传导 \(Q\)做功 \(W\)
驱动温度差有方向的外力（活塞推 / 摩擦 / ...）
微观图像分子无规则碰撞 → 传能分子被同方向推 → 传能
能量"形态"能量随机分散到所有方向能量集中在一个方向上注入
宏观举例烧水 / 太阳晒地球 / 火焰加热铁块气体被压缩 / 摩擦生热 / 拉伸橡皮筋
对 \(\Delta U\) 的贡献\(+Q\)（吸热）\(-W\)（系统对外做功）或 \(+|W|\)（外界对系统做功）
思考点 3 · 摩擦生热是 Q 还是 W？
两块铁互相摩擦，最后都变热了。这个过程中，对每一块铁而言，能量传递的方式是热 \(Q\) 还是功 \(W\)？为什么？
关键看"外界的能量是通过什么方式注入这块铁的"。摩擦时，对方的铁块以宏观有方向的力压在我这块铁上并相对滑动 → 我的表面分子被这个有方向的力定向推动 → 这是做功 \(W\)。注意：不是热传导（不存在温度差先于摩擦发生的情形）。从宏观上看，"摩擦生热"是功转化为内能的过程：外界对铁做功 → 铁内能增加 → 表现为温度升高。一旦温度升高，之后铁可能向环境热传导散热（这是 \(Q\) 出去），但"摩擦让铁热起来"的过程本身是做功。
在纸上：① 写"摩擦生热"过程中 \(Q\) 和 \(W\) 的判断；② 用 \(\Delta U = Q - W\) 写出"对每一块铁"的方程（注意 \(W\) 是外界对它做的功 → 在系统对外做功约定下应取负号 \(W \lt 0\) → \(\Delta U = Q - W = 0 - (\text{负数}) = \text{正}\)）；③ 一句话总结"功 vs 热的判据：看微观上能量注入是有方向的还是无方向的"。
"焦耳的热功当量实验"就是这个图像 — 焦耳用重物下落驱动桨叶搅动水，做功让水内能增大、温度升高。最终他得出"\(1\) 卡 \(\approx 4.18\) 焦耳" — 功和热可以等量互换。
STEP 3反思 — 两种方式宏观等价、微观不等价
  • 宏观上: \(Q\) 和 \(W\) 都改变 \(\Delta U\)，对 \(U\) 的最终值贡献等价（差的只是符号）。第一定律 \(\Delta U = Q - W\) 把它们加在一起
  • 微观上: 二者的"能量注入方式"完全不同 —— 无序 vs 有序
  • 这就是为什么 \(Q\) 和 \(W\) 是独立的过程量，不能由内能反推（同样的 \(\Delta U\) 可以由多种 \(Q, W\) 组合实现）
  • 这个"微观差异"是后续板块的重要线索：第二定律会问"做功能否 100% 转化为热？热能否 100% 转化为做功？" —— 二者并不对称，这与微观图像中的"有序 / 无序"密切相关（细节留给后续板块）
✓ Card 2 检查点
素养字段：宏观-微观桥接 · 概念区分 · 物理图像

4 种典型过程 + 综合应用

▼
学习目标：把 \(\Delta U = Q - W\) 应用到 4 类典型过程 —— 等容 / 等压 / 等温 / 绝热。每类的关键特征是"哪一个量被固定 / 哪一个量等于零"，由此简化第一定律的等式
建议时长：50-60 分钟
STEP 14 类过程的概览

对一团理想气体，常考的 4 种过程及第一定律的简化形式 ——

过程固定量关键简化第一定律变形
等容（isochoric）\(V\) 不变\(W = p \cdot \Delta V = 0\)\(\Delta U = Q\)（吸热全部变内能）
等压（isobaric）\(P\) 不变\(W = p \cdot \Delta V\)\(\Delta U = Q - p \cdot \Delta V\)
等温（isothermal）\(T\) 不变对理想气体 \(\Delta U = 0\)\(Q = W\)（吸热全部用来做功）
绝热（adiabatic）\(Q = 0\)（不吸不放热）—\(\Delta U = -W\)（做功全部来自内能消耗）
先回顾：等温过程下"理想气体 \(\Delta U = 0\)"这一步用到的是 分子动理论 Card 5 的结论 ——理想气体的内能只依赖温度（动能项 \(\propto N \cdot T\)，势能项 \(\approx 0\) 因为气体分子间距 \(\gg r_0\)）。等温意味着 \(T\) 不变、\(N\) 不变 → \(U\) 不变 → \(\Delta U = 0\)。
注意这一结论只对理想气体成立；实际气体在等温下 \(\Delta U\) 可能略不为零（因为有分子间相互作用势能），不在本板块范围。

4 过程的 P-V 图示意（每条线都从同一个起点出发，看终点跑去了哪里）:

图 3 · 4 过程的 P-V 图（同一起点 A 出发）。红色垂直线 = 等容过程（V 不变，沿垂直方向移动）；橙色水平线 = 等压过程（P 不变，沿水平方向移动）；绿色曲线 = 等温过程（PV = const 双曲线，自 A 向右下方走）；紫色虚线 = 绝热过程（PVγ = const，γ > 1，所以绝热膨胀比等温膨胀压强下降更快、曲线更陡）。同样从 A 出发膨胀（V 增大），等温终点 P 下降较少，绝热终点 P 下降较多。
STEP 24 过程的物理图像 + 一题一例
思考点 4 · 等容加热
把一团气体装在刚性容器（V 不能变）里加热。这是什么过程？外界给的能量都跑哪去了？\(\Delta U\) 和 \(Q\) 什么关系？
等容过程：\(V = \) const → \(\Delta V = 0\) → \(W = p \cdot \Delta V = 0\) → 第一定律变为 \(\Delta U = Q\)。
物理图像：刚性容器装气体，吸收的热量全部变成内能（分子动能增大 → 温度升高）。这是 4 过程中"最简单"的一个。
在纸上：① 写出等容过程的特征 \(V = \) const；② 由 \(W = p \cdot \Delta V\) 推出 \(W = 0\)；③ 代回 \(\Delta U = Q - W\) → \(\Delta U = Q\)；④ 用一句话总结物理图像。
思考点 5 · 等压膨胀
气体被压在无重活塞下方，外界压强是常数 \(p\)。加热气体，活塞被推起来 \(\Delta V\)。这是什么过程？\(\Delta U\)、\(Q\)、\(W\) 各是什么？
等压过程：\(P = \) const。气体顶活塞做功的推导 —— 活塞底面积 \(A\)，被推位移 \(\Delta x\)。气体对活塞施压力 \(F = p \cdot A\)（外压恒等于气压 \(p\)）→ 做功 \(W = F \cdot \Delta x = p \cdot A \cdot \Delta x\)。注意到 \(A \cdot \Delta x = \Delta V\)（活塞扫过的体积），故 \(W = p \cdot \Delta V\)。几何上 = P-V 图中"等压过程线"下方那块矩形的面积。
代回第一定律 → \(\Delta U = Q - p \cdot \Delta V\)。物理图像：吸收的热量一部分变成内能（温度升高），另一部分用来对外做膨胀功（顶活塞）。
注意这里 \(\Delta U \lt Q\)（因为有一部分给了功）— 与等容（\(\Delta U = Q\)）相比，加同样的热温度升得少一些。
在纸上：① 自己推一遍 \(W = p \cdot \Delta V\)（从 \(F = pA\)、\(W = F \cdot \Delta x\) 推出）；② 写第一定律的等压形式；③ 在 P-V 图上画"等压过程线下方矩形面积 = W"的几何意义；④ 用一句话总结"为什么等压加热温度升得比等容少"。
思考点 6 · 等温膨胀
理想气体被装在导热壁容器里、与恒温浴接触。让它慢慢膨胀（推活塞），温度始终保持不变。这是什么过程？\(\Delta U\)、\(Q\)、\(W\) 什么关系？
等温过程：\(T = \) const。对理想气体，内能只依赖温度 → \(\Delta U = 0\)。第一定律变为 \(Q = W\)。
物理图像：气体膨胀对外做功 \(W\) → 内能本应下降 → 但导热壁让恒温浴源源不断补充热量 \(Q\) 进来恰好抵消 → 温度保持不变。
核心：等温膨胀时"吸收的热量全部变成了对外做的功"（\(\Delta U = 0\) 是关键）。
在纸上：① 用"理想气体 \(U\) 只依赖 T"得 \(\Delta U = 0\)；② 代第一定律得 \(Q = W\)；③ 在 P-V 图上画等温膨胀的双曲线 + 在图下方面积标"= W = Q"。
"理想气体内能只依赖 T"是关键。等温膨胀做功的具体公式 \(W = nRT \ln(V_2/V_1)\) 留给进阶板块（卡诺循环 / 微观推导）。
思考点 7 · 绝热膨胀
气体被装在绝热壁容器里（不与外界交换热量），让它快速膨胀。这是什么过程？\(\Delta U\)、\(Q\)、\(W\) 什么关系？气体温度会怎么变？
绝热过程：\(Q = 0\)。第一定律变为 \(\Delta U = -W\)（注意符号！）。
物理图像：膨胀时气体对外做功 \(W \gt 0\) → 内能必须减少 \(|W|\) 来弥补（因为没有热量补充） → 内能减少 → 温度下降。
反过来：绝热压缩（\(W \lt 0\) 外界对系统做功）→ 内能增加 → 温度升高。日常例子：自行车打气筒底部很烫（绝热压缩升温）；高山上空气稀薄、温度低（绝热膨胀降温）。
在纸上：① 写出 \(Q = 0\) → \(\Delta U = -W\)；② 解释为什么绝热膨胀温度下降；③ 画绝热曲线 P-V 图，比较与等温曲线的相对陡峭程度。
绝热为什么比等温更陡（关键推理）：由 \(PV = nRT\) → \(P = nRT/V\)。等温过程 \(T\) 不变 → \(P\) 单纯随 \(V\) 反比下降；绝热膨胀\(V\) 增大的同时 \(T\) 也下降（如上所述）→ 分子分母双向作用让 \(P\) 下降得更多 → 绝热曲线在 P-V 图上比等温曲线更陡。
绝热过程的精确方程 \(PV^\gamma = \) const（其中 \(\gamma = C_p / C_v\)）留给进阶卡诺循环板块推导。本板块只要求定性理解"绝热膨胀 → 内能减少 → 温度下降"。
STEP 3综合应用 — 复合过程

真实场景常常是多个过程接连发生。第一定律由于是状态量等式（\(\Delta U\) 只依赖始末态），所以处理复合过程很方便：

处理复合过程的 3 步法：
① 把过程拆成简单段（每段是等容 / 等压 / 等温 / 绝热中的一种）
② 每段单独用第一定律算 \(\Delta U_i = Q_i - W_i\)
③ 累加: \(\Delta U_\text{总} = \sum_i \Delta U_i\)、\(Q_\text{总} = \sum_i Q_i\)、\(W_\text{总} = \sum_i W_i\)
注意 \(\Delta U_\text{总}\) 只看始末态（因为 U 是状态量），即使中间路径不同，总 \(\Delta U\) 相同。
思考点 8 · 综合实例 — 压缩后冷却
一团气体先绝热压缩（外界对它做功 5 焦耳；注意按本板块约定 \(W\) 是系统对外做的功 → 外界对系统做功 5 J 等价于系统对外做功 \(W_1 = -5\) J）；然后等容冷却（系统放热 3 焦耳，按约定吸热为正 → 放热 \(Q_2 = -3\) J，且等容 \(W_2 = 0\)）。总过程的 \(\Delta U_\text{总}\)、\(Q_\text{总}\)、\(W_\text{总}\) 各是多少？
第一段（绝热压缩）：\(Q_1 = 0, W_1 = -5\) J → \(\Delta U_1 = 0 - (-5) = +5\) J。
第二段（等容冷却）：\(Q_2 = -3\) J, \(W_2 = 0\) → \(\Delta U_2 = -3 - 0 = -3\) J。
累加：\(\Delta U_\text{总} = +5 + (-3) = +2\) J；\(Q_\text{总} = 0 + (-3) = -3\) J；\(W_\text{总} = -5 + 0 = -5\) J。
验证：\(\Delta U_\text{总} = Q_\text{总} - W_\text{总} = -3 - (-5) = +2\) J ✓ 与逐段累加一致。
物理结论：气体最终内能增加了 2 焦耳，过程中净吸热 -3 焦耳（实际放热 3 焦耳），外界对它做功 5 焦耳。
在纸上：① 算每段 \(\Delta U_i\)；② 累加得总；③ 用第一定律对总过程做一次验证（应该与累加结果一致 — 因为 U 是状态量）。
