关于本板块的深度口试

本板块是热力学大主题进阶链终点：基础版给你"会用 ΔG = ΔH − TΔS 判自发性"；进阶版要求你说清楚为什么 —— G 是怎么从熵增定律严格推出来的、ΔG 在物理上意味着什么有用功、它怎么把电化学与化学平衡接上同一根线。

口试由教师当面进行，从下方 6 条主问题中抽 3 条。每条主问题先开放讲，再由教师根据你的回答追问。讲推导链 ＞ 报数—— 教师听的是你能不能把"宇宙视角 → 系统视角"的等价改写讲透。

📌 通关条件：抽中的 3 条里，至少 2 条把核心推导讲清楚并能回应一层「为什么」，且整场至少一次在纸上写出 G ≡ H − TS 并指着定义说清每一项含义与等温等压前提。

口试流程 总时长 15-20 分钟

① 暖场 · 自述
0-3 分钟
你先讲：「进阶版 vs 基础版，对我理解 ΔG 最大的改变是什么？」—— 不卡时间，讲到能切入主问题就行。
② 主问题 ×3
3-15 分钟
从下方 6 条主问题中由教师抽 3 条，每条约 4-5 分钟。先开放讲，再回答追问。
③ 推导手稿 / 错题联动
15-18 分钟
带 1-2 张自己整理的推导手稿（4 步链 / dG 推 W_max 链 / ΔG° = −RT ln K 推导），现场挑一步教师指着问「这步为什么能这么写」；或带 1 道 mock-check 错题讲反思。
④ 判定 · 签字
18-20 分钟
通过 = 当场签字；暂缓 = 给补学方向 + 预约二试（换未用过的主问题）；不通过 = 回到学习阶段重学。
主问题 1

G 的引入动机与定义 —— 写出来再说话

约 4-5 分钟
🎤 开放主问：请你在纸上写出 G 的严格定义，然后告诉我：第二定律已经有了 ΔS宇宙 ≥ 0 这条判据，为什么还要再引入 G？换句话说，G 是来"取代"熵增判据的，还是来"翻译"它的？
👂 教师在听这些要点
  • ⭐⭐ G ≡ H − TS 严格定义 能在纸上写出，并指认 H、T、S 每一项含义（T 是系统温度，不是环境温度）。
  • ⭐⭐ G 引入的物理动机 "宇宙视角"换"系统视角" —— 为方便计算，不是因为熵增判据"不对"。
  • ⭐ G 是状态函数 H、T、S 都是状态量；代数组合仍是状态量 ⇒ ΔG 路径无关。
  • 盖斯定律 for G 主动提 ΔG 也可以拆步算 / 用 ΔGf° 表。
主问题 2

从 ΔS宇宙 ≥ 0 严格推 ΔG ≤ 0 —— 四步链复现

约 5 分钟
🎤 开放主问：有人说「ΔG ≤ 0 是一条独立的自发性判据」。请你不要直接反驳，而是在纸上从 ΔS宇宙 ≥ 0 出发，一步一步推到 ΔG ≤ 0，途中每一步要交代你用了什么假设。最后告诉我：这两条判据之间的关系到底是什么？
👂 教师在听这些要点
  • ⭐⭐ 4 步推导链 ΔS宇宙 = ΔS系 + ΔS环 → 代入 ΔS环 = −ΔH系/T → 整理 → 乘 −T 反转得 ΔG ≤ 0。本主问题核心。
  • ⭐ 环境熵变 ΔS环 = −Qp/T 等压前提 + 大热库假设 + 负号来自系统视角换环境视角。
  • ⭐ 乘 −T 反转不等号 代数事实（T > 0 ⇒ 乘 −T 必反转）+ 物理含义（两条判据等价改写）。
  • ⭐⭐ ΔG ≤ 0 ⟺ ΔS宇宙 ≥ 0 等温等压下完全等价；不是"取代"是"翻译"。
主问题 3

ΔG = −Wmax,非膨胀 —— 为什么叫"自由"能

约 4-5 分钟
🎤 开放主问：等温等压下，一个反应的 ΔH = −100 kJ/mol、TΔS = −30 kJ/mol。这个反应能对外界提供的最大非膨胀有用功是多少？为什么剩下的那部分焓变不能也拿来做功？顺便告诉我：「自由能」里"自由"两个字到底自由在哪？
👂 教师在听这些要点
  • ⭐⭐ ΔG = −Wmax,非膨胀 算出 ΔG = −70 kJ/mol ⇒ Wmax,非膨胀 = 70 kJ/mol；强调"最大"、"可逆才取等"、"等温等压前提"。
  • ⭐ 可逆 = 最大功的物理原因 不可逆过程的额外熵产生 T·dS额外 必须以热的形式放掉，吃掉了可做功的份额。
  • 「自由」的含义 ΔG 这部分焓变是可被"自由用作有用功"的；TΔS 那部分必须以热释放维持总熵不减，不是凭空多出。
  • dG = −δW非膨胀,可逆 推导 学生若展开 8 步链 → 记为亮点。
主问题 4

电化学桥 ΔG = −nFE —— 把化学能换成电能

约 4-5 分钟
🎤 开放主问：一个铅酸电池放电反应 ΔG° = −393.0 kJ/mol，每生成 1 mol 产物转移 2 mol 电子。请算它的标准电动势 E°。算之前请先告诉我：为什么自发放电对应 E° > 0、ΔG° < 0？这套符号约定是怎么和「电池要给外界供能」配合上的？
👂 教师在听这些要点
  • ⭐⭐ 电化学桥 ΔG = −nFE 写出公式 + 算出 E° = −ΔG°/(nF) = 393000/(2 × 96485) ≈ 2.04 V；强调 kJ → J 换算与 n = 转移电子数（不是反应物摩尔数）。本主问题核心。
  • 单位与符号双校验 ΔG 单位 J，F = 96485 C/mol，E 单位 V；自发 ⇔ E > 0 ⇔ ΔG < 0。
  • 理论电压 vs 实测电压 E° 是可逆上限；实测低于 E° 是不可逆耗散，不是公式错；E° 提供"上限 + 基准 + 诊断"。
主问题 5

反应商 Q + 平衡常数桥 ΔG° = −RT ln K —— ΔG 到底跟谁走

约 5 分钟
🎤 开放主问：有人讲：「合成氨在 25 ℃ 下 ΔG° ≈ −33 kJ/mol < 0，所以无论我怎么放气体浓度，反应都向产物自发。」这句话错在哪？请你用 ΔG = ΔG° + RT ln Q 与 ΔG° = −RT ln K 这两条把他纠正过来；并告诉我：平衡的本质判据到底是什么？
👂 教师在听这些要点
  • ⭐⭐ ΔG° = −RT ln Keq 写出公式 + 指出 ΔG° 是标准态固定参数（只随 T 变）。
  • ⭐ ΔG = ΔG° + RT ln Q ΔG 随工况浓度变；判方向看 ΔG 而非 ΔG°。
  • ⭐ ΔG 符号 → 方向 ΔG < 0 向产物 / > 0 向反应物 / = 0 平衡；平衡判据是 Q 追上 K（即 Q = K ⇒ ΔG = 0），不是 ΔG° = 0。
  • 反应商 Q 的定义 产物在上 / 反应物在下 / 带化学计量数次幂；气相分压 / 溶液浓度 / 纯固液不入式。
  • Keq 与 ΔG° 符号对应 K > 1 ⟺ ΔG° < 0；K 只随 T 变。
主问题 6

范特霍夫方程 + 合成氨折中 —— K 为什么随温度走

约 4-5 分钟
🎤 开放主问：合成氨 N2 + 3H2 → 2NH3，ΔH° ≈ −92 kJ/mol（放热）。请你从范特霍夫方程出发，推理 Keq 是随温度升高变大还是变小，每一步要交代符号怎么来的。然后告诉我：既然低温对 K 更有利，为什么工业上还要把炉温做到 400-500 ℃？
👂 教师在听这些要点
  • ⭐ 范特霍夫方程（微分形式） 写出 d ln K / dT = ΔH° / (RT2)；分母 RT2 > 0 恒正，符号完全由 ΔH° 决定。本主问题核心。
  • 放热 / 吸热 K 走向 放热 ΔH° < 0 ⇒ d ln K / dT < 0 ⇒ T↑则 K↓；不能只背"勒夏特列"。
  • 合成氨工业折中 热力学(低温利 K) vs 动力学(高温利速率) vs 经济(成本) —— 多维妥协，不是单维优化。
  • ΔG° vs T 直线（可选展开）截距 = ΔH°、斜率 = −ΔS°、x 截距 = T转 = ΔH°/ΔS°。

📝 怎么准备

  • 不要背口诀。教师追问的第一刀往往就是「为什么」—— 把每个核心结论的推导链 + 一句话物理理由想清楚（比如：4 步推导链里"乘 −T 反转不等号"是因为 T > 0 必反转，物理含义是从宇宙视角换系统视角）。
  • 带纸和笔。主问题 1、2、3、4、5、6 都涉及在纸上写定义 / 推导链 / 计算 —— 教师会指着式子追问每一项含义与符号来源。
  • 带 1-2 张推导手稿（4 步链 / dG = −δW非膨胀,可逆 推导链 / ΔG° = −RT ln K 推导链均可），手稿不要求漂亮，要求你能现场指着任一步讲来源。这是流程第 ③ 步。
  • 边界声明给你「松绑」：本板块有些「更深层」的概念不在要求内，没学过也不扣分；如果主动谈到是亮点，教师会简短肯定但不展开（这些深入内容在后续板块或进阶板块里学）。
  • 讲推导链 ＞ 报答案。教师听的是因果重建：你能不能把"宇宙视角 → 系统视角"的等价改写、"ΔG = 最大有用功"的物理意义讲清楚，比报出数值更重要。
