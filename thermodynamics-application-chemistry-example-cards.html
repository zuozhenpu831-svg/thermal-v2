1
热化学方程式辨析 · 4 燃料对比
A3
基础
题目
某新能源研究室对比 3 种家用 / 车用燃料的能量密度. 已知 (298 K, 101 kPa):
· CH4(g) 燃烧热 ΔH1 = -890.3 kJ/mol
· H2(g) 燃烧热 ΔH2 = -285.8 kJ/mol
· C2H5OH(l) 燃烧热 ΔH3 = -1366.8 kJ/mol

下列热化学方程式中, ΔH 表示燃烧热的是 ( )

A. 2CH4(g) + 4O2(g) → 2CO2(g) + 4H2O(l), ΔH = -1780.6 kJ/mol
B. H2(g) + ½O2(g) → H2O(g), ΔH = -241.8 kJ/mol
C. C2H5OH(l) + 3O2(g) → 2CO2(g) + 3H2O(l), ΔH = -1366.8 kJ/mol
D. CH4(g) + 2O2(g) → CO2(g) + 2H2O(g), ΔH = -802.3 kJ/mol
📚 ① 知识点梳理
  • 涉及燃烧热定义的 4 大约定 (锚: 燃烧热与中和热 Card 1 STEP 2):
    • ① 1 mol 可燃物 (不是 1 mol O2, 也不是 2 mol 可燃物)
    • ② 完全燃烧 (C→CO2, H→H2O, S→SO2)
    • ③ 稳定氧化物 (H2O 必须是 液态 l)
    • ④ 标准条件 (25°C, 101 kPa)
  • 跨板块综合: 用『燃烧热的严格定义』辨识 4 个看似合理的热化学方程式
🧭 ② 解题思路与方法
  • 拆题: 已知 = 4 个候选方程式; 求 = 哪个符合燃烧热全部 4 大约定
  • 选路径: 用逐项辨识法 — 4 选项逐个对照 4 大约定
  • 执行计划:
    • 查 ① 可燃物系数是否 = 1
    • 查 ② 是否完全燃烧
    • 查 ③ 产物物态是否 H2O(l)
    • 查 ④ 数值是否合理 (与已知燃烧热数据一致)
📝 ③ 详细解题步骤
A 2CH4 + 4O2 → 2CO2 + 4H2O(l), ΔH = -1780.6 kJ/mol
查约定: 可燃物 = 2 mol CH4, 违反 ① "1 mol 可燃物".
注: 数值 -1780.6 = 2 × (-890.3), 数据本身没问题, 但归一化错.
→ 违反约定 ① → ❌ 不是燃烧热
B H2 + ½O2 → H2O(g), ΔH = -241.8 kJ/mol
查约定: 可燃物 = 1 mol ✓; 完全燃烧 ✓; 但 H2O 是气态 (g), 违反 ③ "稳定氧化物 H2O(l)".
注: -241.8 是低位热值 (LHV), 与已知 -285.8 (高位 HHV, H2O(l)) 差 +44 kJ/mol = 水汽化热.
→ 违反约定 ③ → ❌ 不是燃烧热
C C2H5OH(l) + 3O2 → 2CO2 + 3H2O(l), ΔH = -1366.8 kJ/mol
查约定: 可燃物 = 1 mol C2H5OH ✓; 完全燃烧 (C→CO2, H→H2O) ✓; H2O(l) ✓; 数值 = -1366.8 与已知一致 ✓
→ 全部 4 约定满足 → ✓ 是燃烧热
D CH4 + 2O2 → CO2 + 2H2O(g), ΔH = -802.3 kJ/mol
查约定: 可燃物 = 1 mol ✓; 但 H2O 是气态 (g), 违反 ③.
注: -802.3 = -890.3 + 2 × 44 (减去 2 mol 水液化热), 是 CH4 的低位热值.
→ 违反约定 ③ → ❌ 不是燃烧热
最终答案: C
✓ 验证: 选项 C 的 1 mol 可燃物 + 完全燃烧 + 稳定氧化物 H2O(l) + 数值一致, 4 约定全过. 其他 3 项各破 1 约定. 选 C 唯一.
⚠️ ④ 易错点
易错 1: 把 "1 mol 可燃物" 错记为 "1 mol O2"
❌ 错答: 看到 A 项 4 mol O2 就觉得『O2 系数不对』
✓ 正答: O2 系数随可燃物变化, 但可燃物必须 1 mol(用 O2 归一无法对比不同可燃物)
易错 2: 忽略 H2O 物态
❌ 错答: 看 B 选项 H2 + ½O2 → H2O 觉得『1 mol 可燃物 + 完全燃烧』就对
✓ 正答: 燃烧热约定 H2O(l) (高位热值 HHV), 写成 H2O(g) 是低位热值 LHV, 差 +44 kJ/mol 水汽化热, 不算燃烧热
易错 3: 把 低位热值当燃烧热
❌ 错答: D 选项 -802.3 = -890.3 + 88, 觉得"数值算对了就行"
✓ 正答: 燃烧热定义本身要求 H2O(l). 工程上工业锅炉常用低位热值(蒸汽不冷凝回收), 但化学反应原理严格定义用高位
易错 4: 数值看对了但 归一化忘检
❌ 错答: A 选项 -1780.6 = 2 × (-890.3) 数值合理就觉得对
✓ 正答: 燃烧热严格要求 1 mol 可燃物 归一. 写 2 mol CH4 的方程式即便数值对, ΔH 也不是燃烧热
📖 锚回: 燃烧热与中和热 task-cards · 基础检测 · 盖斯定律 task-cards
2
多反应组合 · 工业制甲醇 ΔH 选择
A2.1
中等
题目
工业上 CO2 催化加氢制甲醇 (CH3OH) 是『双碳目标』的重要研究方向. 已知:
· 反应 ①: CO2(g) + H2(g) → CO(g) + H2O(g), ΔH1 = +41.2 kJ/mol
· 反应 ②: CO(g) + 2H2(g) → CH3OH(g), ΔH2 = -90.8 kJ/mol
· 反应 ③: H2O(g) → H2O(l), ΔH3 = -44.0 kJ/mol

则目标反应 CO2(g) + 3H2(g) → CH3OH(g) + H2O(l) 的 ΔH 为 ( )

A. -93.6 kJ/mol
B. -49.6 kJ/mol
C. +5.6 kJ/mol
D. -137.6 kJ/mol
📚 ① 知识点梳理
  • 涉及盖斯定律 4 大代数规则 (锚: 盖斯定律 Card 2):
    • 加规则: ① + ② → ΔH合 = ΔH1 + ΔH2
    • 减规则: ① - ②
    • 反向规则: ① 反向 → ΔH-1 = -ΔH1 (符号取负)
    • 乘 k 规则: ① × k → ΔH 也乘 k
  • 涉及状态函数性质 ΔH = H末 - H初 (路径无关)
  • 跨板块综合: 多反应组合凑出目标反应 + 用代数加法算 ΔH
🧭 ② 解题思路与方法
  • 拆题: 已知 = 3 反应 + 各自 ΔH; 求 = 目标反应 ΔH; 桥接 = 用 ①②③ 代数组合凑出目标
  • 选路径: 用逐物质消元法 (净计量数法) 定各反应系数
  • 执行计划:
    • 对每个物质列净计量数方程 (目标方程系数 = 各已知反应贡献系数之和)
    • 解出 ①②③ 系数 (反向 / 乘 k)
    • 累加 ΔH
    • 量纲 / 数值合理性验证
📝 ③ 详细解题步骤
1 用净计量数法对每个物质列方程, 设组合系数 a·① + b·② + c·③:
· CO2: 目标左 1 → a·1 + b·0 + c·0 = 1 → a = 1
· CO: 目标 0 → a·(-1) + b·(-1) + c·0 = 0(因 ① 右 1 → a; ② 左 1 → -b)→ a - b = 0 → b = 1
· H2O(g): 目标 0 → a·1 + b·0 + c·(-1) = 0 → c = a = 1
· H2O(l): 目标右 1 → c·1 = 1 → c = 1 ✓
· 验 H2: 目标左 3 → a·1 + b·2 = 1 + 2 = 3 ✓
· 验 CH3OH: 目标右 1 → b·1 = 1 ✓
→ 系数 ① = 1, ② = 1, ③ = 1 (全部正向)
2 累加 ΔH:
ΔH目标 = 1·ΔH1 + 1·ΔH2 + 1·ΔH3
= (+41.2) + (-90.8) + (-44.0)
= 41.2 - 90.8 - 44.0
= -93.6 kJ/mol
→ 由加规则
3 物理含义验证: ΔH < 0 → 放热反应. 合理(气相 → 液相 + 形成 CH3OH 释能).
最终答案: A (-93.6 kJ/mol)
✓ 验证: 量纲 kJ/mol ✓; 放热合理(双碳工业反应放热); 数值与 ① + ② + ③ 累加自洽. 干扰项 B (-49.6) = 忘加 ③(液化); C (+5.6) = 反向 ② 错; D (-137.6) = ② 系数算错.
⚠️ ④ 易错点
易错 1: 忘考虑 H2O 物态变化(③ 液化)
❌ 错答: 直接 ① + ② = -49.6 kJ/mol → 选 B
✓ 正答: 目标反应右是 H2O(l), ① 反应生成 H2O(g) → 必须加 ③ 液化步骤
易错 2: 系数认错(②, ③ 是否反向)
❌ 错答: 觉得 ② 应反向, ΔH 取负
✓ 正答: 检查 CO 在目标中 0, ② 中 CO 在左, ① 中 CO 在右 → ① 右 + ② 左 = ① + ② 正向消 CO. 没有反向
易错 3: 忘对 每个物质列方程, 漏 H2O(g)
❌ 错答: 只对反应物 CO2, H2 列方程
✓ 正答: 必须对所有出现的物质列净计量数方程, 包括中间体(CO) 和不同物态的同物质 (H2O(g) vs H2O(l))
易错 4: 符号翻车(正反号)
❌ 错答: 算 ΔH 时把 ΔH1 = +41.2 当负数
✓ 正答: ΔH1 = +41.2 kJ/mol(吸热), 累加时保留正号. 反向才取负, 正向用原符号
📖 锚回: 盖斯定律 Card 2 (4 大代数规则) · 盖斯定律基础检测
3
多反应组合填空 · 水煤气变换 ΔH
A2.2
中等
题目
工业制氢的核心反应是水煤气变换反应 (Water-Gas Shift): 用 CO 与 H2O 生成 H2 和 CO2. 已知 (298 K, 101 kPa):
· 反应 ①: C(s) + ½O2(g) → CO(g), ΔH1 = -110.5 kJ/mol
· 反应 ②: H2(g) + ½O2(g) → H2O(g), ΔH2 = -241.8 kJ/mol
· 反应 ③: C(s) + O2(g) → CO2(g), ΔH3 = -393.5 kJ/mol

则水煤气变换反应: CO(g) + H2O(g) → CO2(g) + H2(g) 的 ΔH = ________ kJ/mol
📚 ① 知识点梳理
  • 涉及盖斯定律 + 净计量数法 (锚: 盖斯定律 Card 2)
  • 涉及反向规则: 反应反向 → ΔH 取负 (锚: 盖斯定律 Card 2)
  • 跨板块综合: 利用『不能直接测量的反应 ΔH (水煤气变换实际可测但本题练方法) 可借能直接测的反应组合算出』 (路径无关的工程价值)
🧭 ② 解题思路与方法
  • 拆题: 已知 = 3 反应 + ΔH; 求 = 目标反应 ΔH; 桥接 = 用 ①②③ 组合
  • 选路径: 净计量数法 + 反向/加和; 注意目标物质 CO, H2O(g), CO2, H2 都在已知中, 找出哪些反应反向
  • 执行计划:
    • 识别目标反应物 / 产物在 3 个已知反应中的位置
    • 定哪些反应反向 (取负 ΔH)
    • 累加 ΔH
    • 验证消元正确
📝 ③ 详细解题步骤
1 识别目标反应中各物质在已知反应的位置:
· CO: 目标左 (反应物), 但 ① 中 CO 在右 → ① 反向(取负)
· H2O(g): 目标左, ② 中 H2O 在右 → ② 反向(取负)
· CO2: 目标右, ③ 中 CO2 在右 → ③ 正向
· H2: 目标右, ② 反向后 H2 在右 ✓ (由 ② 反向贡献)
· C(s): 目标 0, ① 反向 + ③ 正向 → -C + C = 0 ✓
· O2: 目标 0, ① 反向 + ② 反向 + ③ 正向 → -½ - ½ + 1 = 0 ✓
→ 组合 = -① - ② + ③
2 验证目标反应:
-① = CO → C + ½O2; -② = H2O(g) → H2 + ½O2; ③ = C + O2 → CO2
累加: CO + H2O(g) + C + O2 → C + ½O2 + H2 + ½O2 + CO2
消 C, O2: CO + H2O(g) → H2 + CO2 ✓ 目标!
→ 由消元法验证
3 累加 ΔH:
ΔH = -ΔH1 - ΔH2 + ΔH3
= -(-110.5) - (-241.8) + (-393.5)
= +110.5 + 241.8 - 393.5
= 352.3 - 393.5
= -41.2 kJ/mol
→ 由反向规则 + 加规则
最终答案: ΔH = -41.2 kJ/mol
✓ 验证: 量纲 kJ/mol ✓; 实际水煤气变换实验值 ≈ -41.2 kJ/mol(放热反应)✓; 物理含义合理(CO 较活泼, 变成更稳定的 CO2, 释能).
⚠️ ④ 易错点
易错 1: 反向规则忘取负号
❌ 错答: -ΔH1 时直接写 ΔH1 = -110.5 当 -110.5 用
✓ 正答: -ΔH1 = -(-110.5) = +110.5. 反向规则: ΔH 取负 = 原 ΔH 加负号
易错 2: 系数定向错
❌ 错答: 不分析目标位置直接 ① + ② + ③ 累加
✓ 正答: 逐个物质看在目标 / 已知中的位置 → 若反向定哪个反应反 → 才能确定 ±
易错 3: 忘消元验证
❌ 错答: 算完 ΔH = -41.2 就提交, 没验消元
✓ 正答: 必须把累加后的方程消元 → 看是否得到目标反应. 若中间体没消干净 (C / O2), 系数错
易错 4: 填空题忘单位
❌ 错答: 填 "-41.2"
✓ 正答: 答案是 ΔH = -41.2 kJ/mol. 高考填空若单位题面给, 数值即可; 若没给, 必须带单位

4
ΔHf° 表算综合 · 葡萄糖代谢能量
A4
较难
题目
生命科学研究: 人体内葡萄糖完全氧化代谢(对应有氧呼吸总反应) 为:
    C6H12O6(s) + 6O2(g) → 6CO2(g) + 6H2O(l)

已知标准摩尔生成焓(298 K, kJ/mol):
· ΔHf°(C6H12O6, s) = -1273.3
· ΔHf°(O2, g) = 0 (稳定单质)
· ΔHf°(CO2, g) = -393.5
· ΔHf°(H2O, l) = -285.8

葡萄糖摩尔质量 M = 180 g/mol.

(1) 求葡萄糖完全氧化反应的 ΔH = ________ kJ/mol;
(2) 一颗中等大小苹果含约 27 g 可代谢葡萄糖, 完全代谢可释放能量约 ________ kJ;
(3) 解释为何此反应可看作 6 个『C(s) + O2 → CO2』 + 6 个『H2 + ½O2 → H2O(l)』 + 1 个『分解葡萄糖为稳定单质』的代数加和 — 这就是 ΔHf° 公式末减初的物理本质.
📚 ① 知识点梳理
  • 涉及标准摩尔生成焓 ΔHf°定义: 25°C, 1 atm 下『稳定单质 → 1 mol 物质』反应的 ΔH (锚: 盖斯定律 Card 3)
  • 涉及反应焓变公式: ΔH反应 = Σν·ΔHf°(产) - Σν·ΔHf°(反), 按化学计量数加权
  • 涉及稳定单质零点参考: ΔHf°(O2) = 0, ΔHf°(石墨 C) = 0, ΔHf°(H2) = 0
  • 跨板块综合: ΔHf° 公式 + 摩尔数 → kJ 总能量(代谢应用)
🧭 ② 解题思路与方法
  • 拆题: 3 小问 — (1) 算反应 ΔH (套公式); (2) 算 27 g 葡萄糖能量 (摩尔数 × |ΔH|); (3) 解释公式物理本质
  • 选路径: 用 ΔHf° 公式『末减初 + 化学计量数加权』, 不用代数组合法 (无现成 3 反应)
  • 执行计划:
    • (1) ΔH = [6·ΔHf°(CO2) + 6·ΔHf°(H2O,l)] - [ΔHf°(C6H12O6) + 6·ΔHf°(O2)]
    • (2) n = m / M; 能量 = n × |ΔH|
    • (3) 用『两步法』解释: ① 葡萄糖分解为单质 (反向 ΔHf°) + ② 单质合成 CO2 + H2O (生成焓加和)
📝 ③ 详细解题步骤
(1)-1 套反应焓变公式:
ΔH = Σν·ΔHf°(产) - Σν·ΔHf°(反)
= [6 × ΔHf°(CO2) + 6 × ΔHf°(H2O, l)] - [1 × ΔHf°(C6H12O6) + 6 × ΔHf°(O2)]
→ 用末减初 + 按化学计量数加权
(1)-2 代入数值:
= [6 × (-393.5) + 6 × (-285.8)] - [1 × (-1273.3) + 6 × 0]
= [-2361 + (-1714.8)] - [-1273.3]
= -4075.8 + 1273.3
= -2802.5 kJ/mol
(2)-1 算 27 g 葡萄糖摩尔数:
n = m / M = 27 / 180 = 0.15 mol
(2)-2 算释放能量:
E = n × |ΔH| = 0.15 × 2802.5 ≈ 420.4 kJ
换算: 1 cal ≈ 4.184 J → 420.4 kJ ≈ 100 kcal(即营养标签的『100 千卡』, 一颗苹果的典型能量值)
(3) 物理本质解释:
反应 C6H12O6(s) + 6O2 → 6CO2 + 6H2O(l) 可拆为两步:
· 第 1 步: 葡萄糖分解为稳定单质 C + H2 + O2 (反向 ΔHf°, 数学上是 -ΔHf°(葡萄糖) = +1273.3 kJ/mol, 吸热)
· 第 2 步: 稳定单质合成 6CO2 + 6H2O(l) (生成焓加和, 6·ΔHf°(CO2) + 6·ΔHf°(H2O,l) = -4075.8 kJ/mol, 放热)
· 总 ΔH = 第 1 步 + 第 2 步 = +1273.3 - 4075.8 = -2802.5 kJ/mol ✓
→ 这就是末减初公式的物理本质: 任何反应都可看作『反应物分解为稳定单质 + 单质合成产物』, 用 ΔHf° 表中数据直接算 ΔH
(1) ΔH = -2802.5 kJ/mol
(2) E ≈ 420.4 kJ (≈ 100 kcal, 一颗苹果的典型能量)
(3) 末减初公式 = 反应物分解为单质 + 单质合成产物 (两步法)
✓ 验证: 量纲 kJ/mol, kJ ✓; 一颗苹果约 100 kcal 与营养学常识一致 ✓; -2802.5 kJ/mol 与文献值 (-2800 ~ -2820) 一致 ✓; 物理本质 (两步法)与盖斯定律路径无关一致 ✓.
⚠️ ④ 易错点
易错 1: 化学计量数忘乘
❌ 错答: ΔH = ΔHf°(CO2) + ΔHf°(H2O) - ΔHf°(C6H12O6) (忘乘 6)
✓ 正答: 必须乘化学计量数! 6 mol CO2 时是 6 × ΔHf°(CO2)
易错 2: 漏 稳定单质 ΔHf° = 0
❌ 错答: 忘把 6 × 0 (O2) 写进公式
✓ 正答: 公式必须完整包含所有物质. 稳定单质 ΔHf° = 0, 但在公式里要写 6 × 0 = 0 (虽不影响数值, 形式上不能漏)
易错 3: 物态选错(H2O(l) vs H2O(g))
❌ 错答: 用 ΔHf°(H2O, g) ≈ -241.8 代入
✓ 正答: 反应方程式中是液态水, 用液态生成焓 -285.8. 物态决定哪个值
易错 4: 能量单位换算错
❌ 错答: 0.15 mol × 2802.5 kJ/mol = 420.4 J (单位错)
✓ 正答: ΔH 单位 kJ/mol, 乘 mol 得 kJ(不是 J)
易错 5: 第 (3) 小问解释空洞
❌ 错答: "因为是状态函数所以路径无关"(没说怎么对应两步法)
✓ 正答: 必须显式拆出两步(分解为单质 + 单质合成产物), 并说明数值=两步 ΔH 加和

5
键能 + 盖斯联用 · 合成氨求 N≡N 键能
A5
较难
题目
工业合成氨 (Haber-Bosch) 是 20 世纪最重要的化工反应, 让人类从大气中固氮 → 大幅提高化肥产量. 反应:
    N2(g) + 3H2(g) → 2NH3(g), ΔH = -92 kJ/mol

已知 (298 K):
· H-H 键能 a = 436 kJ/mol
· N-H 键能 b = 391 kJ/mol

(1) 求 N≡N 键能 (用字母 a, b 表示) = ________ kJ/mol;
(2) 代入数值, N≡N 键能 = ________ kJ/mol;
(3) 解释为何 N≡N 键能如此大 (~946 kJ/mol), 这对工业合成氨的反应条件有何意义?
📚 ① 知识点梳理
  • 涉及键能与反应焓变的关系: ΔH = Σ键能(反应物) - Σ键能(产物) (锚: 键能与反应焓变 Card 2)
  • 涉及计数原子键: NH3 含 3 个 N-H, 每个 NH3 贡献 3 个 N-H 键
  • 涉及盖斯定律: 反应 ΔH 已知 → 反推某键键能 (相当于反推某个单分子的『分解能』)
  • 跨板块综合: 键能式 + 给定 ΔH → 解字母代数方程
🧭 ② 解题思路与方法
  • 拆题: 已知 = 反应 ΔH + 2 个键能; 求 = N≡N 键能 (字母 + 数值)
  • 选路径: 用键能公式 ΔH = Σ反应物键能 - Σ产物键能, 解出 N≡N
  • 执行计划:
    • 计数反应物键: 1 个 N≡N + 3 个 H-H (N2 + 3H2)
    • 计数产物键: 2 个 NH3 × 3 个 N-H/NH3 = 6 个 N-H
    • 列方程: ΔH = (N≡N + 3a) - 6b = -92
    • 解出 N≡N = -92 - 3a + 6b
    • 代入 a=436, b=391 算数值
    • 解释 N≡N 大键能的工业意义
📝 ③ 详细解题步骤
(1)-1 列键能公式:
反应物键能和 = 1 × (N≡N) + 3 × (H-H) = (N≡N) + 3a
产物键能和 = 2 × NH3 × 3 (N-H 每 NH3) = 6 × (N-H) = 6b
ΔH = Σ反 - Σ产 = [(N≡N) + 3a] - 6b
→ 由键能 ΔH 公式
(1)-2 代入 ΔH = -92, 解 N≡N:
-92 = (N≡N) + 3a - 6b
(N≡N) = -92 - 3a + 6b
= 6b - 3a - 92 kJ/mol
(或写作 -3a + 6b - 92)
(2) 代入 a = 436, b = 391:
(N≡N) = 6 × 391 - 3 × 436 - 92
= 2346 - 1308 - 92
= 946 kJ/mol
(3) N≡N 键能大的物理与工业意义:
· 物理: N≡N 是三键 (1 σ + 2 π), 比一般的单键 (~300-400) 强 2-3 倍. 这是 N2 在大气中极稳定的根本原因(占大气 78% 但常温下化学惰性).
· 工业意义: 打开 N≡N 需要 ~946 kJ/mol 的极高活化能 → 故合成氨反应虽然 ΔH = -92 (放热), 但动力学极慢 → 必须用高温 (400-500°C) 提高分子动能 + 催化剂 (Fe-based) 大幅降活化能, 同时高压 (200-300 atm) 利用气体摩尔数减少推产率.
· 这也是为什么 1909 Haber 发现 + 1913 Bosch 工业化是诺贝尔级的成就 — 攻克的就是 N≡N 键能大的难关.
(1) N≡N 键能 = 6b - 3a - 92 kJ/mol (字母表达式)
(2) N≡N 键能 = 946 kJ/mol (数值, 与文献值一致)
(3) N≡N 大键能 → N2 稳定 → 合成氨需高温 + 高压 + 催化剂
✓ 验证: 字母表达式与数值代入自洽 ✓; 946 kJ/mol 与文献 (~945) 一致 ✓; 解释关联到 Haber-Bosch 工业实际 ✓.
⚠️ ④ 易错点
易错 1: 键能公式方向反 (产 - 反 vs 反 - 产)
❌ 错答: ΔH = Σ产 - Σ反 → 算出 N≡N 错号
✓ 正答: ΔH = Σ反 - Σ产(键能消耗 - 键能生成). 易记: 『反应物断键吸能, 产物成键放能』, 故 ΔH = 断键能 - 成键能 = Σ反键 - Σ产键
易错 2: 计数键数错 (NH3 有几个 N-H)
❌ 错答: 觉得 NH3 只有 1 个 N-H → 2 NH3 = 2 个 N-H
✓ 正答: NH3 含 3 个 N-H(画结构: H-N(-H)-H), 2 NH3 = 6 个 N-H. 画结构核对
易错 3: 系数没乘到键能
❌ 错答: 3 H2 只算 H-H 键能 a, 没乘 3
✓ 正答: 3 mol H2 有 3 个 H-H 键 → 键能贡献 = 3a
易错 4: 字母代数整理失误
❌ 错答: -92 = (N≡N) + 3a - 6b → 解 N≡N = -92 + 3a - 6b (符号错)
✓ 正答: 移项: (N≡N) = -92 - 3a + 6b = 6b - 3a - 92. 检验代入数值合理性
易错 5: 第 (3) 小问没联系工业实际
❌ 错答: 只说"N≡N 键能大所以稳定", 没说为何高温 / 高压 / 催化剂都需要
✓ 正答: 必须说放热 ≠ 自发, 动力学障碍 → 高温升速率 + 催化剂降活化能 + 高压推产率

⚠️ 跨主题前置 (反应动力学)
本题图涉及『活化能 Ea / 过渡态 / 决速步 / 中间体』, 来自反应动力学. 本题对动力学只作概念引用, 计算重点仍在 ΔH (热力学). 最小前置概念:
  • 活化能 Ea: 反应物 → 过渡态最高点的能量差(反应能否进行的『能垒』)
  • 过渡态: 反应过程中能量最高的瞬态(图上波峰)
  • 决速步: 多步反应中活化能 Ea 最大的一步(限速)
  • 中间体: 反应过程中临时产物(图上波谷, 本题为 CO + H2O + 2H2)
若不熟, 详见未来『反应动力学』板块. 本例题所有定量计算(ΔH / 反应能否放热 / 催化剂影响)都靠热力学解决.
回答下列问题:
(1) 总反应 CO2(g) + 3H2(g) → CH3OH(g) + H2O(g) 的 ΔH = ________ kJ/mol;
(2) 反应第 1 步 ΔH1 = ________ kJ/mol; 第 2 步 ΔH2 = ________ kJ/mol;
(3) 该两步反应的决速步骤是第 ____ 步 (从图上活化能判断);
(4) 若加入新型催化剂, 使过渡态 1 能量降至 +120 kJ/mol, 过渡态 2 不变. 总反应 ΔH 是否改变? 解释原因 (从盖斯定律 / 状态函数角度).
📚 ① 知识点梳理
  • 涉及盖斯定律核心: ΔH 由始末态决定, 与路径无关 (锚: 盖斯定律 Card 1, 4)
  • 涉及分步反应 ΔH 加和: 第 1 步 ΔH1 + 第 2 步 ΔH2 = 总反应 ΔH (锚: 盖斯定律 Card 2)
  • 涉及状态函数性质: 焓 H 是状态函数, ΔH = H末 - H初 (锚: 状态函数 (进阶))
  • 涉及催化剂作用(关键): 改变路径 → 降低 Ea (动力学), 不改 ΔH (热力学)
  • 跨板块综合 + 跨主题前置 (动力学最小自包含): 用图上能量值读 ΔH (热力学) + 用活化能判决速步 (动力学引用)
🧭 ② 解题思路与方法
  • 拆题: 4 小问 — (1)(2) 读图算 ΔH (热力学); (3) 判决速步(动力学概念引用 + 比较 Ea); (4) 催化剂改不改 ΔH (盖斯核心)
  • 选路径:
    • (1)(2)(3): 读图法 — ΔH = 终点 - 起点; ΔH分步 = 中间能量差; Ea = 过渡态 - 起点
    • (4): 盖斯定律论证 — ΔH 由始末态决定, 中间路径任意
  • 执行计划:
    • (1) 读图: 终点 -49.6 - 起点 0 = -49.6 kJ/mol
    • (2) 读图: ΔH1 = 中间体能量(+41.2) - 起点(0); ΔH2 = 终点(-49.6) - 中间体(+41.2)
    • (3) 比较 Ea1 = 178 vs Ea2 = 过渡态 2(130) - 中间体(41.2) = 88.8; 取 max 步
    • (4) 用盖斯定律: ΔH 与路径无关, 催化剂改路径(过渡态高度), 不改始末态 → ΔH 不变
📝 ③ 详细解题步骤
(1) 总反应 ΔH = 终点 - 起点:
从图: 终点能量 = -49.6, 起点 = 0
ΔH总 = -49.6 - 0 = -49.6 kJ/mol
(放热反应, 合理)
→ 由状态函数性质: ΔH = H末 - H初
(2) 分步 ΔH:
· 第 1 步: 起点(0) → 中间体(+41.2)
ΔH1 = +41.2 - 0 = +41.2 kJ/mol (吸热)
· 第 2 步: 中间体(+41.2) → 终点(-49.6)
ΔH2 = -49.6 - 41.2 = -90.8 kJ/mol (放热)
验: ΔH1 + ΔH2 = 41.2 + (-90.8) = -49.6 = ΔH总 ✓ (与 (1) 自洽, 这就是盖斯定律加和)
→ 由读图 + 盖斯定律加和
(3) 判决速步:
· 第 1 步活化能 Ea1 = 过渡态 1 (+178) - 起点 (0) = 178 kJ/mol
· 第 2 步活化能 Ea2 = 过渡态 2 (+130) - 中间体 (+41.2) = 88.8 kJ/mol
· 比较: Ea1 = 178 > Ea2 = 88.8 → 第 1 步活化能更大
· 由决速步定义: 第 1 步是决速步
· 物理含义: 第 1 步速率最慢, 决定了整个反应的速率
→ 由决速步 = Ea 最大者 (动力学引用)
(4) 催化剂改变后 ΔH 是否变?
· 答: ΔH 不改变, 仍为 -49.6 kJ/mol
· 原因: 由盖斯定律(状态函数性质), ΔH = H末 - H初 只与始末态决定, 与中间路径无关
· 催化剂作用机理: 改变反应路径(降低过渡态能量 + 改变中间体), 但起点(CO2 + 3H2) 和终点(CH3OH + H2O(g)) 物质和能量不变 → ΔH 不变
· 催化剂改变的是动力学(降 Ea → 升速率), 不改热力学(ΔH 不变)
· 工业意义: 催化剂让反应在低温下加速(节能), 但放热总量不变(总能量收益固定)
→ 由盖斯定律 = 状态函数性质 + 催化剂作用机理
(1) ΔH总 = -49.6 kJ/mol
(2) ΔH1 = +41.2 kJ/mol, ΔH2 = -90.8 kJ/mol
(3) 决速步 = 第 1 步 (Ea1 = 178 > Ea2 = 88.8)
(4) ΔH 不改变(路径无关 / 状态函数性质 / 催化剂不改始末态)
✓ 验证: 量纲 ✓; ΔH1 + ΔH2 = ΔH总 自洽 ✓; 总反应放热 (CO2 加氢制甲醇实际放热, 文献 ΔH ≈ -49 kJ/mol) ✓; (4) 答案与盖斯定律核心一致 ✓.
⚠️ ④ 易错点
易错 1: 把过渡态能量当 ΔH
❌ 错答: ΔH总 = 过渡态 1 (+178) - 起点 (0) = +178 kJ/mol
✓ 正答: ΔH = 终点 - 起点(看产物 vs 反应物). 过渡态是中间瞬态, 不是产物. 过渡态 = 活化能 (动力学), 终点 = ΔH (热力学)
易错 2: Ea 起点错(从哪算)
❌ 错答: Ea2 = 过渡态 2 (+130) - 起点 (0) = 130 kJ/mol
✓ 正答: Ea 从该步起点算. 第 2 步起点 = 中间体(+41.2), 不是初始反应物. Ea2 = 130 - 41.2 = 88.8
易错 3: 决速步判错 (取较小 Ea)
❌ 错答: 因第 2 步 Ea2 = 88.8 较小, 觉得第 2 步是决速步
✓ 正答: 决速步 = Ea 最大的一步(限速). 越大 = 越慢. 第 1 步 Ea1 = 178 大, 它是决速步
易错 4: 催化剂改不改 ΔH 答错 (说『催化剂使反应放更多热』)
❌ 错答: 催化剂降 Ea → 反应更容易 → 放更多热 → ΔH 更负
✓ 正答: 催化剂只改动力学(降 Ea → 速率快). ΔH 由始末态决定(状态函数), 催化剂不改始末态 → ΔH 不变. 总放热量 = ΔH × 摩尔数, 不变
易错 5: 盖斯定律的物理本质说浅
❌ 错答: "因为盖斯定律所以路径无关"
✓ 正答: 必须说焓 H 是状态函数 → ΔH 由始末态决定. 路径(包括催化剂改变的)只影响动力学, 不影响 ΔH
